# 整合交易系统 SOP（2026-07-09 定版，回测验证）

> 一页纸决策底座。两层叠加：**原筛选系统答"买什么/为什么/多重"，趋势纪律答"何时买卖/拿多久/怎么不死扛不卖飞"**。谁也替代不了谁。

## 优先级（一阶大事，别被止损细节淹没 · PIT回测校准）

**1. 选股 + 执行纪律 —— 两个都是一阶alpha，都重要（缺一都亏）**
- **选股(alpha来源)**：供给侧成长（SABCT≥A- + 真edge + 营收/利润高增）。**PIT两年回测铁证**：同一套趋势规则，随机股组合=-0.4%（跑输指数38pp），换成"当时可知的成长股"=+114%（超额+76pp）。**edge几乎全在选股。**
- **执行纪律(alpha来源)**：让利润跑满整段趋势（不卖飞）+ 破位/灾难线快砍（不死扛）。这是把选股的账面alpha真正兑现的另一半——**选对了守不住照样亏（这就是我们一直亏的根，复盘19个不该犯的守仓错）。执行也是alpha，不是"只是执行"。**

**2. 仓位（按信心）** — A+大/A中/A-小；说不出thesis不建仓；集中3-5个不铺一堆

**3. 止损纪律（二阶，但必须执行）** — 前低破/灾难-12%/round-trip；硬信号thesis不能override（防死扛）

**⚠️ regime闸门 ≠ "最高杠杆"（我早先错判，PIT证伪）**：PIT两年回测里它反而是拖累（+114%→+40%，回撤还没降）。它是**熊市保险，趋势年是drag**→**默认关**，只在明确普跌/系统性风险时手动收缩，不做机械日频闸门。

## 买入 = 双确认（缺一不买）
- **Layer1 基本面**：SABCT ≥ A- + 真供给侧edge + 估值有边际（PEG/前瞻PE）
- **Layer2 趋势**：突破前25日高确认，且**距突破点 ≤8%**（追高cap，冲太高不追）

## 卖出 = 5道出场门，哪道先响走哪道（回测锁定参数）
| 门 | 触发 | 说明 |
|---|---|---|
| 1 破位（前低止损） | 收盘 < 前10日最低 | 主力出场门，让赢家跑、破了就走 |
| 2 灾难线 | 收盘 ≤ -12% | **绝对地板，thesis不能override（先减半）** |
| 3 round-trip | 峰值≥+15% 又吐回成本 | 防"坐电梯"（金石/永和教训） |
| 4 thesis证伪 | 供给约束/主beta/催化时间线变坏 | 基本面独立判 |
| 5 催化剂兑现 | 利好落地+动能衰竭 | sell the news；突破买入后主动卖点 |
| 6 regime转空 | 大盘转普跌 | 全体收紧、抬现金 |
| — | 没门响 | **让它跑**（这才吃得到+128%） |

⛔ **防死扛硬规则**：破位 + 灾难线同时破 = 无条件出，thesis/信心/催化剂都不能当挡箭牌。基本面只决定"给多宽空间"，不决定"破了还留"。

## 加仓（让利润奔跑的进攻侧）
突破后续创新高 + 回踩不破前低 → 加；总仓受信心上限。

## 持有期匹配买入性质
- 埋伏仓（深研+回踩买）→ 长持、宽结构止损（配基本面证伪）
- 追高仓（突破动量、无深研）→ 短持、紧前低止损、机械执行

## 回测验证（2026-07-09，逐层加严到PIT金标准）
- 参数：前高突破25 / 前低止损10 / 灾难-12% / round-trip+15% / 追高cap8%
- **⭐PIT金标准（无后视镜，随机299只无手选，2年，按报告期+60天才算可知）**：
  - ① 纯趋势·无筛 = **-0.4%**（跑输沪深300约38pp）→ **趋势规则本身零edge**
  - ② 趋势+PIT成长筛（营收>15%）= **+114%**，超额+76pp，胜率38%，盈亏比2.28 → **edge全在选股**
  - ③ ②+regime闸门 = +40%（闸门是拖累，非最高杠杆）
- **胜率38%=62%时候错，全靠盈亏比（赢+30%/亏-8%）；纪律吃下所有信号+快砍=执行难点不是策略难点**
- **⛔别被大数骗**：手选universe的组合引擎曾跑出+134%/+435%=后视镜污染的海市蜃楼，不可信。只有PIT数可信。
- 局限：仍1个2年窗口、集中8只高波动（大赢家驱动）、成长筛是机械代理（真质化选股或更强也可能更脆）。强证据非铁证。

## 不钻牛角尖
系统是纪律底座不是唯一镜片。会漏"横盘突破前埋伏""基本面反转未突破"的机会——框架内保留judgment自由度，别锁死成纯机械。
