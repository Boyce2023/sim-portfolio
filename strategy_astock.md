### §2.2 统一SABCD评级系统（v8.1）

**评级 = Conviction强度。** 止损管实际风险，评级管conviction。Bear case不决定评级上限。

| 等级 | Conviction标准 | 仓位硬顶 | ATR K值 | 最大硬止损 |
|------|---------------|---------|---------|-----------|
| **S** | 多维信号命中 + ≤4分析师 + PEG极端 + 三层thesis | 50% | 3.5 | -20% |
| **A+** | 三层thesis验证 + 催化剂<30天 + 竞争壁垒确认 | 35% | 3.0 | -18% |
| **A** | 强thesis + 催化剂明确 + risk/reward好 | 25% | 3.0 | -15% |
| **A-** | 好thesis + 一个不确定性（时机/执行/宏观） | 20% | 2.5 | -15% |
| **B+** | 不错thesis + 催化剂稍远或稍模糊 | 15% | 2.0 | -12% |
| **B** | 边际thesis，观望为主 | 12% | 2.0 | -10% |
| **B-** | 弱thesis，最小仓位 | 10% | 1.5 | -10% |
| **C** | **观察池（0%）** — 有潜力但催化剂不明确 | — | — | — |
| **D** | **排除/清仓** — bear case>40%或thesis证伪或板块结束 | — | — | — |

**Bear Case的唯一角色——二元过滤器**:
- ≤40%: **通过**，进入conviction评级。bear case 25%的票照样可以给A+，只要conviction够强
- >40%: **D级排除**，不浪费时间
- Bear case不再决定S/A+/A之间的等级差异。不确定性是alpha来源，不是降级理由

**深度研究的正确用法**: 确认thesis是否成立，不是搜集降级弹药。问"这个机会真不真"，不是"能找到多少问题"。中际旭创和寒武纪在起飞前都有一堆bear case——但thesis是对的，催化剂兑现了。

**共识信号（F15 A股修正v2）**: 卖方全看多 ≠ 排除。看价格折让：
- 股价≥共识目标价 → 真拥挤，降优先级
- 股价低于目标15%+ → 共识方向对，alpha在择时
- 结构性供给变化刚开始 → 共识可能低估，优先研究

**S级特别条件**: ①多维度信号同时命中 ②分析师覆盖≤4人 ③业绩beat频率>80% ④PEG<0.6或行业绝对最低 ⑤已持有≥5天。历史标杆：中际旭创(2024)、寒武纪(2023)。

**数量约束**: S≤1。A+/A/A-数量不设硬上限(05-27取消旧限制)，按conviction排序灵活配置。C级不占持仓名额。

**升降级规则**:
- 升级: 催化剂兑现+符合预期 → 升1级，加仓至新等级。写理由
- 降级: 催化剂推迟>14天 → 降1级；thesis被证伪 → D级
- 板块结束信号(§4.3) → 同板块全部降为C或D
- D级 → 48h内清仓，无例外

### §2.4b 双轨Thesis系统（v9.1新增）

A股牛市中两种thesis并行，选股时必须明确走哪条轨：

| | **Track A: 基本面** | **Track B: 溢价外溢** |
|---|---|---|
| **买什么** | 瓶颈节点/板块龙头 | 同板块小市值（龙头涨完后的接力票） |
| **Thesis** | 公司质量+竞争壁垒+业绩验证 | 板块热度+资金搬家+小市值弹性 |
| **典型标的** | 中际旭创(1.3万亿/毛利46%) | 蘅东光(386亿/北交所唯一光连接) |
| **催化剂** | 自身业绩/产品/客户 | **绑定板块龙头催化剂**(龙头的α=小票的β) |
| **评级上限** | 无上限(S级) | **与Track A同等SABCT**(v9.2取消B+上限) |
| **仓位** | conviction决定(最高50%) | **由Track B SABCT等级决定**(与Track A同表) |
| **止损锚** | 自身ATR+thesis | **板块龙头跌>5%=同日出场** |
| **持有周期** | 中长(催化剂兑现周期) | **短(1-3周)** |
| **出场信号** | R倍数/催化剂兑现/thesis证伪 | 龙头见顶/板块轮动/成交量萎缩 |

**Track B触发条件（全满足才做）**:
1. 🟢牛市 + 吸气/强吸气
2. 板块龙头已大涨(涨幅>50%+)且仍在创新高
3. 目标票在同板块，市值<500亿，业务相关
4. 龙头催化剂仍在前面（如COMPUTEX未到）

**Track B铁律**:
- 龙头一旦跌>5%，Track B全部出场，不等自己的止损线
- 不做逆势Track B——板块已在回调时不抄底小票
- Track B不升级为Track A（除非公司本身thesis验证后重新评级）

→ 蘅东光案例: 用Track A评=B-(基本面弱)，用Track B评=B+(溢价外溢有效)。**框架对了结论才对。**

**★★★ Track B完整操作手册**: `rotation-strategy/TB_SYSTEM_V1.md`（V1.0，1756行，§0-§12）。
本节仅为Track B概述。评级、建仓、退出、发现系统、PlayBook、Pain/Reward、生命周期管理、双轨整合、应急协议等完整规则均在TB_SYSTEM_V1.md中。

### §2.4c 双轨整合协议 DIP v1.0（v9.2新增）

> Track A = conviction filter（买什么）。Track B = timing amplifier（何时买、买多少）。
> 单一轨道不完整：Track A选好公司但可能时机差；Track B抓动量但可能质量差。

**DIP-1: 情报互通（Cross-Pollination）**

| 方向 | 触发条件 | 动作 |
|------|----------|------|
| B→A | Track B发现板块D4=启动且D1≥A | 自动触发：对该板块龙头做Track A基本面评估 |
| B→A | Track B D4=退潮 | 通知：同板块所有Track A持仓进入退出审查 |
| B→A | Track B D1=S（龙虎榜机构共振） | Track A同板块标的entry timing绿灯 |
| A→B | Track A确认板块龙头thesis + 催化剂 | 自动触发：Track B扫描同板块小盘票 |
| A→B | Track A降级板块龙头 | Track B同板块持仓触发退出审查 |
| A→B | Track A供应链研究完成 | Track B获得更准确的D3评分依据 |

**DIP-2: 同票双轨评级协调**

当同一标的同时拥有Track A和Track B评级时：

| Track A | Track B | Combined Grade | Sizing | 逻辑 |
|---------|---------|---------------|--------|------|
| ≥A- | ≥A- | max(A,B) + **升1档**(cap S) | 升档后的仓位上限 | **双轨共振**：基本面+动量同时确认 |
| ≥B+ | ≥B+ | max(A,B) | 较高等级的仓位上限 | 双轨确认，取强侧 |
| ≥B+ | C/D | Track A grade | Track A cap **× 0.7** | 基本面好但momentum缺失，减仓等确认 |
| C/D | ≥A- | Track B grade | Track B cap, **TB规则退出** | 纯动量play，短周期+龙头止损 |
| D | 任何 | **D级排除** | 0 | Track A D级一票否决 |
| 任何 | D | Track A **降1档** | 降档后cap | Track B D级 = momentum极差信号 |

**双轨共振案例**：思源电气 Track A=A（电力设备thesis+催化剂）+ Track B=A-（D1=A/D2=T2/D4=启动）→ Combined=A+（升1档）。sizing从25%→35%。

**DIP-3: 仓位动态分配**

F20 market breath决定Track A和Track B的资金分配上限：

| F20状态 | Track A上限 | Track B上限 | 合计可用 | 逻辑 |
|---------|------------|------------|---------|------|
| **强吸气** | 60% | 50% | 110%(可杠杆) | TB最大化，小票先飞最佳窗口 |
| **吸气** | 70% | 40% | 100% | 正常双轨并行 |
| **中性** | 80% | 25% | 95% | TB收缩至高conviction only |
| **呼气** | 85% | 10% | 85% | TB仅保留S/A+级 |
| **深度呼气** | 60% | 0% | 60% | TB全退，TA减至防御 |

Track B在呼气→深度呼气过程中逐步退出：先退B级以下 → 再退B+/A- → 最后退A/A+。
Track A在同期保持核心持仓但停止新建仓。

**DIP-4: 分歧信号解读**

Track A和Track B方向分歧时，分歧本身就是alpha信号：

| Track A动态 | Track B动态 | 信号含义 | 建议操作 |
|-------------|-------------|----------|---------|
| 升级中 | D4=启动, D1≥A | **★最强买入** | 满仓位建仓 |
| 稳定 | D4=主升早 | 正常看多 | 按双轨各自规则 |
| 稳定 | D4=高潮分歧 | 板块到顶信号 | 减TB仓位，TA保核心 |
| 降级中 | D1=S, D4=主升早 | ⚠️末班车风险 | 可小仓TB试探，不追TA |
| 稳定 | D1=X, D4=退潮 | 板块休整期 | 持有TA(thesis在)，不做TB |
| 升级中 | D4=退潮 | 🔍逆风基本面 | 减仓等momentum确认才加回 |
| 降级 | D4=退潮 | 双杀信号 | 全板块退出 |

**DIP-5: 退出先后序列**

1. **Track B先退**：短期momentum信号消失(D4变退潮 / 龙头-5% / 成交量萎缩)
2. **Track A视原因**：
   - TB退出原因仅是板块冷却(非thesis相关) → Track A保持，不受影响
   - TB退出原因涉及thesis(供给过剩暴露/竞争格局恶化) → Track A降级审查
3. **反向触发**: Track A thesis证伪 → 同板块TB持仓同日退出审查
4. **双杀紧急**: Track A降级 + Track B D4=退潮 → 同板块全部持仓24h内清仓

## §3 防守系统

### §3.1 三层ATR止损

**Layer 2 — 棘轮锁利**:

| 阶段 | 止损调整 |
|------|---------|
| +1R | 移至保本 |
| +2R | high - 2×ATR |
| +3R | high - 1.5×ATR |
| +50%+ | high - 1×ATR |

止损只上移不下移。每session检查所有持仓是否触发棘轮移动（不依赖手动记忆）。

**Layer 3 — 时间衰减**:

| 条件 | 动作 |
|------|------|

| 每多5天 | 再紧0.5×ATR |

| 20天无催化兑现 | 强制重评 |

## §4 持仓管理

### §4.1 加仓规则

全部满足: ①持仓<等级硬顶 ②Thesis未证伪 ③催化剂仍在30天内

### §4.2 R倍数出场梯

1R = Entry - 止损价。出场按R倍数分级：

| 阶段 | 操作 | 止损 |
|------|------|------|
| +2R | 卖25% | 保本 |
| +3R | 再卖25% | high-1.5×ATR |
| +4R+ | Trail 50% | high-1×ATR |
| 催化剂兑现 | 3天决策窗口 | — |

目标价: T1≥+2R, T2≥+4R。达不到+2R=risk/reward不够→降级。
A股T+1: 大涨日(+1R以上)不卖。收盘前30分决策，次日执行。

**催化剂前减仓规则（L11修正, 05-28纠偏）**:
- α催化剂T-5天：已盈利持仓减至≤15%仓位，锁定大部分利润
- 小仓位（10-15%）赌催化剂超额，大仓位赚确定性钱——不可反过来
- "在等什么"答不出具体事件+日期 = 没理由持有大仓位
- 鹏鼎教训：+30%后仍持大仓位等WWDC，回补后亏。应先锁利+小仓位博事件

**化工/周期股特例(B4)**: 标准10日窗口对化工周期股偏短。巨化类标的可扩展至15-20日，但必须有催化剂在窗口内（如旺季/配额/价格数据）。回测验证: 巨化10日出-1.75%, 20日后才兑现。

## §10 Pain/Reward Architecture — A股版

**FOMO判定系统（v9.1重写 — 旧系统50%错误率）**:

旧系统问题：RT亏损就标FOMO，PE>60x就标FOMO。但歌尔(WWDC催化剂, 卖后+6.6%)和恒瑞(ASCO前3天, 卖后+41%)都被错标——有催化剂的票不是FOMO。

**FOMO判定三步检查（先过才标FOMO）**:
1. **催化剂检查**: 买入时有α/β级催化剂→**不是FOMO**，是催化剂交易（即使亏了也是timing问题不是FOMO）
2. **Thesis检查**: 买入时有完整thesis→**不是FOMO**，是thesis错误（不同pattern tag）
3. **以上都没有→才是FOMO**: 无催化剂+无thesis+看别的涨了想买 = 真FOMO，tag`fomo_rotation`

**FOMO vs 其他亏损模式（必须区分）**:
| 模式 | 特征 | Tag | 处罚 |
|------|------|-----|------|
| **真FOMO** | 无催化剂+无thesis+冲动买入 | `fomo_rotation` | L3降档+同周第2次→🟡 |
| **Timing错误** | 有催化剂但入场太晚/太早 | `timing_error` | 不降档，优化entry规则 |
| **Thesis错误** | 有thesis但thesis本身错了 | `thesis_wrong` | 审查研究质量 |
| **Round Trip** | 卖了又买回（无新催化剂） | `round_trip` | §4.7获利回补禁令 |

→ **亏损≠FOMO。** 亏损先分类再处理。错误标签比亏损本身危害更大——错标FOMO会让系统回避有催化剂的好机会。

| **V形反弹** | 政策底确认+放量 | **超跌优质成长**(最深+最强) | 50-60% | 2022 May-Jun, 2024年9月底 |
| **磨底** | 政策底→市场底(间隔1.5-3月) | 政策受益+券商+分批 | 30-40% | 2018 Oct-Dec |

→ **慢熊和V形反弹的alpha来源完全相反。** 慢熊买防御/供给，V反弹买超跌成长。
→ V形反弹板块顺序：新能源/科技(最先) → 材料/工业 → 消费(最晚)

### BM3 熊市建仓五步法（替代§2.1）

```
Step 1: Regime确认 — 必须是🔴熊市。§10.5评分≤-3连续2周
Step 2: 子类型判断 — 慢熊/急跌/V反弹/磨底？决定策略方向
Step 3: Alpha来源匹配 — BM1排序，优先供给收缩>国产替代>政策脉冲
Step 4: 催化剂质量 — 熊市只做α级(特定日期)。γ级禁止。β级cap C级
Step 5: 参数应用 — §10.5表自动：K≤1.5, 止损-8~-10%, 单只≤12%, 现金≥50%
```

### BM4 熊市催化剂规则

**政策级别→Alpha量级（T3验证）**:

| 级别 | 1周超额 | 持续性 | 入场窗口 |
|------|--------|--------|---------|
| 政治局/习近平主持 | +15-66% | 1-3月 | **当天或T+1** |
| 国常会/副总理级 | +5-15% | 2-4周 | T+1 |
| 部委单独发文 | +3-8% | 1-2周 | T+1~T+3 |

- **T+3后收益率大幅缩减**——政策催化的最佳窗口极短
- **政策底≠市场底**——平均滞后1.5-3个月。轻仓参与反弹，不All-in
- **政策alpha最大杀手=外部冲击**（贸易战/疫情），不是政策反复

### BM5 熊市出场规则（按子类型分化，Wave 2迭代）

**出场参数按alpha来源分化（VAL5压力测试验证）**:

| Alpha来源 | 止盈目标 | 时间衰减起始 | 单只仓位上限 |
|----------|---------|------------|------------|
| **供给收缩** | +2.0R | 10-15天 | 15% |
| **国产替代** | +1.5R | 7-10天 | 12% |
| **政策脉冲** | +1.5R | 5天 | 10% |
| **超跌反弹** | +1.0R | 3天 | 8% |
| **高股息/防御** | trailing stop | 无(长持) | 10% |

**通用出场规则**:

| 场景 | 操作 |
|------|------|
| 达到止盈目标 | 卖50%,剩余trailing |
| 持有超过时间衰减 | 收紧0.5×ATR |
| 持有>20天无催化剂进展 | 强制重评,大概率清仓 |
| 超跌反弹+2R | 全清（反弹幅度有限） |

**⚠️ 政策底V反弹专用出场规则（VAL3 Critical Gap修复）**:
- 触发条件: V反弹子类型 + 政治局级政策底 + α催化剂
- **不使用固定R倍数止盈**，改用**动态trailing stop**: 跌破5日均线才清仓
- 原因: 924当天+1.0R止盈只捕获~3%，实际行情+25-30%。遗漏90%收益
- 此规则仅在"政治局级政策底+V反弹"同时满足时启用

**现金底线动态化（VAL5验证）**:
| 子类型 | 现金底线 |
|--------|---------|
| 慢熊/磨底 | ≥40-45% |
| 急跌 | ≥70% |
| V反弹 | ≥55-60%（蓄力等待买点） |

→ 熊市持仓窗口20-45天（vs牛市60-120天）。超过45天必须重新验证催化剂。

### BM6 三底共振逆向规则（T4基金方法论验证）

**逆向买入的前提——三底缺一不可**:
1. **估值底**: 板块PE/PB处于近5年<10%分位
2. **政策底**: 政治局级政策已出（不是部委喊话）
3. **情绪底**: 成交量<20日均量50% + 融资余额月环比<-10%

三底全齐 → 分批建仓（3天内完成）
缺政策底 → **禁止逆向**（景林加仓中概的教训：估值便宜+情绪恐慌但监管风险=接飞刀）
缺情绪底 → 等（情绪没到极端说明还没跌够）

### BM7 熊市因子优先级

Quality > Low Volatility > Dividend > Value

→ **Value单独用有价值陷阱风险**。必须叠加Quality(ROE>15%+自由现金流为正)才安全。
→ 熊市小市值(30-100亿)+高业绩增速是alpha富集区（T1验证），但2024年1月量化踩踏提醒：小盘在极端去杠杆时有流动性风险。

## §10.7 Anti-Overfitting System（反过拟合系统）

> 防止规则过拟合单一regime。每条规则标记有效域+权重衰减。

### Rule Validity Tags（规则有效域标记）

每条交易规则标记以下有效域：

| Tag | 含义 | 仓位调整 |
|-----|------|---------|
| `all_market` | 牛熊均验证有效 | ×1.0 |
| `bull_confirmed` | 牛市验证,熊市未验证 | 熊市中×0.5 |
| `bear_confirmed` | 熊市验证,牛市未验证 | 牛市中×0.5 |
| `tentative` | 样本<50笔,待确认 | ×0.7 |
| `regime_mismatch` | 与当前regime不匹配 | ×0.1 |

**当前v8.4规则分类**:
- `all_market`: F20市场呼吸矩阵 / 板块结束信号(龙头跌>5%) / ATR止损框架 / 催化剂αβγ分级框架 / 催化剂前减仓
- `bull_confirmed`: 顺势入场偏好 / 量能>8000亿门槛 / +2R止盈 / 单只25-50%
- `bear_confirmed`: 超跌反弹优先 / 三底共振 / +1.0R止盈 / 单只≤12% / γ禁止

### Paradigm Failure Detection（范式失效检测）

| 触发器 | 条件 | 严重度 | 行动 |
|--------|------|--------|------|
| PF-1 | 催化剂交易连续3周跑输CSI300 buy-and-hold | Warning | 审查催化剂质量 |
| PF-2 | Rolling 10笔win rate < 45% | Warning | 减仓至震荡参数 |
| PF-3 | 月度交叉验证:反向regime规则连续3月更优 | Critical | **强制Regime Review** |
| **PF-4** | **A/A+级交易平均收益 < B/C级交易** | **Critical** | **立即停止新建A级仓位** |
| PF-5 | 累计alpha vs CSI300连续4周为负 | Critical | 全仓review+切换regime |

→ **PF-4是过拟合最早的信号**——高信心特征变成负选择器,比整体胜率崩塌提前2-4周。

### Freshness Decay（权重衰减）

| 验证级别 | 权重 | 月衰减率 | 说明 |
|---------|------|---------|------|
| 回测验证 | ★ | 20%/月 | 5个月后权重≈33%,需实盘验证续命 |
| 实盘验证 | ★★ | 10%/月 | 实盘确认的规则更可靠 |
| 用户纠偏确认 | ★★★ | 5%/月 | 互动验证的规则最持久 |

→ 2025年回测规则如果半年后仍未实盘验证,权重自动降到≈10%。防止回测过拟合固化。

### Monthly Cross-Validation（每月末执行）

1. 取最近3个月实盘交易
2. 用当前regime规则算EV(A)
3. 用反向regime规则算EV(B)
4. 如果EV(B) > EV(A)连续3个月 → 触发PF-3,强制Regime Review
