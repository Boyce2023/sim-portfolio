## 铁律 (L10-L16)

- **L10**: 看到别的涨想换仓→问thesis变了吗？因比较想卖？是→关行情等1h
- **L11**: 催化剂前默认持有。"涨了不少"不是减仓理由
- **L12**: "现在价格我会新买吗？"不会→卖。会→持有
- **L13**: 三行检查：①理由不是"涨了" ②跌20%割不割 ③今天才知道→不买
- **L14**: 建仓定信心等级：A(3层逻辑)≤25% / B(1-2层)≤15% / C(直觉)≤8%
- **L15**: 搜"它最近为什么涨"——驱动力≠买入理由→不买
- **L16**: S/A级必查管理层：减持/处罚/质押/审计/实控人。1红旗→降2级

## Multi-Strategy铁律 v5.0

1. 买前先分类: 确定原型→分配Book→规则全程适用
2. 规则跟Book走: Book A跌了=止损 / Book C跌了=可能加仓
3. 单只≤15%: A级最高15%。投机≤3%。全组合投机≤10%
4. 总杠杆≤1.3×: VIX>25降至1.0×
5. 月≤15笔
6. 同一标的不亏两次: 再入场需新催化剂+冷却2周+C级起步
7. R1赚了≠做R2: 投机不加仓
8. 回撤阶梯: Book -15%减半 / -20%清仓 | 账户-10%停建仓
