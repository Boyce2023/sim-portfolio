## BEFORE OPENING A POSITION（开仓前每笔必过）

### 强制闸门

- [ ] **Regime检查**（30秒）：VIX<18=绿灯 / 18-25=黄灯 / >25=红灯禁止做空 / >35=逆向做多
- [ ] **5维度评分≥20/50**（排除线；≥28=C级；≥35=A级；≥42=S级）
- [ ] **L13三行检查**：①理由不是"涨了" ②跌20%割不割（说不出→不买）③今天才知道→不买
- [ ] **L15 Thesis验证**：搜"[标的]最近为什么涨"→与thesis一致？不一致=不买
- [ ] **thesis一句话能说清楚**（说不清=C级，上限8%）
- [ ] **持仓数检查**：多头≤6只 + 空头≤3只（总≤9只），超限先清最弱再建新仓
- [ ] **最低建仓$7,500**（低于此金额一律拒绝）
- [ ] **Bear case≤35%**（>35%排除；25-35%仅T级+明确止损；15-25%无约束但注意）
- [ ] **R/R比≥1.5**（目标涨幅÷止损幅度）
- [ ] **止损价已设定**（S=-7%硬止损 / A=-12%trailing / B=-10%trailing / C/T=-8%硬止损）
- [ ] **操作后现金≥15%**（$22,500；加仓前须≥20%）
- [ ] **催化剂30天内有明确日期**（A级例外）
- [ ] **GAP规则**：>15%当日gap→等3-5天回调到gap 50%再入；5-15%→当天半仓次日加满
- [ ] **VIX>25禁止开新空头**（H2 Agent-09：4笔全输-48%，最贵教训）

### S级额外检查（S1-S5全部YES才可建仓）

- [ ] S1：供应链节点物理不可替代（同级供应商≤2家）
- [ ] S2：催化剂30天内确认，收入传导≤2跳
- [ ] S3：同链小票已先飞（信号已出现）
- [ ] S4：入场点为底部放量/反转形态（非高位追涨）
- [ ] S5：Momentum分≥8/10
- [ ] 当前S级持仓数=0（最多同时1只S级）
- [ ] **S级止损预设-7%**，催化剂当天减至50%，催化剂后Day 3全退，持仓≤10个交易日

## BEFORE CLOSING A POSITION（减仓/清仓前）

### 卖出前必须先执行ABCD分类（禁止跳过）

| 类型 | 判断标准 | 行动 |
|------|---------|------|
| A | SPY同步跌≥2.5%，无个股新闻 | Hold，不恐慌 |
| B | 指数稳，无新闻，下跌<3% | Hold，观察1-2天 |
| C | 有新闻但不涉及core thesis | 减仓0-30%，24h内给升D或维持C结论 |
| C+ | 单一信源/传闻，无硬数据 | 24h核实，核实前不动；超48h视为D |
| D | 财报miss≥2季/重大客户流失/政策文件否决 | 立即清仓，48h内执行，无例外 |

### 卖出逻辑检查（L10-L12三问）

- [ ] **L10**：想卖的原因是"别的涨了它没涨"？→ 是=关行情等1小时再决定
- [ ] **L11**：催化剂还在前面就想减仓？→ 需给出"非催化剂"充分理由（D类或大幅超目标价）
- [ ] **L12**："如果这是新投资我会买吗？" → 是=持有；不是=才是真卖出信号

### 止盈触发（命中任一执行）

- 涨至目标价50% → 卖1/3
- 涨至目标价100% → 再卖1/3，剩余转trailing stop（A级-12%/B级-10%）
- 高PE(>40x)财报后 → 持有8-10天蜜月期后Day 10主动减半
- C/T级达目标价 → 直接全退，不做trailing

### 止损/时间止损触发（命中立即执行，无商量）

- S级：从入场价-7%（硬止损，EOD确认，一次全退）
- C/T级：从入场价-8%（硬止损）
- A级：从高点trailing -12%
- B级：从高点trailing -10%
- **所有空头**：VIX>25立即cover（硬规则）
- **空头亏损**：-10%硬止损
- **单日暴跌>15%**：无视等级，一次全退
- T级：10个交易日无catalyst落地→全退
- C级：20个交易日无推进→全退

## DAILY RISK CHECKS — L17执行链（W3开盘前5步，不可跳过）

1. **读portfolio_state.json**确认当前美股仓位/现金/NAV（唯一真相源）

3. **Regime Detection**（30秒）：VIX水位 + VIX 5日delta + TNX 5日变化 + 2Y-10Y利差
4. **执行pending_actions**（上一session遗留美股操作）
5. **扫描催化剂24h倒计时**（是否有今天到期的财报/事件）

### Regime状态速查

| VIX水位 | Regime | 行动 |
|--------|--------|------|
| <18 | RISK-ON | 正常交易，momentum入场 |
| 18-25 | CAUTIOUS | 暂停新建仓，等2天确认 |
| >25 | RISK-OFF | 禁止新空头；24h内关闭现有空头；多头减至60% |
| >35 | EXTREME FEAR | 逆向做多quality（历史91%WR），不追空 |

**三重信号**：S1 VIX 5日delta（Warning>+20% / Action>+30%）; S2 TNX 5日（Warning>+10bp / Action>+15bp）; S3 利差（Warning收窄>10bp/周）。1个触发=Cautious；2+同时=Regime Shift Confirmed。

## WEEKLY SHORT SCAN（每周三22:00，★★★强制，不可跳过）

1. 读watchlist_config.json short_candidates
2. 查VIX → >25则跳过本周扫描（记录原因）
3. 对每个候选打空头评分卡（满10分）：基本面恶化×0.30 + 估值过高×0.25 + 负面催化剂×0.25 + 技术破位×0.20
4. ≥8.5分→立即执行；7.0-8.4→加候选/优先Put；<7.0→跳过
5. 写扫描记录（0笔也必须写）

**空头目标仓位：总portfolio 10-15%，单只≤5%（$7,500），同时≤3只**

做空双重门槛：① 5维度框架（空头方向）≥28/50 AND ② 空头专项评分≥7.0/10，两者均须满足。
