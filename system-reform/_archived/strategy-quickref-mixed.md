## BEFORE OPENING A POSITION（开仓前每笔必过）

### 通用闸门（A股和美股均适用）

- [ ] **L13三行检查**（三条缺一不可，否则不建仓）：①买的理由不是"涨了" ②跌20%我割不割（说不出→不买）③今天才知道→不买
- [ ] **L15 Thesis验证**：搜"[标的名]最近为什么涨"→与自己thesis一致？不一致=不买
- [ ] **thesis一句话能说清楚**（说不清=C级，上限8%）
- [ ] **R/R比 ≥ 1.5**（目标涨幅÷止损幅度，低于1.5不建仓）
- [ ] **止损价已设定**（按等级：S=-7% / A=-12%trailing / B=-10%trailing / C/T=-8%）
- [ ] **操作后现金≥15%**（加仓前须≥20%）
- [ ] **单日新建仓≤3只**（加仓不计）
- [ ] **催化剂30天内有明确日期**（A级例外）

### A股专用闸门

- [ ] **市场温度**：沪市成交<8000亿→不追新仓；>12000亿→减仓等回调
- [ ] **催化剂在前还是后**：在后面→可买（涨停也买）；已兑现→不买
- [ ] **小票先飞信号**：同主题≥2只小票连板=龙头布局窗口开启
- [ ] **A股Bear case≤40%**（>40%硬排除；25-40%仅A级+事件驱动+减半size）
- [ ] **A股Bear case 25-40%时**：需A级conviction + 明确催化剂 + 初始仓位减半（≤12%）
- [ ] **E9禁止**：不买低PE/高分红/破净股（A股价值陷阱）
- [ ] **E10禁止**：不因"涨了多少"等回调；正确问题是"催化剂在前还是后"

### 美股专用闸门

- [ ] **Regime检查**（30秒，每次入场前）：VIX<18=绿灯 / 18-25=黄灯 / >25=红灯禁止做空 / >35=逆向做多
- [ ] **5维度评分≥20/50**（排除线；≥28=C级；≥35=A级）
- [ ] **US持仓数检查**：当前多头≤6只 + 空头≤3只（总≤9只），超限先清最弱再建新仓
- [ ] **美股最低建仓$7,500**（低于此金额一律拒绝）
- [ ] **美股Bear case≤35%**（>35%排除做多；25-35%仅T级试仓+明确止损）
- [ ] **Gap规则**：>15%当日gap→等3-5天回调到gap 50%再入；5-15%→当天半仓次日加满
- [ ] **VIX>25禁止开新空头**（H2 Agent-09：4笔全输-48%，最贵教训）

### S级额外检查（S1-S5全部YES才可建仓）

- [ ] S1：供应链不可替代（同级供应商≤2家）
- [ ] S2：催化剂30天内确认，收入传导≤2跳
- [ ] S3：同链小票已先飞（信号已出现）
- [ ] S4：入场点为底部放量/反转形态（非高位追涨）
- [ ] S5：催化剂失败后bear case<10%（美股加：Momentum分≥8/10）
- [ ] 当前S级持仓数=0（最多同时1只S级）
- [ ] **S级止损预设-7%**，催化剂当天减至50%，催化剂后Day 3全退，持仓≤10个交易日

## BEFORE CLOSING A POSITION（减仓/清仓前）

### 卖出前必须先执行ABCD分类（禁止跳过）

| 类型 | 判断标准 | 行动 |
|------|---------|------|
| A | 指数同步跌≥1.5%(A股)/≥2.5%(美股)，无个股新闻 | Hold，不恐慌 |
| B | 指数稳，无新闻，下跌<3% | Hold，观察1-2天 |
| C | 有新闻但不涉及core thesis | 减仓0-30%，24h内给升D或维持C结论 |
| C+ | 单一信源/传闻，无硬数据 | 24h核实，核实前不动；超48h视为D |
| D | 财报miss≥2季/重大客户流失公告/政策文件否决 | 立即清仓，48h内执行，无例外 |

### 卖出逻辑检查（L10-L12三问）

- [ ] **L10**：想卖的原因是"别的涨了它没涨"？→ 是=关行情等1小时再决定
- [ ] **L11**：催化剂还在前面就想减仓？→ 需给出"非催化剂"充分理由（D类或大幅超目标价）
- [ ] **L12**："如果这是新投资我会买吗？" → 是=持有；不是=才是真卖出信号

### 止盈触发（命中任一执行）

- 涨至目标价50% → 卖1/3
- 涨至目标价100% → 再卖1/3，剩余转trailing stop
- 美股高PE(>40x)财报后 → 持有8-10天蜜月期后Day 10主动减半
- A股催化剂已兑现 → 3天内决定是否退出

### 止损/时间止损触发（命中立即执行，无商量）

- S级：从入场价-7%（硬止损，不等ABCD分类）
- C/T级：从入场价-8%
- A级：从高点trailing -12%
- B级：从高点trailing -10%
- **所有空头**：VIX>25立即cover（硬规则）
- **空头亏损**：-10%硬止损
- **单日暴跌>15%**：无视等级，一次全退
- T级：10个交易日无catalyst落地→全退
- C级：20个交易日无推进→全退
