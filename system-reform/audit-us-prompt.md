**现状**: Prompt完全没有Regime Detection。VIX仅在做空框架里出现一次（"VIX>25=停止一切做空"），没有三重信号体系，没有决策树，没有事件后SOP。

**实际规则（US_TRADING_SYSTEM_V4.md §7）**:
- 三重信号：S1 VIX 5日delta（Warning>+20% / Action>+30%）；S2 TNX 5日变化（Warning>+10bp / Action>+15bp）；S3 2Y-10Y利差（收窄>10bp/周）
- 触发逻辑：1个信号=Cautious；2+同时=Regime Shift Confirmed
- 四种Regime状态（RISK-ON/CAUTIOUS/RISK-OFF/EXTREME FEAR）各有不同操作规则
- 事件后24h强制检查SOP（FOMC/CPI/NFP触发）
- 五条宏观传导链速查表（FOMC鹰派/鸽派/CPI超预期/地缘冲突/停火等）

**缺失代价**: H1 Agent-14(-26%) + H2 Agent-09(-48%)，两次合计-73%（均为Regime盲目导致）。这是**唯一最高ROI改进**。

### 1.2 五维度评分框架 — 系统骨架缺失

**现状**: Prompt用的是"Step 4 Bear case 4-tier"替代选股判断，没有任何评分机制。

**实际规则（§1）**:
- 5维度加权：Momentum 35%（×1.75）/ 基本面 25%（×1.25）/ 催化剂 20%（×1.00）/ 估值 10%（×0.50）/ 资金流 10%（×0.50）
- 总分50分制，计算公式：M×1.75 + F×1.25 + C×1.00 + V×0.50 + Flow×0.50
- 等级映射：S≥42 / A 35-41 / B 28-34 / C 20-27 / 排除<20
- 每个维度有详细子项评分标准（见§1.2）
- S级额外5项强制检查（S1-S5全通过才允许建仓）
- Bear case在估值分项内做修正（≤15%+2分 / 25-35%-2分 / >35%-4分），不一票否决

**缺失代价**: Prompt的Bear case 4-tier（T1/T2/T3/T4）是A股规则的直接移植，不是美股系统。美股bear case门槛是35%（非20%），且bear case只影响估值子项得分，不一票否决。当前Prompt使用"T3 25-40%：事件驱动短期，不拿过催化日期"这类表述，与US_TRADING_SYSTEM_V4.md不一致。
