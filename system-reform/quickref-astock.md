## 开仓前必过（每笔）

- [ ] **L13三行检查**: ①理由不是"涨了" ②跌20%割不割（说不出→不买）③今天才知道→不买
- [ ] **L15 Thesis验证**: 搜"[标的名]最近为什么涨"→与thesis不一致→不买
- [ ] **thesis一句话能说清楚**（说不清=C级，上限8%）
- [ ] **R/R比 ≥ 1.5**（目标涨幅÷止损幅度）
- [ ] **止损价已设定**（S=-7% / A=15-20% / B=10-15% / C/T=7-10%）
- [ ] **操作后现金 ≥ 15%**（加仓前须 ≥ 20%）
- [ ] **单日新建仓 ≤ 3只**（加仓不计）
- [ ] **催化剂30天内有明确日期**（A级例外）
- [ ] **市场呼吸B1**: 沪市成交 <8000亿→不追新仓；>12000亿→减仓
- [ ] **催化剂在前还是后**: 在后面→可买（涨停也买）；已兑现→不买
- [ ] **小票先飞信号**: 同主题 ≥2只小票连板 = 龙头布局窗口开启
- [ ] **Bear case ≤ 40%**（>40%硬排除；25-40%仅A级+事件驱动+初始减半≤12%）

### S级额外检查（S1-S5全部YES才可建仓）

- [ ] S1: 供应链不可替代（同级供应商 ≤ 2家）
- [ ] S2: 催化剂30天内确认，收入传导 ≤ 2跳
- [ ] S3: 同链小票已先飞（信号已出现）
- [ ] S4: 入场点为底部放量/反转形态（非高位追涨）
- [ ] S5: 催化剂失败后bear case < 10%
- [ ] 当前S级持仓数 = 0（最多同时1只）
- [ ] **S级止损 -7%（硬止损）**，催化剂当天减至50%，3日内全退，持仓 ≤ 10个交易日

## 减仓/清仓前必过

### 先执行ABCD分类（分类前禁止卖出）

| 类型 | 判断标准 | 行动 |
|------|---------|------|
| A | 沪深300跌 ≥ 1.5%，无个股新闻 | Hold，不恐慌 |
| B | 指数稳，无新闻，下跌 <3% | Hold，观察1-2天 |
| C | 有新闻但不涉及core thesis | 减仓0-30%，24h内给结论 |
| C+ | 单一信源/传闻，无硬数据 | 24h核实，核实前不动；超48h视为D |
| D | 财报miss ≥ 2季/重大客户流失公告/政策文件否决 | 48h内清仓，无例外 |

### 卖出三问（L10-L12）

- [ ] **L10**: 想卖因为"别的涨了"？→ 是=关行情等1小时再决定
- [ ] **L11**: 催化剂还在前面就想减仓？→ 需给出D类或大幅超目标价理由
- [ ] **L12**: "如果这是新投资我会买吗？"→ 是=持有；不是=才是真卖出信号

### 止盈分批

- 达目标价70% → 卖1/4；达目标价 → 再卖1/4；超目标价10% → 再卖1/4；剩1/4 trailing stop

### 止损（命中立即执行，无商量）

- S级: -7%硬止损（不等ABCD分类）
- A级: 从高点trailing -12%（+15%后生效）
- B级: 从高点trailing -10%（+15%后生效）
- C/T级: -8%硬止损
- 单日暴跌 >15%: 无视等级，一次全退
- T级: 10交易日无catalyst落地→全退；C级: 20交易日无推进→全退

## 每日必查（每个A股session开盘前）

1. 读 `portfolio_state.json`（唯一真相源）
2. B1市场呼吸: 沪市成交额 + TMT占比（TMT>45%=不加仓）
3. 执行 `pending_actions.json`（A股部分）
4. 催化剂24h倒计时扫描

### 组合级风控

| 触发 | 行动 |
|------|------|
| 单日亏损 >-2.5% | 暂停当日剩余新建仓 |
| 单日亏损 >-4% | 暂停所有交易，全面review |
| 组合回撤 -10% | 全持仓review |
| 组合回撤 -15% | 强制减仓至50%总仓位（硬规则） |
| 周度亏损 >5% | 下周前3日禁止新建仓 |
| 周度亏损 >8% | 全仓review，B/C级重新评级 |

## Bear Case 4-tier（A股，12个月horizon）

| Tier | Downside | 处置 |
|------|---------|------|
| T1 绿灯 | ≤15% | 正常建仓，可到等级上限 |
| T2 黄灯 | 15-25% | 减半size，明确止损点 |
| T3 橙灯 | 25-40% | 仅A级conviction+事件驱动+初始≤12% |
| T4 红灯 | >40% | 硬排除，无例外 |

## A股T+1约束

- 大涨日不减仓（卖了当天买不回来）
- 止损决策须在盘中提前作出
- 持仓 ≤200股时，分批止盈不可执行→改为全仓一次性出场
