## §1 策略定位

### 1.1 这是什么策略

**Track B轮动交易策略**。专门捕捉板块热度外溢带来的小市值弹性收益。

核心逻辑：主板龙头涨完后，同板块小市值/北交所标的因资金溢出获得2-3倍弹性空间。这不是基本面研究（Track A），是动量接力。两者共存，不混用。

| | **Track A（不在本策略范围）** | **Track B（本策略）** |
|---|---|---|
| 买什么 | 板块龙头/瓶颈节点 | 同板块小市值接力票 |
| Thesis | 公司质量+业绩验证 | 板块热度+资金搬家+小市值弹性 |
| 评级上限 | 无上限(S级) | B+级（动量交易不给A） |
| 仓位 | conviction决定（最高50%） | 单只≤15%，北交所≤5% |
| 持有周期 | 中长期（催化剂兑现周期） | 短期（1-3周） |

来源：strategy.md §2.4b

### 1.2 适用市场环境

**必须在牛市/吸气期执行。**

吸气期判断标准（F20市场呼吸，每周一更新）：
- 北向资金净流入连续3日（月度配置盘方向）
- 中证1000跑赢沪深300（小票领涨 = 吸气信号）
- 全市场成交额 > 前20日均量 × 130%（量能确认）
- 主题/概念板块成交占比 > 消费/金融（风险偏好高）

呼气期不做Track B。无例外。

### 1.3 与strategy.md的关系

本文档补充并细化 strategy.md §2.4b。§2.4b是纲领，本文档是操作手册。

**优先级**：strategy.md的铁律（龙头跌>5%同日出场、Track B不升级为Track A等）优先于本文档任何规则。

### 1.4 市场环境硬开关（V1.1新增，审计G8/反茧房#5修复）

**策略自动关闭条件（任一触发=暂停所有入场，仅保留止损执行）:**
```
□ 沪深300过去20日收益率 < -8%（系统性下行）
□ 全市场成交额 < 1.5万亿连续5日（流动性枯竭）
□ F20判定呼气期（strategy.md前置条件）
□ 涨停家数连续5日 < 30家（情绪冰点）
```

**紧急关闭触发器（V1.2新增，回测bt_style_switch修复）：**
```
□ 当日跌停家数 > 50家 → 立即执行Track B清仓审查（不依赖"连续5天"条件）
  依据：2025年11月21日跌停71家，硬开关4条均未完整触发但实际已崩
□ 涨停家数48小时内降幅 > 40% → 自动触发§6.1 ★★★级评估
  依据：2025年11月17日85家→18日45家(降47%)是Q4真正清仓窗口
```

**重启条件（V1.2精确化，回测bt_924修复）**:
```
上述4条全部解除（连续2日以上）+ 有明确新催化剂（非传闻）→ 重新开启Track B
  2日满足=谨慎重启（半仓参与）
  3日满足=正常重启
  单日满足=不重启（防假信号）
  依据：924行情924当日单日突破门槛，但策略正确等到926确认3日后才重启
```

### 1.5 元规则：退出信号优先于入场信号（V1.1新增，审计C1修复；V1.2精确化）

**当入场信号和退出信号同日触发时，退出优先，无例外。**
- 龙头断板当日，即使回调-5%看似"加仓机会"，仍执行退出
- TMT拥挤度>48%当日，即使新催化剂出现，仍执行减仓
- 理由：交易系统中假阳性入场的代价远大于错过一次入场

**退出信号有效期（V1.2新增，回测bt_ai_may2026修复）：**
- 退出信号有效期 = **48小时（2个交易日）**
- 有效期内：即使有新催化剂，仍执行退出优先
- 有效期过后：新催化剂可重置入场判断
- 依据：2026年5月19日L6信号→5月22日Rubin BOM新催化剂，间隔3天。L6信号不应永久阻止新的高置信度催化剂入场

## §2 轮动分类与参数

### 2.1 五类轮动识别特征与参数

| 类型 | 驱动力 | 主升持续天数 | 仓位上限 | 止损规则 |
|------|--------|------------|---------|---------|
| **Type 1 事件冲击型** | 外部重大事件（技术突破/业绩爆雷） | 3-5天（第一波） | 正常仓位 | 无关公司蹭热点涨停=开始减仓；批量澄清公告当日全出 |
| **Type 2 产业催化型** | BOM拆解/订单公告/季报超预期 | 15-45天 | 正常仓位 | 主力净卖出>10亿当日减至25%；季报爆雷当日全出 |
| **Type 3 情绪扩散型** | 主线资金溢出（行情末段） | 3-7天 | **≤15%**(strategy.md Track B硬顶) | 5天内过去有2天炸板率>40%=全出；持仓第5天无新催化全出 |
| **Type 4 政策驱动型** | 方向性政策/重磅规划 | 15-75天（方向性）；1-3天（落地性） | 正常仓位，会议前3-5天减至半仓 | **方向性**:跟催化; **落地性**:发布次日减50%；利好不涨即日全出 |
| **Type 5 游资坐庄型** | 特定游资席位集中拉升 | 5-12天 | **5-10%**(信息滞后折价) | 核心席位净卖出T+1开盘全出；监管任何限制当日全出 |

来源：taxonomy_lifecycle.md §A

### 2.2 类型识别检查清单

**Type 2（最优，首选参与）**识别要素：
- 有具体数字：BOM价值量+X% / 订单金额 / 季报净利同比+X%
- 龙头有实际业务验证（非蹭热点）
- 2个以上子主题同日共振

**Type 3（次优，谨慎参与）**识别要素：
- 主线龙头已涨20-50%
- 新涨板块与主线叙事有联系但此前未炒作
- 在主线板块回调日出现首板

**Type 1（高风险，严格控仓）**识别要素：
- 事件受益方涨停 + 受损方同日暴跌（两极分化信号）
- 成交额 ≥ 2x 5日均值
- 快速判断"直接受益方"vs"蹭热点"

### 2.3 持仓天数上限

```
Type 1 追跟风股：5个交易日
Type 1 核心受益方：10个交易日
Type 2（产业催化）：跟催化剂验证节点；无新催化剂5天考虑减仓
Type 3（情绪扩散）：5个交易日绝对上限
Type 4（政策方向性）：跟政策落地节点；重大文件发布当日开始减仓
Type 4（政策+爆量科技）：按Type 4处理，不受Type 3的5天上限截断
Type 5（游资）：不超过12板；核心席位转卖立即出逃
```

> **V1.2修订(bt_924)**: 924行情中，政策驱动+科技爆量组合被错误分类为Type 3（情绪扩散），导致5天持仓上限过早截断收益。政策+爆量科技=Type 4，不是Type 3。判断标准：有政治局级催化+当日成交爆量（>前5日均量2倍）→ Type 4。

来源：taxonomy_lifecycle.md §D; bt_924.md §4.2

### 2.4 T1→T2升级检查（V1.2新增，回测bt_deepseek修复）

**问题**: V1.0/V1.1对Type 1事件使用40%拥挤度阈值，但DeepSeek回测显示：当T1事件进化为T2产业催化时（出现具体行业数字），40%阈值导致提前8天离场，错失并行科技+180%。

**T1→T2升级条件（缺一不可）：**
```
□ 初始事件为Type 1（外部冲击/技术突破）
□ 事件发生后D+3至D+7内，出现具体产业数字（BOM价值量/订单金额/季报验证）
□ 龙头从"蹭热点"转为"实际业务验证"（有营收/订单数据支撑）
□ 2个以上子主题出现同日共振
```

**升级后变化：**
- 拥挤度阈值从40%升级至48%（T2豁免生效，§4.2适用）
- 持仓天数上限从5天延长至"跟催化剂验证节点"
- 止损规则从"蹭热点涨停=减仓"变为"主力净卖出>10亿=减至25%"

**历史案例：**
- DeepSeek 2025年1月：D+0事件冲击(T1) → D+4出现AI算力BOM拆解+季报验证 → 升级为T2 → 行情持续至2月中（40天）
- 若仍用T1规则：TMT 40%触发退出在D+8，错过后续+180%(并行科技)和+80%(光模块二次高潮)

来源：bt_deepseek.md §4.1(D4)

### 3.2 入场检查清单（缺一不可）

**基础门槛（所有Track B必须通过）：**
```
□ 牛市/吸气期确认（F20市场呼吸，当周状态）
□ 板块龙头（>500亿）已涨幅>50%且仍创新高
  ★ Type 4政策转折豁免(V1.2): 政治局级催化+当日成交爆量(>2x 5日均量) → 龙头涨幅门槛降至20%
  依据：924行情龙头在政策发布当天才从底部启动，等涨幅>50%已错过整个主升浪
□ 目标票同板块，市值<500亿，业务相关
□ 龙头催化剂仍在前面（会议/季报未发布）
□ 全市场成交额 > 1.5万亿（流动性门槛）
□ 今日无A类下跌（大盘跌≥1.5%=A类，不入场）
```

**Track B北交所专用附加条件（全满足才做）：**
```
□ 主板龙头（>500亿）已连续2日涨幅>20%或连板≥2板
□ 北交所对应标的尚未启动（仍在平台期，无涨停）
□ 北交所标的市值>30亿（流动性门槛）
□ 目标标的日均成交>2000万
```

### 3.3 Track B专用入场时机

| 标的类型 | 最佳入场时间窗口 | 操作要点 |
|---------|--------------|---------|
| **主板龙头二线（>500亿）** | 龙头T+1当日 | 积极介入，封板概率40-60% |
| **主板中盘（100-500亿）** | 龙头T+1至T+2 | 次日跟进为主 |

> "买在涨停前，卖在连板第一天"。等到第2-3连板再入场，追板溢价极高，次日-30%概率大幅上升。

### 3.5 节假日特殊入场规则（V1.2新增，回测bt_deepseek修复）

**长假后一字板处理（春节/国庆/五一后首日）：**
```
场景：假期内发生重大催化剂，开盘一字板（无法买入）
  Day 1 一字板 → 不追，等开板
  Day 2 继续一字板 → 仍不追

  → Day3开板后追入者平均亏损>15%
  → 正确做法：等开板+确认资金接力（涨停>60家），而非追一字板
```

## §4 退出系统

### 4.1 S级退出信号（无条件执行，不分析）

```
Level S+（即刻清仓，T+0当日）：
1. 监管三连发（保证金上调+账户暂停+临停）→ 当日市价全出
   案例：2026年1月15日商业航天，次日全线一字跌停
2. 监管发声 + 核心游资大额净卖出叠加 → 不等次日，即刻清仓
3. 开盘前收到批量澄清公告 → 分级处理（V1.2细化，bt_deepseek修复）：
   ★ 盘前≥5家 → 开盘即出，无论昨日是否涨停
   ★ 盘中累计≥10家 → 尾盘减仓50%
   ★ 盘后≥5家 → 次日开盘即出
   ★ 3个交易日内累计≥15家 → 升级为S级（等同监管三连发），当日全清
   案例：2025年2月17日晚DeepSeek，20余家发减持+澄清，次日14只跌停

Level S（T+0或T+1）：
4. 核心游资席位净卖出额 ≥ 历史买入额50% → T+1开盘全出
5. 监管任何形式限制措施（保证金/账户限制/临停）→ 当日出逃，不过夜
6. 季报业绩爆雷（净利润下滑>50%）→ 当日全出
   案例：英维克Q1净利-82%，连续一字跌停
```

来源：taxonomy_lifecycle.md §D；signal_ranking.md §B3

### 4.2 拥挤度退出规则（V1.1修订：加Type 2豁免）

| 指标 | 阈值 | T1/T3/T4/T5操作 | **T2(产业催化)操作** |
|------|------|------|------|
| 板块成交占比 | >40% | 开始减仓 | **关注但不减**(有BOM/季报支撑时) |
| 板块成交占比 | >48% | 减仓>50% | **减仓至半仓**(非清仓) |
| 拥挤度修正值 | >2.0 | 减仓>50% | 减仓>50% |
| 拥挤度修正值 | >2.4 | 即日清仓 | 即日清仓(无豁免) |

**T2豁免原因**: 2025年2月TMT成交占比>40%持续8天行情不跌反涨(兴业证券: "40%隐形约束在本轮失效")。失效条件: 产业催化型+持续外部催化剂叠加时，拥挤度信号可能过早离场。但>2.4极端值仍无条件执行。

历史校准：
- 2023年4月6日：TMT占比50.24%（历史最高）→ 4月10日计算机单日-5.34%，蒸发5806亿（**非T2型，验证有效**）
- 2025年1月：46.5%→ DeepSeek行情顶点
- 2025年2月：>40%持续8天不崩 → **T2豁免案例**（AI算力有基本面支撑）

来源：signal_ranking.md §B1 + 反茧房审计#1

### 4.3 炸板率退出规则

| 炸板率 | 状态 | 动作 |
|--------|------|------|
| <30% | 启动期/加速期 | 正常持仓 |
| 30-50% | 分歧期开始 | 减仓，不新建 |
| >50% | 退潮期 | **当日减仓，不追低** |
| **>60%** | **崩溃期(V1.2)** | **★当日清仓，无例外** |
| 连续2天>40% | 分歧确认 | T+1开盘减仓 |

> **V1.2修订(bt_924)**: 924行情中9月30日炸板率69.8%是最敏感的顶部指标（10月8日大阴线前最明确的预警信号）。>50%时"减仓"不够，>60%必须当日清仓——这个阈值在924和DeepSeek行情中均无假信号。

**全市场涨停家数对应操作（V1.1统一标签，审计M2修复）：**
```
≥80家 + 连板≥5板 → 高潮期（谨慎，准备退出）
≥60家              → 主升期（积极参与Track B）
30-60家             → 启动期或分歧期（谨慎，需判断方向）
<30家               → 退潮期（不新建仓）
```

来源：taxonomy_lifecycle.md §B；signal_ranking.md §B9

### 4.4 龙头分化退出规则（Track B核心铁律）

**铁律（来源：strategy.md §2.4b）：**
> 板块龙头跌>5%，Track B当日全部出场，不等自己的止损线。

| 龙头信号 | 判断 | Track B动作 |
|---------|------|-----------|
| 龙头连板≥2板，全市场≥60家涨停 | 传导有效 | T+1积极入场二线 |
| 龙头1板或高开低走 | 传导弱 | 观察，不追 |
| 龙头断板（涨停后开板） | 出场信号 | 同日清北交所仓位 |
| 龙头跌>5% | 龙头见顶信号 | **Track B全部出场** |
| 龙头"换人"（原龙头断板，新龙头出现）| 行情末期 | 不追新龙头，旧仓清出 |
| 主线龙头成交额较峰值缩减>30% | 流动性萎缩 | 即日退出北交所仓位 |

> V1.1审计C2修复：taxonomy_lifecycle.md原写"龙头回撤7-10%退出"，与strategy.md的5%冲突。**统一为5%**——5%适用于Track B跟风票（更严格保护）；7-10%仅适用于Track A自身thesis判断。对北交所±30%涨跌停的标的，5%和10%之间差1-2天，这个差距是灾难性的。

**多龙头定义（V1.2新增，bt_ai_may2026修复）：**
```
主龙头 = 板块内市值最大 + 成交额最大的标的（如中际旭创/胜宏科技）
非主龙头 = 同板块其他涨停/高位标的

退出规则分级：
  主龙头 -5% → Track B无条件全出（铁律不变）
  非主龙头 -5% → 减仓50% + 立即检查：
    ① 主龙头是否同日走弱？是→全出
    ② 是否系统性板块回调（3只以上同日-5%）？是→全出
    ③ 仅个别非主龙头补跌？→ 减50%后观察1天
```
> 依据：2026年5月AI算力链中，光模块5只龙头走势分化（中际旭创仍创新高+天孚通信已回调-8%），若按"任一龙头-5%全出"执行会过早离场。

来源：timing_model.md §A3 + 审计C2修复; bt_ai_may2026.md §4.4

### 4.5 龙虎榜退出信号

| 信号 | 等级 | 动作 | 案例 |
|------|------|------|------|
| 核心游资席位从净买→净卖，卖出额≥买入50% | S | T+1开盘全出 | 陈小群2026年1月16日 |

**梭哈魔咒规则（来源：signal_ranking.md §B5）：**

```
3. 时间止损（≥8板减仓/≥10板清仓，详见§7.5）

```
> 龙虎榜对Type 5的核心价值：入场参考（确认游资建仓）> 退出信号（延迟致命）

## 附录：信号速查手册

### 入场三步检查（每次入场前过）

```
Step 1：市场环境（每日必做）
  涨停家数≥60家 → 主升期，可积极参与Track B
  涨停家数30-60家 → 谨慎，只看启动信号
  涨停家数<30家 → 不新建Track B

Step 2：龙头强度（T+0确认）
  龙头≥2连板 + 全市场≥60家 → 传导有效，T+1入场二线
  龙头1板 → 观察
  龙头断板 → 出场信号

Step 3：选择入场层级
  T+1：主板二线（>500亿、100-500亿）
  T+3~T+5：北交所建仓
  北交所持仓上限5天，单只仓位≤5%
```

### 退出优先级速查

```
立即出逃（T+0）：
  1. 监管限制措施（止损铁律1）
  2. 监管+核心游资大额卖出叠加
  3. 开盘前收到批量澄清公告

次日出逃（T+1）：
  4. 核心游资席位大额净卖出
  5. 利好发布后价格不涨
  6. 板块成交占比>48%

当日减仓（判断后）：
  7. 板块龙头跌>5%（Track B全出，铁律2）
  8. 机构龙虎榜净卖出+股价大涨
  9. 量化拥挤度>2.0

观察减仓（T+1~T+3）：
  10. 涨停家数从>60回落，炸板率>30%
  11. 主线成交量较峰值缩减>30%（北交所先出）
```
