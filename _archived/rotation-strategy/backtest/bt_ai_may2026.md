**规则触发分析（这是全报告最关键节点）**

§5.2规则原文：
```
光模块冲顶（主力净卖出>10亿）
    → 减光模块仓位50%，同日建封装/PCB
```

5月12日情况：

**⚠️ 问题2（核心）：冲顶触发条件不精确**

策略触发条件是"主力净卖出>10亿"，但：
- 5月12日旭创股价创新高（+6.54%），封装开始接棒（长电涨停）
- 实际卖出信号（净卖出13.73亿）晚了7天（5月19日）
- 期间5月12日→5月19日，旭创继续震荡（+4~+6%到-3~-6%）

规则设计者意图：破千元+封装同日涨停 = 情绪高点，应减光模块
但规则文字："净卖出>10亿" = 必须等到大资金明确离场才触发
**这两个信号错位了7天**，导致在规则的训练数据上，策略减仓光模块的时机存在歧义：
- 按字面规则（净卖出>10亿）：正确时机是5月19日
- 按意图（情绪高点）：应该是5月12-13日

### 5月19日 — 规则触发集中日（最高密度决策节点）

**市场状态（高置信度数据）**
- 光模块大幅调整：旭创-3~-4%，天孚-6%，新易盛-4%
- 旭创主力净卖出13.73亿（净卖出>10亿触发！）
- **江海股份涨停**（超级电容，L6末端扩散首板）
- 长电科技涨停（封装继续）

**规则触发逻辑（逐条）**

**触发1：§5.2 光模块冲顶规则**
```
光模块冲顶（主力净卖出>10亿）→ 减光模块仓位50%，同日建封装/PCB
```
- 净卖出13.73亿 > 10亿 → 触发
- 同日建封装（长电涨停，封装已有仓位）
- PCB：+0~+2%，温和 → 可逢低建仓PCB（但PCB主升浪在5月22日，还有3天）

**触发2：§4.4 龙头分化退出规则**
```
龙头跌>5% → Track B全部出场（铁律）
```
- 旭创跌幅：-3~-4%（天孚-6%，但天孚是"之一"不是"唯一龙头"）
- 旭创本身只跌-3~-4%，未到-5%阈值
- 天孚-6% → 天孚是光模块板块内的一个龙头
- **⚠️ 问题3（核心）：龙头跌>5%的认定标准歧义**
  - 光模块有3个龙头（旭创/新易盛/天孚）
  - 天孚跌-6% > 5%，旭创跌-3%，新易盛跌-4%
  - 策略未定义：板块内"任一龙头"跌>5%=触发，还是"主龙头"跌>5%才触发？
  - 如果是"任一龙头"：天孚-6%已触发，Track B全出
  - 如果是"主龙头"（旭创）：-3%未触发，不出场
  - 实际数据中旭创-3%（主龙头），天孚-6%（次龙头）
  - 这个歧义在"训练数据"本身也没有明确答案

**触发3：§5.1 L6末端扩散信号**
```
★★★信号含义：出现L6标的涨停=主线进入退潮/分歧信号
```
- 江海股份涨停 → L6末端扩散首板 → "主线进入退潮/分歧"
- 但同日封装（长电）涨停！
- 两个信号同时出现：L6（退潮信号）+ L4封装涨停（轮动继续信号）
- **§1.5 退出信号优先于入场信号** → 应该执行减仓
- 但具体减哪个仓位？减光模块（本来就要减）？减PCB？减封装（刚建）？

**5月19日操作结论（按V1.1）**
1. 减光模块仓位50%（净卖出>10亿触发）✓
2. PCB：是否减仓？L6出现→主线分歧信号→应减PCB
3. 封装（长电涨停）：按退出优先原则 + L6信号，新建仓位应暂停
4. 超级电容（江海）：L6末端品种，§7.1失败模式明确"追末端=失败模式"→ 不追

**⚠️ 规则执行结果**：策略在5月19日的操作是正确的——减光模块、观望PCB。但如果严格执行§1.5"退出优先"，应该减PCB仓位。然而5月22日PCB爆发+13%——如果减了PCB，就错过了3天后的最大行情。

**这正是"In-Sample不完美fit"的核心体现：L6信号触发减仓PCB的规则，在3天后被证明是假信号**。

### 5月22日操作结论（按V1.1规则执行顺序）

如果严格执行V1.1：
1. 检查退出信号：无（当日无龙头跌>5%，无净卖出>10亿）

**实际能否捕捉到Rubin BOM行情：取决于信号来源时间（盘前/盘中摩根士丹利报告），若开盘前已知 → 可以入场，若盘中才知道 → 追入溢价高**

策略§5.1注明："★★★出现L6标的涨停=主线进入退潮/分歧信号"

5月19日：正确（光模块开始退潮）
5月27日：正确（PCB集体回调，板块退潮）
5月28日：江海2连板，PCB消化（-1~+1%而非崩塌）——此时超级电容已不再只是"退潮信号"，已成为独立主线

**In-Sample数据显示L6信号的信号含义在第3次出现时已发生偏移**。策略将其定义为每次出现都是退潮信号，但实际数据表明：
- 第1次出现 = 退潮信号（正确）
- 第2次出现 = 退潮信号（正确）
- 第3次出现（连续）= 独立主线启动（信号含义改变）

规则未区分"首次L6出现"和"L6连续2-3天强势"两种情形的含义差异。

## 二、核心规则验证总结（5个验证问题）

### 验证1：§5.1 产业链传导Lag

**策略参考范围（V1.1注明为"参考，非硬规则"）：**

| 层级 | 策略Lag | 实测Lag | 误差 | 结论 |
|------|--------|---------|------|------|
| L1光模块（领先） | 0-3天 | 0天（节后即启动） | 无误差 | ✓ 准确 |
| L2 PCB（主升浪） | 滞后光模块4-8周 | 5月6日启动→5月22日主升（约16天=2-3周） | **短于策略估计4-8周** | ⚠️ 策略高估Lag |
| L2 PCB（日内短暂联动） | 1-2天 | 5月7日跟涨 | 1天 | ✓ 准确 |
| L5 CCL（滞后PCB主升） | 0-2周 | 5月22日（同日）| 0天 | ✓ 偏快 |
| L6末端扩散 | 主线高位回调当日 | 5月19日、5月27日（光模块/PCB回调当日）| 0天 | ✓ 准确 |

**关键发现：PCB主升浪Lag被高估**

策略文件写的是"滞后光模块启动：4-8周（主升浪级别）"，但实际：
- 光模块启动：5月6日
- PCB主升浪（Rubin BOM）：5月22日
- 实际Lag：**16个交易日 ≈ 3周**，显著短于策略的4-8周下限

5月7日PCB已出现首波涨停（鹏鼎涨停），策略正确识别这是"日内短暂联动（1-2天）"，但Rubin BOM的到来（L2主升浪）比"4-8周"快了4-5周。

策略自己的注脚已经说明"2018→2025轮动速度加速250%"，这里实测进一步确认Lag已大幅压缩。

**修正建议**：PCB主升浪Lag应改为1-4周（非4-8周），日内联动保留1-2天参考。

### 验证2：§5.2 "光模块冲顶→建PCB"规则（5月12-22日）

**策略规则：**
```
光模块冲顶（主力净卖出>10亿）
    → 减光模块仓位50%，同日建封装/PCB
```

**实测时序：**

| 日期 | 事件 | 规则触发？ |
|------|------|----------|
| 5月12日 | 旭创破千元（情绪高点），长电涨停（封装接棒） | 净卖出<10亿，**未触发** |
| 5月13日 | 生益科技涨停，市场整体新高 | 净卖出数据不详，**可能未触发** |
| 5月14-16日 | 光模块高位震荡，无主线 | 未触发 |
| 5月19日 | 旭创净卖出**13.73亿** | **触发！减光模块50%** |
| 5月20-21日 | 封装主线（长电连续涨停），PCB温和 | 封装已建，PCB观望 |
| 5月22日 | PCB爆发（Rubin BOM，胜宏+13.3%） | 新催化剂，重新入场PCB |

**规则执行结果**：
- 减光模块：5月19日执行，正确
- 建PCB：5月19日同日建仓（PCB+0~+2%，低调入场），5月22日Rubin BOM爆发，持仓获益
- 整体：规则执行结果良好

**但存在的不完美**：
1. 5月12日旭创破千元是更明显的情绪高点（市场赋予的"第10只千元股"叙事），但因净卖出未到10亿而未触发减仓 → 实际上5月12日减半光模块可获得更好的平均减仓价格
2. 净卖出>10亿这个阈值，在5月12日-18日这段窗口（约7天）造成"明显该减但规则不触发"的局面
3. **规则误差**：触发条件（净卖出>10亿）比情绪高点（破千元）滞后7天

**结论**：规则最终捕捉到了正确的时机，但在最佳减仓点（5月12日）存在7天延迟触发。

### 验证3：§4.4 5月19日旭创净卖出 → 减仓后PCB5月22日爆发

**这是策略中"过早退出"的核心检验。**

**完整逻辑链：**

1. 5月19日：旭创净卖出13.73亿 → §5.2触发减光模块50% ✓
2. 5月19日：L6江海涨停 → §5.1"主线进入退潮/分歧信号"
3. 5月19日：如果§1.5退出优先，则PCB也应减仓
4. 但5月22日：Rubin BOM爆发，PCB+9~+13%

**策略的行为：**

| 步骤 | 规则 | 操作 | 结果 |
|------|------|------|------|
| 5月19日减光模块 | §5.2触发 | 减50%光模块 | ✓ 正确 |
| 5月19日观望PCB | L6信号+§1.5 | 不新建PCB，已有仓位视情 | ✓/⚠️ 歧义 |
| 5月22日PCB新催化剂 | §3.4 D+0入场 | 重新入场PCB | ✓ 逻辑自洽 |

**关键判断**：策略并没有"过早退出"PCB，原因是：
- 5月19日仅是减光模块（§5.2），PCB持仓在没有"龙头跌>5%"信号的情况下未被强制清出
- PCB龙头（鹏鼎/沪电）在5月19日的跌幅是+0~+2%（温和），未触发§4.4

**结论**：策略在5月12-22日的行为是合理的：
1. 5月19日减光模块（正确）
2. PCB持仓保留（无退出信号，正确）
3. 5月22日Rubin BOM入场（可能已持有PCB+新建，正确）

"过早退出"的问题并不存在于此处——**真正的问题是5月27日鹏鼎-8%触发§4.4清仓，PCB是否应该全出**。

### 验证4：§5.4 5月27日资金搬家日判断

**策略文件给出的结论：**
```
案例：2026年5月27日 — PCB集体回调(-8%/-5%)但江海股份涨停
→ 全市场涨停数仍高（>60家）→ 判断为板块内切换，非退潮
```

**问题**：数据文件（deep_ai_infra_may2026.md）对5月27日的记录：
- 涨停个股：江海股份+10%（唯一明确涨停）
- 宏达电子：走强（未明确涨停）
- 其他：全文未提及5月27日全市场涨停家数

**核验结论**：策略文件中写入的"全市场涨停数仍高（>60家）"这一关键判断前提，**无法从数据文件直接验证**。这是策略直接将结论嵌入规则说明中，而非通过数据推导出来。

对回测的影响：
- 如果5月27日涨停家数确实>60家 → §5.4判断"板块内切换"正确，不全面退出PCB ✓
- 如果5月27日涨停家数在30-60家 → §5.4判断"整体分歧开始" → Track B全部减仓

**这意味着策略的5月27日案例在自身数据上存在一个无法核验的关键假设**。

### 验证5：L6末端扩散信号稳定性（江海股份3次涨停）

| 出现次数 | 日期 | 主线状态 | L6信号含义 | 准确性 |
|---------|------|---------|-----------|--------|
| 第1次 | 5月19日 | 光模块大跌（-3~-6%） | 退潮信号 | ✓ 准确 |
| 第2次 | 5月27日 | PCB集体回调（-3~-8%） | 退潮信号 | ✓ 准确（PCB短期见顶） |
| 第3次 | 5月28日 | PCB消化（-1~+1%） | 独立主线？ | ⚠️ 信号含义已变 |

**分析**：

第1次和第2次出现均在主线明显回调当日，符合策略"退潮信号"定义。

第3次（5月28日）：
- PCB仅温和消化（-1~+1%），非明显回调
- 江海2连板（1分钟封板），说明有专门资金在追这条主线
- 策略规则写的是"出现L6标的涨停=主线退潮/分歧"，但第3次出现时主线并非退潮

**根本问题**：策略把L6定义为一个无条件的退潮警报，但实际上它的信号含义与"是否同时伴随主线明显回调"高度相关：
- 主线当日明显回调 + L6涨停 = 退潮信号（准确）
- 主线温和消化 + L6连续强势 = L6自身成为新主线（信号失效）

## 三、策略规则缺陷汇总

按严重程度排序：

### 缺陷1（严重）：L6信号无有效期 + 不区分"首次vs连续"

**现有规则**："出现L6标的涨停=主线退潮/分歧信号"
**问题**：5月28日江海2连板时，主线（PCB）仅温和消化，L6已成为独立主线。
**修正方向**：
- L6首次出现 = 退潮预警信号（降仓50%，不全出）
- L6连续2-3天强势 = 独立主线启动，不再是退潮信号
- L6出现后72小时内无主线新催化剂 = 退潮确认（才全出）

---

### 缺陷2（严重）：§5.2"光模块冲顶"触发条件（净卖出>10亿）滞后情绪高点7天

**现有规则**：净卖出>10亿才触发减仓
**问题**：5月12日破千元是情绪高点，但净卖出信号在5月19日才出现，窗口错位7天
**修正方向**：增加辅助触发条件，满足任一即触发：
- 主力净卖出>10亿（现有）
- 股价创历史新高+涨幅<5%（缩量新高=冲顶特征）
- 同日次级领域（封装/PCB）出现涨停接力（接力出现=情绪转移）

---

### 缺陷3（重要）：§4.4"龙头跌>5%"未定义多龙头板块的认定规则

**现有规则**："板块龙头跌>5% → Track B全部出场"
**问题**：5月19日天孚-6%但旭创-3%；5月27日鹏鼎-8%但沪电-3%、胜宏-3%
- 同一板块多龙头时，是"任一跌>5%触发"还是"主龙头跌>5%才触发"？
- 鹏鼎跌>5%原因是个股业绩（一季报差），不是板块系统性风险
**修正方向**：
- 定义"主龙头"：市值最大或成交额最大的1只
- 主龙头跌>5% = 无条件全出（现有规则）
- 非主龙头跌>5% = 减仓50% + 检查是否系统性 + 次日决策

---

### 缺陷4（重要）：L6信号与新催化剂的优先级冲突（§1.5 vs §3.4）

**现有规则**：§1.5退出优先于入场；§3.4 BOM拆解D+0入场
**问题**：5月19日L6信号 → 退出PCB（按§1.5）；5月22日Rubin BOM → 入场PCB（按§3.4）
- L6信号是3天前的，新催化剂是今天的
- 策略未定义"退出信号的有效期"
**修正方向**：定义L6退出信号有效期为48小时（2个交易日）。有效期内，即使有新催化剂，仍执行§1.5退出优先。有效期过后，新催化剂可重置入场。

---

### 缺陷5（中等）：§5.4判断依赖"全市场涨停家数"这一实时数据，但策略文件写入了结论而非判断过程

**现有规则**：涨停数>60家=板块切换；<60家=整体退潮
**问题**：5月27日的判断在策略文件中直接写为结论（"涨停数仍高>60家"），但数据文件中无此支撑数据
**修正方向**：§5.4应明确要求"执行此判断前，先核实当日全市场涨停家数"，不允许从板块表现反推涨停数。这本是一个实时数据查询要求，不应该被脱水为结论直接写入规则示例。

---

### 缺陷6（轻微）：PCB主升浪Lag参数过时

**现有规则**：PCB主升浪滞后光模块4-8周
**实测**：约3周（16个交易日）
**修正方向**：调整为1-4周（基于本轮实测+轮动加速250%的历史趋势），并在每轮新行情开始时要求校准。

---

### 缺陷7（轻微）：算力租赁/CPO边际插入缺乏显式规则

**现有规则**：策略覆盖L1-L6传导链，但L1中CPO的处理方式仅有"信号"定义，无操作规则
**实测**：5月8-9日算力租赁/CPO成为边际最强方向（协创数据20%涨停），策略对此无明确指引
**修正方向**：在§5.1中增加"临时最强子方向（不在L1-L6主链中）"处理规则：参照Type 3情绪扩散，仓位≤8%，持有5天上限，主链复苏立即换回。
