## 六、规则修正建议（基于回测发现）

### 修正建议1：增加Type 1 → Type 2升级检查节点

**问题**：DeepSeek行情D4出现"国家超算平台接入"催化，此时仍按T1的严格拥挤度规则操作，导致踏空D4-D8最大主升浪。

**建议规则（V1.2新增）**：

```
§2.5 Type 1 → Type 2升级触发条件（V1.2建议）

当Type 1行情进行中（D1-D5），若同时满足：
  □ 出现具体数字驱动的产业催化（平台上线/接入合同/订单公告）
  □ 龙头有实际业务验证（非蹭热点）
  □ 成交量维持在前次均值200%以上
  → 将部分持仓类型升级为Type 2，允许持有至T2拥挤度阈值（>48%才减仓）
  → 升级仓位上限保持≤15%，但拥挤度豁免从T1的40%放宽到T2的48%
  → 整体行情仍设10个交易日绝对上限（不延长）

注：升级检查每日执行，不可事后追认（防止事后合理化追高）
```

**案例锚定**：DeepSeek D4（2月8日）国家超算平台接入 = Type 2升级信号满足。若2月8日执行升级，可持有超算中心仓位至2月12日（D8），而非在D2（拥挤度>40%）就开始减仓。

### 修正建议2：春节假期"信息缓冲"特殊处理规则

**问题**：春节7天假期使得"D+0尾盘入场"逻辑变成"假期后开盘 = 信息充分消化的D+0"，第一波爆发比普通事件更快（3天而非5天），策略如不提前准备，D1一字板密集会买不到好筹码。

**建议规则（V1.2新增）**：

```
§3.5 假期后首个交易日特殊入场规则（V1.2建议）

若重大催化事件发生在A股休市期间（春节/国庆/十一）：
  → 假期结束前最后2天：完成"受益方vs受损方"分类研究
  → 假期结束首日（D0后的第一个交易日）：
      ① 不追一字板（§3.3口诀不变）
      ② 对未涨停但方向正确的标的：D0首日尾盘小仓位建仓（≤5%）
      ③ 对已涨停的标的：等D+1回调/第二个交易日再入
  → 等效理解：假期=充分消化期，首个交易日=DeepSeek型的"D+0"，不是"D+1"
```

### 修正建议3：批量澄清公告分级处理

**问题**：澄清公告的时机不一定是"开盘前"（如2月12日盘中/盘后也出现了20家澄清），现行规则只覆盖"开盘前"收到的情况。

**建议规则（V1.2新增）**：

```
连续3天内累计≥15家澄清 → 触发S级退出（无论单日数量）

理由：盘中/盘后信号虽不如盘前严峻，但累积效应同样触发系统性退出风险。
```
