## A. 龙头→跟风传导时序模型

### A1. 龙头→二线最佳入场窗口

**核心规律：T+1是二线最佳入场，T+2是三线入场但风险大幅上升，T+3以后基本不入场。**

| 时间节点 | 二线封板概率 | 三线封板概率 | 操作建议 |
|---------|------------|------------|---------|
| T+0（龙头涨停当日）| 30-50%，但高开低走为主，真正封板率低 | 约10-15% | 观察，不追 |
| T+1（次日）| **40-60%**（最佳窗口）| 约20-30% | 积极介入二线 |
| T+2 | 20-35%（已显著衰减）| 约15-25% | 只做三线，高风险 |
| T+3+ | 若无新催化剂：<20%，开始分歧 | <10% | 不入场 |

**触发条件（必须同时满足才有效）：**
1. 龙头已连板≥2板（单板带动效果仅为2板的30-50%）
2. 全市场涨停家数≥60家（高潮期确认）
3. 龙头是"新龙头"而非反复炒过的老面孔（老龙头带动效果弱30-50%）

**案例验证：**

**案例1：AI算力链 5月6-9日（2026年5月，deep_ai_infra_may2026.md）**
- T+0：光模块（中际旭创等）启动，节后首日+8~+12%
- T+1（5月7日）：PCB（鹏鼎控股）接棒涨停，兴森科技跟涨，净流入9.3亿——验证T+1是二线最佳窗口
- T+2（5月8日）：算力租赁/CPO（更小弹性标的）才跟上，协创数据20%涨停——T+2三线入场，弹性品种
- T+3（5月9日）：分化开始，部分涨停、部分回调

**案例2：机器人主题 2025年1-2月（theme_ev_robot.md + theme_bse_smallcap.md）**
- 主板龙头（拓普集团、三花智控）1月初启动
- T+1~T+2：主板中盘（200亿以下）机器人跟随
- T+5至T+7：万达轴承（北交所）涨29.99%，此后连板——二线→北交所的传导在T+5-7完成

**案例3：PCB第二波 5月22日（deep_ai_infra_may2026.md）**
- 催化剂：摩根士丹利Rubin BOM报告（PCB价值量+233%）
- T+0（5月22日）：胜宏科技+13.3%（首涨）、鹏鼎控股一字涨停——外部催化型，非自然传导
- T+1（5月23日）：鹏鼎2连板，深南/沪电延续
- T+4（5月26日）：CCL上游（生益电子20%涨停）接力——上游材料滞后3-4天是规律

### A2. 不同市值档位传导速度差异

| 市值档位 | 从龙头启动到跟涨天数 | 机制 | 案例支撑 |
|---------|-----------------|------|---------|
| >500亿（主板次龙头）| T+1 | 机构资金覆盖，信息传导快，流动性好 | PCB鹏鼎T+1跟涨；封装长电T+1 |
| 100-500亿（主板中盘）| T+1至T+2 | 半机构半散户，次日跟进为主 | 生益科技5月7日（T+1）跟鹏鼎；兴森科技T+1 |
| <100亿（主板小盘）| T+2至T+3 | 散户主导，信息滞后；流动性中等 | 算力租赁标的T+2-3才启动 |
| 北交所（30-100亿）| T+3至T+7 | 流动性差，±30%涨跌停导致追入困难，需更高情绪累积 | 万达轴承：主板机器人龙头启动后约5-7个交易日（来源：theme_bse_smallcap.md第5.1节） |
| 北交所超小盘（<15亿）| T+5至T+14 | 极低流动性，需主板行情充分扩散后才有资金流入 | 浩淼科技：每次滞后主板启动1-2周（来源：theme_bse_smallcap.md第4.2节）|

**关键洞察：市值越小，传导越慢，但弹性越大**

北交所弹性倍数（相对主板龙头）：
- 北交所30亿+标的：主板龙头涨幅的1.5-2倍（万达轴承+317% vs 长盛轴承+433%，北交所弹性约为主板的74%，这里要注意主板龙头本身弹性也极高）
- 北交所超小盘<15亿：可以达到主板龙头涨幅的2-3倍，但流动性陷阱极深

### A3. "龙头涨但跟风不涨"= 分化信号→出场

**判断框架（来源：quant_rotation_rhythm.md第三节情绪周期四阶段）：**

| 信号 | 含义 | 操作 |
|------|------|------|
| 龙头涨停，二线仅小幅跟涨（<5%），无涨停 | 分歧初起，板块情绪不足 | 减仓已持有的跟风仓，不新建 |
| 龙头高位断板（涨停后开板），次日同板块集体低开 | 分歧期进入，高风险 | 清仓信号，不抄底 |
| 全市场涨停家数从60家+回落至30-60家，炸板率升至30-50% | 行情由高潮转分歧 | 清仓信号，不等第二天 |
| 龙头"换人"（原龙头断板，新龙头出现）| 行情末期，情绪在末路挣扎 | 不追新龙头，旧仓清出 |

**5月27日案例（AI算力链）：**
- PCB集体回调（鹏鼎-8%，深南-5%），但江海股份涨停——这是"主线休整+新副线接力"，不是整体分化。判断规则：如果全市场涨停数仍高（>60家），分化只是板块内切换，不是退潮。

## B. 二线→三线/北交所传导（Track B）

### B1. 主板启动后第几天是北交所最佳入场

**核心结论：主板龙头启动后第3-5个交易日是北交所的最佳入场窗口，第5-7天是最大涨幅期，第7-10天开始退潮。**

```
时序图：
主板龙头启动（T+0）
        ↓
主板二线普涨（T+1~T+2）
        ↓
北交所对应标的开始启动（T+3）← 最佳建仓点
        ↓
北交所最强上攻（T+5~T+7）← 最大涨幅区间
        ↓
主板龙头开始回调（T+7~T+10）
        ↓
北交所迅速跟跌（T+7~T+10）← 必须在此前退出
        ↓
北交所深度回调（-30%至-50%）
```

**数据支撑：**

| 案例 | 主板启动日 | 北交所启动日 | Lag天数 | 北交所最大涨幅 | 退出窗口 |
|------|---------|----------|--------|------------|---------|
| 人形机器人主题（2025年1-2月）| 2025-01-06（指数启动）| ~2025-01-13~15（万达轴承2月19日单日30%） | 3-7天 | +317.95%（1-3月累计）| 主板高位调整前 |
| DeepSeek AI算力（2025年2月）| 2025年1月底 | 2025年2月初（并行科技）| 2-5天 | +212.31%（2-3月区间）| 2025年3月初板块热度下降前 |
| 化工周期（2025年中）| 主板化工龙头涨完 | 北交所化工标的补涨 | 1-2周（7-10天）| 联合化学+438%（2025上半年最强）| 主板化工停止创新高前 |

**来源：theme_bse_smallcap.md 第5.1节（外溢规律定量表）**

### B2. 最大涨幅在哪个阶段

**北交所标的最大涨幅集中在：主板龙头启动后第3-7个交易日内的"情绪传导期"。**

具体机制：
1. 第3天：聪明钱开始在涨停前建仓（此时股价仍平稳）
2. 第4-5天：首板出现，散户发现→追入
3. 第5-7天：连板行情，30%涨跌停放大弹性，这是最大涨幅区间
4. 第7天后：若无新催化，主板资金撤回→北交所流动性急速萎缩→暴跌

**警示：北交所"买在涨停前，卖在连板第一天"是最佳实操策略。等到第2-3连板再入场，追板溢价极高，次日-30%概率大幅上升。**

### B3. 必须在龙头回调前几天退出

**强规律：必须在主板龙头第一次断板（涨停后开板回落）前1-2天退出北交所仓位，即主板启动后第6-8天退出（对应北交所涨幅最高峰）。**

**实操口诀（源自theme_bse_smallcap.md 第9.2节）：**
- 触发退出信号1：主板龙头开始回调或量能萎缩（日成交额较峰值缩减30%以上）
- 触发退出信号2：北交所标的单日涨幅>25%后次日高开（高开低走概率>60%）
- 强制退出：持仓超过5个交易日必须重新评估

**风险提示：** 北交所退出比主板更难——流动性萎缩时跌停板挂单难成交，因此必须在流动性充裕时（主板仍活跃时）提前退出，不等主板完全见顶。

## C. 大盘风格切换节奏

### C1. 科技→周期切换间隔

**数据汇总（来源：quant_rotation_rhythm.md 第九节 + theme_semiconductor.md + theme_ev_robot.md）：**

| 切换案例 | 上一主线 | 新主线 | 间隔（自上一主线峰值）| 触发信号 |
|---------|---------|--------|-----------------|---------|
| 2025年6月（AI降温→化工周期）| AI/科技（2025年1-3月北证AI行情）| 化工周期（北交所化工反转）| 约3个月 | 化工涨价周期确认，AI行情进入消化 |
| 2025年9月（机器人调整→横盘）| 机器人（6月底-9月18日+40%）| 无明显新主线（横盘调整）| — | 指数创新高后自然回调 |
| 2025年12月（机器人+智驾共振）| 机器人稳定主线 | 智驾接棒（L3准入12月16日）| 12天（同一主线内切换）| L3政策催化独立触发 |
| 2025年12月→2026年1月（科技强化）| AI+机器人 | AI+机器人继续（DeepSeek V3）| 持续，无切换 | 连续催化，主线不变 |
| 2026年5月（AI算力内部切换）| 光模块领涨 | PCB接棒 | 1-3天（子板块切换，非风格切换）| Rubin BOM报告，外部分析师催化 |

**规律提炼：**
- 大风格（科技vs周期vs消费）切换周期：**2-4个月**（基于2025年数据，置信度中）
- 主线内部子板块轮动：**1-5天**（日内到一周，2024年后已加速到1-3天）
- 同一科技板块连续多个波次：大主线可持续3-12个月（如机器人2025全年），中间有波段调整

**2024年后加速特征：**
- 2022年前：季度级别切换（3个月+）
- 2023年：月度级别切换（1-2个月）
- 2024年4月后：极端情况"一天一热点"，真正的季度级别主线只有DeepSeek级别事件才能维持（来源：quant_rotation_rhythm.md 第九节）

**信号矩阵（按可靠性排序）：**

|--------|------|------|---------|

| ★★★ | 资金切换方向 | 主力净卖出现主线龙头（如5月19日旭创净卖出13.73亿），同日新标的涨停 | 5月19日光模块主力出货→江海股份涨停，封装接棒 |

| ★★ | 估值极度拉伸 | 现主线PE达历史高位（如2025年3月北证PE68倍）| 2025年5月22日北证暴跌-6%，前一日创历史高点 |

**早期预警：** 当现主线龙头出现"量价背离"（价继续创新高但成交量不放大）时，切换往往在3-5天内发生。

### C3. "无人区"持续时间

**定义：** 现主线退潮但新主线尚未确立，全市场赚钱效应差，涨停家数<30家，跌停家数>30家。

| 市场状态 | 无人区持续时间 | 特征 | 案例 |
|---------|------------|------|------|
| 情绪型退潮（概念炒完）| **7-15个交易日** | 无新催化剂，等待政策/事件触发 | 北交所概念2023年11月：3-5天退潮后进入约10天整理 |
| 主题行情调整 | **15-30个交易日** | 板块有基本面支撑，震荡整理后可再启 | 机器人2025年9月18日高点→11月18日（调整约2个月）才反弹 |
| 大风格切换前 | **5-15个交易日** | 资金暂时流向高股息/防御板块 | 2025年9月调整期，创业板整体震荡 |

**关键发现：** 2024年后"无人区"已大幅压缩，资金不再等待——现主线见顶后，资金直接切入高股息（银行/公用）或已有热度的副线（如超级电容在PCB调整时立刻接力）。真正的"空白期"通常只有3-7天，除非整体市场情绪崩塌（如2025年5月22日北证暴跌）。

## D. 综合操作框架（整合A+B+C）

### 完整时序决策树

```
Step 1：确认市场情绪（每日必做）
  ├── 涨停家数 ≥ 60家 → 主升期，可积极参与
  ├── 涨停家数 30-60家 → 启动期或分歧期，谨慎参与
  └── 涨停家数 < 30家 → 退潮期，不新建仓

Step 2：确认龙头强度
  ├── 龙头 ≥ 2连板 + 全市场≥60家涨停 → 传导有效，看T+1二线
  ├── 龙头仅1板 + 市场一般 → 传导弱，观察
  └── 龙头断板（涨停后开板）→ 出场信号

Step 3：选择入场层级
  ├── T+1：主板二线（>500亿、100-500亿）
  ├── T+2：主板小盘（<100亿），风险升高
  ├── T+3~T+5：北交所标的建仓窗口（最优）
  └── T+5~T+7：北交所最强攻势，持续博弈

Step 4：设退出节点
  ├── 主板龙头第一次断板 → 当日清北交所仓
  ├── 北交所单日涨幅>25%后高开 → 次日清仓
  ├── 持仓超过5个交易日 → 强制评估是否继续
  └── 涨停家数回落至30-60家 → 减半仓

Step 5：风格切换监测（每周一更新）
  ├── 现主线龙头分化 + 资金切换信号出现 → 等待新主线
  ├── 无人区识别（涨停<30家 + 跌停>30家）→ 不轻易建仓
  └── 新催化剂出现（政策/事件）→ 重新从Step 1开始
```

### 最优Track B操作规程（北交所专用）

```
条件检查（缺一不可）：
  ✓ 主板龙头（>500亿）已连续2日涨幅>20%或连板2板以上
  ✓ 北交所对应标的尚未启动（仍在平台期，无涨停）
  ✓ 北交所标的市值>30亿（流动性门槛）
  ✓ 日均成交>2000万（来源：theme_bse_smallcap.md 9.2节）

建仓：主板启动后第3-5个交易日，以自然价格（非涨停板）建仓
仓位：单只北交所标的 ≤ 5%总组合
止损：单日-15%强制止损，不等反弹

退出优先级：
  1. 主板龙头成交量萎缩（较峰值缩减30%+）→ 即日退出
  2. 北交所标的涨幅>25%且次日高开 → 开盘即出
  3. 持仓第5个交易日 → 评估并默认减仓
```

## E. 数据局限与置信度声明

| 规律 | 置信度 | 说明 |
|------|--------|------|
| T+1二线最佳窗口 | 中-高 | 来自多案例验证（AI算力链、PCB链），有量化概率支撑（40-60%封板率），但无大样本回测 |
| 北交所Lag 3-7天 | 中 | 案例归纳（万达轴承、并行科技、浩淼科技），quant文件明确标注"非严格统计，案例归纳" |
| 风格切换2-4个月 | 中 | 仅2025年一年数据，样本量不足，历史各周期差异大 |
| 无人区7-15天 | 低-中 | 案例推断为主，2024年后已压缩，规律可能已失效 |
| 北交所弹性1.5-3倍 | 中 | 有具体案例数据，但个股差异极大（并行+212% vs 浩淼+370% vs 联合化学+438%） |
