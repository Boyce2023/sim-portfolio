# 板块轮动分类学与生命周期模型

> 编写日期：2026-05-28

> 用途：直接可用于轮动策略规则制定（止入点/止损点/持仓时长）

## A. 轮动类型分类（Taxonomy）

按驱动力分为5类。区分类型是决定入场时机、持仓时长和止损规则的前提。

---

### Type 1：事件冲击型（Event Shock）

**定义**：外部重大事件瞬间重定价板块估值，资金在1-3天内完成方向切换。
**典型案例**：DeepSeek R1发布（2025年2月），A股算力硬件↓ / AI应用↑；英维克一季报爆雷（2026年4月，净利-82%，连续跌停）

| 维度 | 数据 |
|------|------|
| 主升持续天数 | 3-5个交易日（第一波爆发）|
| 完整周期 | 约20个交易日（含接力分化期）|
| 龙头涨幅中位数 | 40-70%（第一周）；超算/云服务类可达100-180%（如并行科技4日+180%）|
| 跟风/小票弹性倍数 | 龙头1x → 二线跟风股 1.5-2x → 无关概念妖股 2-3x（杭钢股份9连板+112%，但只是骗局）|
| 典型催化剂类型 | 技术突破公告（DeepSeek R1）、业绩爆雷（英维克）、监管政策（商业航天保证金提升）|
| 失败率估算 | 追跟风股的失败率 > 60%（DeepSeek案例：249只成分股中大量妖股2周内崩塌）|

**规律**：事件冲击重定价的是叙事而非业绩。第一批涨的是"受益逻辑直接成立"的标的；第二批涌入的是"蹭热度"标的。蹭热度标的的峰值通常在事件后第7-12天，崩塌在第10-18天。

来源：deep_deepseek_daily.md §三（逐日时间线）、failure_cases.md §类型1-A/4-B

---

### Type 2：产业催化型（Industry Catalyst）

**定义**：有具体产品/订单/技术节点驱动的主题行情，有基本面支撑，可延续1-3个月。
**典型案例**：1.6T光模块放量（2025年7月启动，主升约3个月）；摩根士丹利Rubin BOM拆解（2026年5月22日，PCB单日最强催化）；长电科技CPO样品交付（2026年4月22日）

| 维度 | 数据 |
|------|------|
| 主升持续天数 | 15-45个交易日（视供给验证速度）|
| 完整周期 | 1-3个月，中间有1-2次回调 |
| 龙头涨幅中位数 | 100-400%（光模块龙头中际旭创底部+607%；PCB龙头胜宏科技年内+400%+）|
| 跟风/小票弹性倍数 | 上游材料/小票倍数 1.5-2.5x（南亚新材+722%、源杰科技+1075%，超过整机龙头）|
| 典型催化剂类型 | 季报超预期、大客户订单公告、BOM拆解报告（含具体价值量数据）、行业巨头供应链确认|
| 失败率估算 | 核心龙头失败率低（<20%）；追跟风股仍有 30-40% 回撤风险 |

**规律**：BOM/价值量类催化剂（如Rubin PCB价值量+233%）是产业催化型最强信号——数字具体、可验证、直接映射利润改善。纯叙事型（"受益AI"）寿命比数字催化型短50%。

来源：theme_ai_infra.md §三/五；deep_ai_infra_may2026.md §阶段四

---

### Type 3：情绪扩散型（Emotion Diffusion）

**定义**：主线行情进入高潮期后，资金从龙头溢出，向产业链上下游和弹性小票扩散，通常是主升浪末段。
**典型案例**：2026年5月AI算力链（光模块→PCB→封装→超级电容末尾扩散）；北交所概念2023年11月（凯华材料+456%，但持续仅13天）

| 维度 | 数据 |
|------|------|
| 主升持续天数 | 3-7个交易日（快速扩散快速退潮）|
| 完整周期 | 7-15个交易日 |
| 龙头涨幅中位数 | 已在高位，新入场者获利有限；主线龙头30-60%，扩散末端小票50-150%（但退潮时跌幅更大）|
| 跟风/小票弹性倍数 | 主线龙头1x → 上游材料/小票 2-3x（但持续时间极短）|
| 典型催化剂类型 | 无新催化剂（靠惯性扩散）；偶有小催化（如提价/定增消息）|
| 失败率估算 | 在高潮期末段追入扩散标的：失败率 > 50%；超过10天未出场的资金中，60-70%在退潮期被套 |

**规律**：情绪扩散型的识别信号是"同产业链内越来越边缘的标的开始涨停"。江海股份（超级电容）在5月AI算力链中多次充当情绪扩散型末端接力者——每次主线板块回调时出现涨停。

来源：quant_rotation_rhythm.md §三（四阶段量化指标）；deep_ai_infra_may2026.md §三/轮动路径

---

### Type 4：政策驱动型（Policy Catalyst）

**定义**：政府政策/监管/宏观表态触发的板块重估，持续时间取决于政策是"方向性"还是"落地性"。
**典型案例**：低空经济（924行情启动，持续约75天，万得低空经济指数+31.9%）；AI国产链政策5000亿支持（2025年12月，但"利好不涨"即出逃）

| 维度 | 数据 |
|------|------|
| 主升持续天数 | 方向性政策：15-75天；落地性政策（兑现）：1-3天（随后"买预期卖事实"回调）|
| 完整周期 | 1-4个月（方向性）；1-2周（落地性）|
| 龙头涨幅中位数 | 方向性：100-300%（宗申动力、万丰奥威年内+300-450%）；落地性：10-20%后回调 |
| 跟风/小票弹性倍数 | 政策方向性时期：跟风股弹性2-3x，但只有龙头能持续 |
| 典型催化剂类型 | 政治局会议定调、专项资金落地公告、重要会议纪要 |
| 失败率估算 | "利好出尽"失败率极高：如果政策已提前被定价（板块前期已涨20%+），落地后反弹概率<30% |

**规律**：A股"买预期卖事实"是铁律。重大会议前通常连涨3-5天，会议结束当天或次日是出货信号。但方向性政策（政策打开新赛道）不适用此规律——924行情是方向性政策，不是落地性政策。

来源：failure_cases.md §类型3-B/3-C；quant_rotation_rhythm.md §七（案例4，低空经济75天）

---

### Type 5：游资坐庄型（Hot Money Pump）

**定义**：特定游资席位集中资金主导的单标的/小板块快速拉升，依赖龙虎榜传播效应吸引跟风。
**典型案例**：商业航天板块（陈小群大连黄河路席位，2026年1月；连涨10天后单日全线跌停）；上纬新材借壳概念（12连板后崩塌）

| 维度 | 数据 |
|------|------|
| 主升持续天数 | 5-12个交易日（超过12天的游资主导行情极少维持）|
| 完整周期 | 8-15个交易日 |
| 龙头涨幅中位数 | 50-150%（快速拉升快速崩塌）|
| 跟风/小票弹性倍数 | 全板块几乎同步（游资同时拉多只），不存在明显的弹性梯度 |
| 典型催化剂类型 | 概念蹭热度（借壳/重组/网络事件）+ 游资席位买入+龙虎榜传播 |
| 失败率估算 | 游资席位买入次日胜率约57%（短期正期望），但60日后平均亏损超13%。跟风者10天后失败率 > 70% |

**规律**：游资撤退信号比想象中明显——龙虎榜同一席位单日从净买入切换为净卖出，且卖出金额超过历史买入额的50%，必须T+1开盘止损。监管任何形式介入（保证金/停牌/发声）= 游资撤退触发器，同日出逃。

来源：failure_cases.md §类型5-A/5-B（陈小群案例数据高置信度）

## B. 生命周期阶段模型

以下是每种轮动类型的阶段划分，含可观测指标（策略可直接引用）。

---

### 通用基础框架：5阶段情绪周期

（来源：quant_rotation_rhythm.md §三/八；历史案例验证）

| 阶段 | 持续天数 | 涨停家数 | 连板高度 | 炸板率 | 跌停家数 |
|------|---------|---------|---------|--------|---------|
| 启动期 | 1-3天 | 30-60家 | 2-3板为主 | ~30% | <10家 |
| 加速期 | 3-5天 | 50-70家 | 3-5板 | <30% | <10家 |
| 高潮期 | 1-3天 | >60-100家 | ≥5板 | <30%（但开始出现高位分歧）| <15家 |
| 分歧期 | 2-4天 | 30-60家 | 高位断板回落 | 30-50% | 10-30家 |
| 退潮期 | 3-7天 | <30家 | 连板消失 | >50% | >30家 |

**总计约12-22个交易日完成一个完整情绪周期。**

---

### Type 1：事件冲击型 生命周期

**启动信号**（T+0当日确认）：
- 事件标的在开盘30分钟内板块内3只以上个股涨停
- 与事件直接相关的叙事（客户/合作商/技术路径）明确指向特定子板块
- 成交额当日放大 2x+ 于前5日均值
- **特殊信号**：与事件"受损方"的板块同日暴跌（DeepSeek当日：光模块跌停 + AI应用涨停，两极分化 = 强烈的事件冲击信号）

**加速信号**（T+1至T+5）：
- Wind概念指数4日涨幅 ≥ 30%（DeepSeek指数4日+40%）
- 龙头进入3板以上连板
- 国家级媒体/机构开始覆盖该事件（增量信息持续输入）
- 资金从原主线流出（如DeepSeek期间TMT ETF大幅净流出，但概念指数持续上涨）

**高潮信号**（T+5至T+10）：
- 无关概念股开始蹭热点涨停（钢铁公司、主业无关公司出现在涨幅榜前列）
- 全市场涨停家数 > 80家
- 计算机/通信行业估值达历史90%+分位（DeepSeek期间：计算机98%分位）
- 大股东集中发布减持计划（DeepSeek：20+家在高峰期发减持）

**退出信号**（T+8至T+15）：
- 5家以上成分股发澄清公告（"与X无关"）→ 当日/次日立即止损
- 龙头连板股出现第一个"天地板"（杭钢股份、梦网科技）
- 所持标的业绩/客户无法支撑概念（无一手来源验证业务关联）
- 板块估值极度透支（无法算出合理PE）

**策略规则**：
```
入场条件：T+0当日基本面受益方（非蹭热度）+ 3只以上同板涨停确认
仓位：T+0当日满仓；T+3起减至半仓；T+5起减至1/4仓
止损：看到无关公司蹭热点涨停（高潮信号）即开始分批减仓
强制出场：批量澄清公告当晚/次日开盘全出
```

---

### Type 2：产业催化型 生命周期

**启动信号**：
- 产业链内有具体数字验证的催化剂（季报超预期、订单公告、BOM拆解）
- 龙头突破历史新高或关键价格位（如中际旭创破1000元、2026-05-12）
- 同日2个以上子主题共振（5月6日：光模块+PCB同日强势）
- 可量化的价值量/市占率变化（Rubin BOM：PCB价值量+233%，高置信度催化剂）

**加速信号**：
- 季报/中报超预期且下季度guidance向上
- 机构开始密集发布深度研报（目标价上调）
- 产业链上游/下游开始跟进（光模块涨 → PCB 4-8周后跟进）
- 成交额创历史新高（胜宏科技5月22日成交188亿，当日A股第一）

**高潮信号**：
- 龙头估值达到历史高位（中际旭创2025年10月达70xPE）
- 主力净卖出额超过前期净流入额（中际旭创5月19日主力净卖出13.73亿）
- 产业链内部开始分化（封装/上游材料接力，龙头开始滞涨）
- 全市场都在讨论该板块（散户FOMO顶峰）

**退出信号**：
- 主力龙头单日净卖出 > 10亿（中际旭创5月19日净卖出13.73亿为退出信号）
- 季报/中报不及预期（英维克一季报净利-82%：高潮后业绩回归是最致命退出信号）
- 龙头股价回撤超7-10%（超过正常波动范围）
- 上游材料/末尾小票开始超额涨幅（如江海股份接力时，光模块主线已进入退出窗口）

**策略规则**：
```
入场条件：具体数字催化剂（非模糊叙事）+ 龙头或产业链有实际业务验证
持仓时长：跟催化剂验证节点走（下次季报前）；无新催化剂5个交易日后考虑减仓
加仓时机：主线首次回调（-5%至-10%）有新催化剂支撑时加仓
止损：主力净卖出超10亿当日减至半仓；季报爆雷当日全出
```

---

### Type 3：情绪扩散型 生命周期

**启动信号**：
- 主线龙头已有20-50%涨幅，资金开始寻找"低位补涨"标的
- 产业链内此前未炒作过的子板块开始出现涨停（如AI算力链中超级电容出现在主线板块回调日）
- 扩散方向与主线有叙事关联但此前被忽略（超级电容 → Rubin平台供电架构）

**加速信号**：
- 主线板块出现大资金净卖出，扩散方向同日净流入
- 扩散方向个股连续2日涨停（江海股份3次涨停均在主线回调日）
- 媒体开始构建扩散方向与主线的故事（"下一个光模块是X"）

**高潮信号**：
- 扩散方向个股流通市值极小，成交额占比异常高
- 情绪口诀："高位死、中位死"——连板股在中位数价格开始跌停

**退出信号**：
- 扩散方向涨停炸板率 > 50%（主线分歧传导到末端）
- 主线重新启动（资金回流主线，末端扩散标的涨停封单瞬间消失）
- 持仓第5天无新催化剂注入

**策略规则**：
```
入场条件：主线回调日 + 扩散方向首板 + 与主线叙事有明确联系
仓位上限：50%（情绪末端，风险高）
持仓时长上限：5个交易日（超过5天不出：下一个退潮必在）
止损：炸板率连续2天 > 40% 或 主线重新领涨当日减仓
```

---

### Type 4：政策驱动型 生命周期

**启动信号**：
- 政治局/国务院/部委发布方向性新政策（非"落地验证"）
- 政策覆盖的赛道此前基本面已有支撑（低空经济有真实技术积累，非纯炒作）
- 板块此前处于低位或近期调整（未提前定价）

**加速信号**：
- 地方政府跟进（中央政策后1-2周，各省级政府跟进发文）
- 产业基金/国家队资金流入（龙头企业定增/融资出现国资背书）
- 板块内已有上市公司公告签订订单（叙事落地）

**高潮信号**：
- 重大政策文件正式发布（最终文件，非草稿/征求意见稿）
- 市场已用3-5天提前反应（重大会议前涨完了）
- 成交额在政策发布当日反而缩量（大资金已出货，散户在接盘）

**退出信号**：
- 政策文件正式发布当日（"买预期卖事实"触发点）
- "利好发布后当日涨幅 < 3%"（相比前期累积涨幅，反应力度弱 = 出货信号）
- 下一个政策节点不明确（缺乏续集催化剂）

**策略规则**：
```
入场时机：政策方向明确（1级信号）后第1天建仓；
          重大会议前3-5天进入警戒期（考虑减仓）
强制出场：会议/文件正式发布后次日开盘减仓50%
监控：下一个落地节点（地方跟进/项目开工）是否有明确日期
反例警告：如果利好发布后不涨，即日出逃（无论仓位多少）
```

---

### Type 5：游资坐庄型 生命周期

**启动信号**：
- 龙虎榜出现已知游资席位（如大连黄河路等）大额净买入
- 个股或小板块涨停但全市场情绪平淡（非全市场普涨，是游资单点启动）
- 概念逻辑比较薄弱但市场传播性强（借壳/重组/网红事件）

**加速信号**：
- 同一游资席位连续2-3天净买入
- 龙虎榜总榜前5席位合计买入 > 流通市值的 1-2%
- 散户开始大量讨论（社交媒体热度超过正常水平）

**高潮信号**：
- 连板天数达到8-12板（游资开始考虑出逃）
- 换手率每天超过20%（资金高速流转，主力在出货，散户在接盘）
- 监管开始关注（交易所问询、新闻媒体报道"高度关注"类文章出现）

**退出信号（★★★最高优先级）**：
- **核心游资席位同一天出现大额卖出（由净买→净卖）** → T+1开盘必出，无例外
- 任何监管限制措施出台（保证金/账户限制/个股临停）→ 当日出逃，不等次日
- 板块连涨超过10天无新基本面催化

**策略规则**：
```
入场条件：龙虎榜确认已知游资席位 + 概念有短期传播力 + 连板天数 ≤ 5板
仓位：10-20%（纯博弈，高风险）
持仓纪律：每日检查龙虎榜，一旦核心席位出现，T+1开盘清仓
绝对止损：监管任何限制措施 = 当日出逃，不过夜
不参与：连板超过10天才注意到的游资题材
```

### C.2 "死而复生"条件（已走完的子主题再激活）

**案例1：光模块（2025年1-2月死亡，2025年5月复活）**
- 死亡条件：DeepSeek发布后算力叙事受质疑，光模块连续暴跌（中际旭创-55%）
- 复活催化剂：H20禁令落地 → 国产替代逻辑取代进口算力叙事（外部不确定性消散）
- 复活时间：死亡约90天后
- 复活强度：比第一轮更强（最终创历史新高474元）
- **规律**：只有基本面叙事被"新的一手信息"重新支撑，才能复活；纯情绪退潮后靠情绪回暖的复活，通常幅度为第一轮的50-70%

**案例2：PCB（2025年8月高潮，2026年5月复活）**
- 死亡条件：2025年Q4行情结束后估值回落
- 复活催化剂：摩根士丹利Rubin BOM拆解（PCB价值量+233%，5月22日）
- **规律**：具体价值量数字 + 新一代平台切换 = 完整复活条件。没有具体数字只有叙事，复活不完整

**案例3：超级电容（多次零星激活）**
- 激活模式：每次在主线板块高位回调时出现，但单次持续仅3-7天
- 复活条件：主线资金溢出 + 本身有新叙事（Rubin供电架构）
- **规律**：超级电容属于"被动激活型"，不是独立主线，激活时间有规律性（主线回调日），但每次持续时间短（3-7天）

**"死而复生"通用条件清单**：

| 条件 | 必要/充分 | 说明 |
|------|---------|------|
| 叙事破坏被新信息修复 | 必要 | 仅靠时间流逝无法复活（光模块案例：H20禁令结束了不确定性） |
| 有具体数字驱动 | 充分（有则更强） | BOM拆解/订单金额/季报数字，比"AI受益"更有效 |
| 筹码结构改善 | 必要 | 高位接盘者已割肉，底部有新资金 |
| 外部大背景持续向好 | 必要 | 整体AI叙事不能崩塌（光模块复活依赖AI需求继续增长） |
| 新一代产品/平台切换 | 强充分 | 1.6T光模块、Rubin PCB均是平台切换触发复活 |
| 竞争格局改变 | 可选 | 对手出现问题也会触发（待验证，无清晰案例） |
