# 板块轮动失败案例库

> 更新时间：2026-05-28
> 用途：轮动策略止损依据 / 识别假突破 / 建立退出规则
> 核心逻辑：不知道什么情况下轮动会失败，策略就没有止损依据

| 时间 | 2025年2月5日～2月18日 |
| 持续天数 | 涨幅累积约13天，关键假突破在第14天崩塌 |
| 峰值涨幅 | 概念指数累计涨超33%；个股如梦网科技8连板后"天地板"，杭钢股份9连板后跌停 |

| 预警信号 | ① 前一晚多家公司发公告"澄清"（澄清=主力已在高位出货的掩护动作）；② 大股东/高管集中发减持计划（超20家）；③ 概念覆盖面异常宽泛（249只），正常主题不会覆盖这么多不相关行业 |

### 类型2：昙花一现型（涨2-3天后崩回起点）

#### 案例2-A：上纬新材"借壳"概念12连板后崩塌（2025年7月）

|------|------|
| 板块/个股 | 上纬新材（科创板，智元机器人借壳预期） |
| 时间 | 2025年7月9日开始 |
| 持续天数 | 涨停12次+多次十几厘米大涨，高位维持不超过5个交易日 |
| 峰值涨幅 | 从启动到峰值累计涨幅超200%（估算） |

| 失败性质 | 预期差炒作——逻辑成立概率极低但市场短暂信以为真 |
| 预警信号 | ① 借壳/重组逻辑在当前严监管环境下可实现性≈0；② 公司自身主业与概念严重不符；③ 连涨12板后换手率持续在高位（资金不断高位接棒） |

#### 案例5-B：龙虎榜游资轮番拉盘后集体撤退（普遍模式）

| 字段 | 内容 |
|------|------|
| 普遍现象 | 2025年龙虎榜总成交额5.56万亿元（同比+70%），刷新2015年高点 |
| 规律 | 游资席位买入次日胜率约57%，但后续（30-60日）平均亏损超13%——跟着游资打板是正期望，但不跑就变成负期望 |

## 二、失败模式分类与规律总结

### 分类1：假突破型
**核心特征**：放量单日涨停，后续无跟涨。量上来了但价维持不住。

| 识别要素 | 说明 |
|---------|------|
| 量价背离 | 成交量创近期新高，但股价次日跳空低开或高开低走 |
| 公司澄清 | 当晚或次日早间大量澄清公告（公司说"我跟这个概念没关系"=主力已出货） |
| 封板质量差 | 涨停封单中散单（<100手）占比高，主力单/大单占比低于50% |
| 概念宽度异常 | 概念股数量超过50只且覆盖无关行业（如AI包含钢铁公司） |

**止损规则**：次日开盘前收到大量澄清公告→无论昨日是否涨停，开盘即出。

---

### 分类2：昙花一现型
**核心特征**：连涨2-3天，有故事但无持续性，第3-5天突然踩踏。

| 识别要素 | 说明 |
|---------|------|
| 逻辑空心化 | 上涨理由无法量化（借壳预期、网红事件、段子） |
| 换手率过高 | 连板期间换手率每天超过20%（资金在高速流动而非积累） |
| 情绪热度峰值 | 板块出现在各大财经媒体头条时，通常是高点（散户最后涌入期） |
| 板块内分化 | 龙头创新高而同板块其他个股已不跟涨（龙头在消耗最后买盘） |

**止损规则**：连板3天后每日观察同板块其他个股是否出现涨停炸板，出现第一个炸板即减仓。

---

### 分类3：消息面反转型
**核心特征**：利好驱动上涨，但利好本身携带"寿命"或被新利空覆盖。

| 识别要素 | 说明 |
|---------|------|
| 利好不涨 | 重大利好发布后股价不涨甚至小跌（资金已提前定价，在利好发布时出货） |
| 利好已兑现 | 重大会议/政策落地后当天或次日走弱（"买预期卖事实"） |
| 期货领先指标 | 大宗商品相关板块，期货出现反转是股票最领先的预警 |
| 监管突然介入 | 证监会发声/保证金提升/账户限制（这是终结游戏的核武器） |

**止损规则**：监管任何形式的限制性动作→即日出逃，不等第二天。

---

### 分类4：跟风陷阱型
**核心特征**：龙头真涨，跟风股假涨；高位追跟风股必输。

| 识别要素 | 说明 |
|---------|------|
| 跟风股逻辑测试 | 无法说出该股与主题的具体业务关系（客户、合同、产品） |
| 涨幅与业务匹配度 | 跟风股涨幅已超龙头，但业务关联度远低于龙头 |
| 分化开始 | 龙头继续创新高，跟风股开始高开低走 |
| 入场时机 | 已在板块启动后第4天以上才入场，基本进入跟风陷阱区域 |

**止损规则**：看持仓标的是否是行业内有实际业务的核心公司；如果是跟风股，在龙头首次出现分化时即清仓。

### 分类5：游资撤退型
**核心特征**：龙虎榜游资单日净买出→板块踩踏。

| 识别要素 | 说明 |
|---------|------|
| 同席位从买→卖 | T+1或T+2龙虎榜出现昨日买入席位今日大额卖出 |
| 成交量萎缩 | 板块高位成交量开始缩量（接盘资金消失） |
| 板块持续天数 | 游资主导的板块通常维持5-10天，超过10天必看风险 |
| 监管升温 | 任何监管限制（融资比例/停牌）是游资撤退的触发器 |

**止损规则**：跟踪持仓板块的龙虎榜，发现核心买入席位出现大额卖出，T+1开盘必止损。

## 四、量化止损触发规则（给轮动策略用）

ROTATION_FAILURE_RULES:

    - 利好公告后次日跌幅 > 3%（利好不涨=出逃信号）

    - 5家以上成分股发澄清公告

    - 板块内龙头涨但超过60%个股高开低走
    - 大宗商品期货（对应板块）单日跌幅 > 5%
    
  next_day_check:

    - 成交量较昨日缩量超30%但价格未创新高

  position_size_cap:
    - 概念启动第1天：正常仓位
    - 第3天以后追入跟风股：仓位上限50%
    - 第5天以后追入任何成分股：仓位上限25%（高位风险窗口）
