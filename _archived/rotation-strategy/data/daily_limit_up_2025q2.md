# 2025年Q2 A股每日涨停板数据

**数据来源**：
- 涨停家数/跌停家数/连板数/涨停成交额：乐咕乐股网 API（legulegu.com，历史数据，验证来源）
- 沪指/创业板指涨跌幅：由乐咕乐股网收盘价数据计算（前一收盘为基准）
- 涨停板块Top3：东方财富复盘文章 + 新浪财经 + 淘股吧 + 搜索验证（置信度：中，部分标"缺失"）
- 全市场成交额：此API字段为涨停股成交额，非全市场；全市场成交额标"缺失"

**字段说明**：
- `涨停家数`：当日收盘时确认涨停数量（10%/20%）
- `跌停家数`：当日跌停数量（4月7日跌停1000+为关税风险暴发日，全市场近半数跌停）
- `连板数`：当日连续涨停股总数（含2板+）
- `涨停成交额`：当日所有涨停股成交额合计（亿元）
- `沪指/创业板指涨跌幅`：由收盘价推算，小数后2位
- `板块Top3`：当日主力涨停题材板块（搜索验证，无法确认标"缺失"）

## 数据质量说明

| 字段 | 数据来源 | 置信度 |
|------|---------|--------|
| 涨停家数 | 乐咕乐股API（uTotalStock字段） | 高 |
| 跌停家数 | 乐咕乐股API（dTotalStock字段） | 高（4月7日为特殊情况，显示1000+） |
| 连板数 | 乐咕乐股API（limitTimes字段） | 高 |
| 涨停成交额 | 乐咕乐股API（uTotalAmount字段，仅涨停股合计） | 高 |
| 沪指涨跌幅 | 由szzsClose字段计算，前一交易日为基准 | 高 |
| 创业板指涨跌幅 | 由cybzClose字段计算，前一交易日为基准 | 高 |
| 涨停板块Top3 | 东方财富/淘股吧/新浪财经复盘文章推断 | 中（部分为日期背景推断，非逐日精确查证） |

**全市场成交额**：乐咕乐股API未提供全市场每日成交额，未填写（需另行查询同花顺/Wind）。
