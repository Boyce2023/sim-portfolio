## §0 设计原则

Track B评级 ≠ Track A评级。

| | Track A（SABCD） | Track B（本文档） |
|---|---|---|
| 评级对象 | 公司质量 + Conviction | 轮动质量 + 时机 |
| 评级主轴 | thesis强度 + 催化剂确定性 | 信号等级 × 轮动类型 × 时间位置 |
| 最高等级 | S（50%） | **B+（15%）** — 硬顶不可突破 |
| 持有周期 | 中长期 | 1-3周 |
| 降级条件 | thesis证伪 | 龙头信号劣化 |

**2分钟完成原则**：所有维度均可用盘面可见数据即时判断，不需要深研。

## §1 评级维度（5维，共100分）

> 总分 → 等级映射见§2。每次入场前5维全过，约90秒完成。

### 维度一：入场信号等级（30分）

来源：ROTATION_STRATEGY_V1.md §3.1

| 信号等级 | 描述 | 得分 |
|---------|------|------|
| S级信号 | 机构+游资+外资三方共振买入（各超1.5亿） | 30 |
| A级信号 | 机构+游资共振 / 情绪启动期炸板率<30% / 龙头T+1跟风（≥2板+≥60家涨停） | 22 |
| B级信号 | 残差动量（行业+市值中性化）/ 均线能量Top 2+能量>0 | 14 |
| C级信号 | 北向月度配置盘变化 | 6 |
| 无有效信号 | 不符合任何入场信号定义 | 0 — 不入场 |

### 维度二：轮动类型质量（25分）

来源：ROTATION_STRATEGY_V1.md §2.1

| 类型 | 质量 | 描述 | 得分 |
|------|------|------|------|
| Type 2 产业催化型 | 最优 | 有具体数字（BOM/订单/季报）+ 龙头业务验证 + 2+子主题共振 | 25 |
| Type 4 政策方向性 | 高 | 政治局级催化 + 明确政策文件 + 非落地性 | 20 |
| Type 1 事件冲击型（核心受益方） | 中高 | 直接受益方业务可验证（非蹭热点）+ 成交≥2x 5日均量 | 15 |
| Type 3 情绪扩散型 | 中 | 主线龙头已涨20-50% + 此前未炒作 + 情绪传导 | 10 |
| Type 5 游资坐庄型 | 低 | 游资席位集中拉升，无机构跟进 | 5 |
| Type 1 蹭热点 / 无关公司 | 无效 | 无业务关联验证 | 0 — 不入场 |

**T1→T2升级加分**：初始T1标的，D+3至D+7出现具体产业数字 → 按T2重新打分（25分）。

### 维度三：产业链传导位置（20分）

来源：ROTATION_STRATEGY_V1.md §5.1

| 传导位置 | 描述 | 得分 |
|---------|------|------|
| L1 主线先行层 | 板块主线（光模块/CPO等），龙头领涨 | 20 |
| L2-L3 价值量跟进层 | PCB/覆铜板/散热，有明确BOM数字驱动 | 16 |
| L4-L5 技术升级/材料层 | 封装/CCL，滞后L1-L2但仍在产业链内 | 10 |
| L6 末端扩散层（首次出现） | 超级电容/元器件，主线退潮信号 | **0 — 不入场，仅退出参考** |
| L6（连续2-3天+独立产业催化） | L6已成独立主线，有业务验证 | 6（最多试仓≤8%） |
| 不在L1-L6的临时子方向 | 铜连接/Chiplet等偶发方向 | 4（上限≤8%，5天强出） |

|------------|---------|------|

| 退潮期 | 炸板率>60% / 涨停家数<30家 / 龙头断板 | 0 — 不入场 |

|---------|------|------|

## §2 等级体系：B+ / B / B- / C

> 入场总分 = 5维分数加总（满分100分）

### 2.1 等级判定表

| 等级 | 总分门槛 | 仓位上限 | ATR K值 | 最大硬止损 | 含义 |
|------|---------|---------|---------|-----------|------|
| **B+** | ≥75分 | **15%** | 2.0 | -12% | 高置信度轮动，积极参与 |
| **B** | 60-74分 | **12%** | 2.0 | -10% | 标准轮动，正常参与 |
| **B-** | 45-59分 | **10%** | 1.5 | -10% | 弱质量轮动，轻仓参与 |
| **C** | <45分 | **0%（观察池）** | — | — | 不入场，列观察 |

**强制降档条件（无论总分）**：
- 游资主导（无机构跟进）→ 上限压至B-（仓位≤10%）
- 北交所标的 → 单只≤5%（不受等级影响，硬约束）
- Type 5游资坐庄 → 最高B-（5-10%硬顶）
- T1蹭热点（无业务验证）→ C级排除

### 2.2 各等级详细判定标准

#### B+（≥75分）— 典型场景
```
必须同时满足：
□ 维度一：A级或S级信号（得分≥22）
□ 维度二：Type 2或Type 4（得分≥20）
□ 维度三：L1-L2（得分≥16）
□ 维度四：启动期或主升期早（得分≥12）
□ 维度五：流动性达标（得分≥8）

典型例子：
  - Rubin BOM拆解当日（A级信号+T2+L2+主升期）→ B+，建至15%
  - 政治局级催化首日（A级信号+T4+L1+启动期）→ B+，建至15%
```

#### B（60-74分）— 典型场景
```
典型例子：
  - 龙头T+1跟风，市场处于主升期中，PCB中盘（L2）
    → A级信号(22)+T2(25)+L2(16)+主升期早(12)+流动性(10) = 已上B+
    → 实际到B通常是信号等级降至A-类或时间位置偏晚

  - 情绪扩散型（T3）小票，启动期：
    → A级信号(22)+T3(10)+L3(10)+启动期(15)+流动性(10) = 67分 → B
```

#### B-（45-59分）— 典型场景
```
典型例子：
  - L5材料层（CCL）+主升中晚期+B级信号
    → B级信号(14)+T2(25)+L5(10)+主升中晚(6)+流动性(8) = 63分 → B
    （实际到B-通常是信号弱+时间偏晚）

  - Type 5游资坐庄，入场信号B级，启动期：
    → B级信号(14)+T5(5)+L1(20)+启动期(15)+流动性(10) = 64 → B，但强制压至B-（游资主导规则）
```

#### C观察池（<45分或触发一票否决）
```
一票否决（直接C，不算总分）：
  □ 维度四：退潮期（炸板率>60%/涨停<30家/龙头断板）
  □ 维度一：无有效入场信号
  □ Type 1蹭热点（无业务验证）
  □ L6首次出现（主线退潮信号）
  □ 北交所超小盘（<15亿）
```

## §3 仓位映射与建仓规则

### 3.1 建仓规则

| 等级 | 首次建仓 | 加仓条件 | 上限 | 分批逻辑 |
|------|---------|---------|------|---------|
| B+ | 5-8% | 信号再确认（龙头次日仍强） | 15% | 分2批：5-8%建仓，确认后补至上限 |
| B | 4-6% | 维持入场逻辑不变 | 12% | 分2批：4-6%建仓，次日确认加 |
| B- | 3-4% | 原则不加仓，持满周期 | 10% | 一次性建仓 |
| C | 0% | — | 0% | 仅列观察，不占名额 |

**北交所附加约束**（覆盖等级上限）：
- 任何等级北交所标的：单只≤5%，2批建：T+3-4建3%，T+4-5确认加至5%

**Type 5游资附加约束**：
- 最高B-等级：入场≤3板，单只≤10%（内地券商席位确认买），不过夜持有>12板

### 3.2 仓位上限优先级

```
硬约束（不可突破）：
  Track B总仓位 ≤ 40%（同时持有多个B级标的时）
  单只北交所 ≤ 5%
  单只Type 5游资 ≤ 10%
  Track B评级上限 = B+（单只≤15%）

软约束（按等级分配）：
  B+标的：≤15%
  B标的：≤12%
  B-标的：≤10%
```

## §4 升降级规则

### 4.1 升级条件（B- → B → B+）

| 触发事件 | 升级幅度 | 加仓动作 |
|---------|---------|---------|
| 持有第2日龙头继续涨停+全市场≥60家 | +1档（B-→B，B→B+） | 加仓至新等级上限 |
| T1→T2确认（D+3-7出现具体产业数字） | 重新评分，通常+1到+2档 | 按新等级加仓 |
| S级信号追加确认（三方共振） | 直接达到B+上限 | 加至15% |

**注意**：Track B不可升级为Track A。即使标的基本面很好，Track B评级最高B+，不突破。

### 4.2 降级条件（B+ → B → B- → C）

| 触发事件 | 降级幅度 | 减仓动作 |
|---------|---------|---------|
| 龙头次日高开低走（未封板） | -1档 | 减仓至新等级上限 |
| 板块拥挤度>40%（T1/T3/T4/T5） | -1档 | 减仓至新等级上限×50% |
| 炸板率升至30-50%（分歧期开始） | -1档 | 减仓 |
| 持仓超过类型天数上限无新催化 | 降至C | 清仓 |
| 龙头断板 | 直接C | 清仓（Track B铁律） |
| 龙头跌>5% | 直接C | **所有Track B当日全出（铁律，不分析）** |

### 4.3 强制C清仓一票否决（不经过降级流程，直接清仓）

```
以下任一触发 → 当日清仓（不等降级）：
  □ 龙头跌>5%（§4.4铁律）
  □ 监管限制措施
  □ 批量澄清公告（≥5家）
  □ 炸板率>60%（崩溃期）
  □ 持仓超过类型天数上限（无新催化剂豁免）
```

## §5 标号规范与Track A区分

### 5.1 标号规则

Track B评级标号格式：`TB-{等级}` 或直接写 `B+/B/B-/C`

区分规则：
- Track A标的写 `[A] 中际旭创 A+`（方括号A前缀）
- Track B标的写 `[B] 胜宏科技 B+`（方括号B前缀）
- portfolio_state.json中加字段 `"track": "A"` 或 `"track": "B"`

### 5.2 同标的同时有Track A和Track B评级时

**原则**：Track A和Track B是独立评级维度，不合并。

| 场景 | 处理规则 |
|------|---------|
| 标的同时入Track A和Track B 观察池 | Track A评级主导，不另建Track B仓位 |
| Track A标的出现轮动信号 | 在已有Track A仓位基础上，最多加仓至**Track A等级上限**，不受Track B B+上限约束 |
| Track A标的龙头断板 | Track A按自身thesis判断（5%铁律仍适用）；Track B跟风仓位当日全出 |
| 想把Track B标的升格为Track A | 需重新过Track A建仓五步法（§2.1 strategy.md），独立决策，不是自动升格 |

**同标的双仓情形（不推荐，但允许）**：
```
例：以600938.SH既有Track A仓位（A级，10%），又收到B+级轮动信号
→ 先检查总仓位是否超Track A等级上限（A级=25%）
→ 在A级25%上限内加仓，超过部分拒绝
→ 不另开Track B独立头寸（避免双重计算风险敞口）
```

## §6 评级速查表（2分钟完成）

```
Step 1（15秒）：市场环境预检
  涨停≥60家 + 无A类下跌 + 成交>1.5万亿 → 继续
  不满足 → 停止，不评级，不入场

Step 2（30秒）：5维打分
  D1: 信号等级  (S=30 / A=22 / B=14 / C=6 / 无=0)
  D2: 轮动类型  (T2=25 / T4=20 / T1核心=15 / T3=10 / T5=5 / 蹭热点=0)
  D3: 传导位置  (L1=20 / L2-L3=16或10 / L4-L5=10或6 / L6独立=6 / 首次L6=0)
  D4: 时间位置  (启动=15 / 主升早=12 / 主升中晚=6 / 高潮分歧=2 / 退潮=0)
  D5: 流动性    (主板≥10 / 北交所=8 / 小北交=4 / 超小=0；萎缩-5)

Step 3（15秒）：一票否决检查
  退潮期? / 无信号? / 蹭热点? / L6首次? / 超小北交? → C，停止

Step 4（15秒）：强制降档检查
  游资主导（无机构）? → 压至B-
  北交所? → 单只≤5%

Step 5（15秒）：等级判定
  ≥75 → B+（建至15%分2批）
  60-74 → B（建至12%分2批）
  45-59 → B-（建至10%一次性）
  <45 → C（观察池）
```
