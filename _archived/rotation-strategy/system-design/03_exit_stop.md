## 总览：Track B退出体系架构

Track A用ATR止损（公司个股层面）。Track B的锚点完全不同——板块龙头状态 + 板块生命周期阶段。

```
Track B退出信号来源（从高到低优先级）

Level 0: 元规则 ─── 退出信号永远优先于入场信号（同日触发时）
Level S: 铁律止损 ─── 监管措施 / 龙头-5% / 批量澄清公告
Level 1: 三层止损体系 ─── Layer1铁律 / Layer2信号 / Layer3时间
Level 2: 出场梯 ─── 板块生命周期阶段→分批退出比例
Level 3: 下跌分类 ─── A/B/C类大盘下跌→差异化应对
Level 4: 风控触发器 ─── Portfolio层面日亏/周亏/连续止损
```

**退出信号有效期（V1.2规则）**：退出信号触发后48小时（2个交易日）内，即使有新催化剂，仍执行退出优先。48小时后，新催化剂可重置入场判断。

## §1 TB三层止损体系

### Layer 1 — 铁律止损（无条件执行，不分析，不等）

**三条铁律**（任一触发=当日同日全出Track B所有相关仓位）：

| # | 信号 | 阈值 | 动作 | 来源 |
|---|------|------|------|------|
| L1-A | **主龙头单日跌幅** | **>5%** | Track B**全部**出场，当日市价 | strategy.md §2.4b铁律 |
| L1-B | **监管限制措施** | 任何形式（保证金上调/账户暂停/临停） | 当日市价全出，不过夜 | ROTATION_STRATEGY §4.1 |
| L1-C | **批量澄清公告** | 盘前≥5家 / 盘中累计≥10家 / 盘后≥5家 / 3日累计≥15家 | 开盘即出（盘前）/ 尾盘减50%（盘中）/ 次日开盘即出（盘后） | ROTATION_STRATEGY §4.1 |

**注意：主龙头定义**

```
主龙头 = 板块内【市值最大 + 成交额最大】的标的（如中际旭创/胜宏科技）
非主龙头 -5%：减仓50% + 立即检查三条：
  ① 主龙头是否同日走弱？ → 是=全出
  ② 系统性板块回调（≥3只同日-5%）？ → 是=全出
  ③ 仅个别非主龙头补跌？ → 减50%后观察1天
依据：2026年5月AI算力链光模块走势分化案例（bt_ai_may2026）
```

**执行铁律**：触发L1 → 当日执行 → 不分类 → 不犹豫 → 不等反弹。同日不回买。

### Layer 2 — 信号止损（分级减仓，按信号强度）

Layer 2信号按强度分S/A/B三级，触发后执行对应减仓动作：

#### 2A — S级信号止损（同S铁律，T+0当日）

| 信号 | 阈值 | 动作 |
|------|------|------|
| 核心游资席位净卖出 | ≥历史买入额50% | T+1开盘全出（延迟1天是因龙虎榜T+1公布） |
| 监管+核心游资叠加 | 双重确认 | T+0当日即刻清仓 |
| 季报业绩爆雷 | 净利润下滑>50% | 当日全出 |

#### 2B — A级信号止损（当日减仓50%以上）

| 信号 | 阈值 | T1/T3/T4/T5操作 | T2(产业催化)操作 |
|------|------|------|------|
| 拥挤度：板块成交占比 | >48% | 减仓>50% | 减至半仓（非清仓） |
| 拥挤度修正值 | >2.0 | 减仓>50% | 减仓>50% |
| 拥挤度修正值 | >2.4 | **即日清仓** | **即日清仓（无豁免）** |
| 炸板率 | >50% | 当日减仓，不追低 | 当日减仓，不追低 |
| 炸板率 | **>60%** | **★当日清仓，无例外** | **★当日清仓，无例外** |
| 机构专用席位净卖出+北向净卖出 | 双确认 | 当日减仓>50% | 当日减仓>50% |
| 梭哈魔咒（单笔>10亿游资+无机构） | 任意 | 次日开盘减仓 | 次日开盘减仓 |

#### 2C — B级信号止损（观察+减仓，T+1~T+3）

| 信号 | 阈值 | 动作 |
|------|------|------|
| 板块成交占比 | >40% | 开始减仓（T2豁免：关注但不减，有BOM/季报支撑时） |
| 涨停家数回落 | 从>60家→30-60家+炸板率升至30-50% | 清仓信号，T+1开始执行 |
| 连续2天炸板率 | >40% | T+1开盘减仓 |
| 主线成交量较峰值缩减 | >30% | 即日退出北交所仓位，主板仓位减50% |
| 游资全面撤离+无机构接盘 | 无数量限定 | T+1减仓 |
| 利好兑现后价格不涨 | 价格平或跌 | 次日开盘减仓（见光死） |

**T2豁免例外规则**（来自V1.1审计+2025年2月DeepSeek验证）：
- 拥挤度>40%：有BOM/季报支撑的T2产业催化→仅关注，不减
- 拥挤度>48%：T2也减至半仓（非清仓），>2.4仍全出
- 豁免目的：2025年2月TMT>40%持续8天行情不跌反涨，40%阈值过早离场

### Layer 3 — 时间止损（按轮动类型的持仓天数上限）

**没有新催化剂，时间本身就是退出信号。**

| 轮动类型 | 持仓天数上限 | 触发动作 | 特殊条件 |
|---------|-----------|---------|---------|
| Type 1 追跟风股 | **5个交易日** | 第5天收盘强制出场 | 发现时已连板>10天=不参与，不止损 |
| Type 1 核心受益方 | **10个交易日** | 第10天收盘强制出场 | T1→T2升级后延长至"跟催化剂节点" |
| Type 2 产业催化 | **跟催化剂验证节点** | 无新催化剂5天→考虑减仓；节点过去→3天内出场 | 季报/BOM发布=催化节点 |
| Type 3 情绪扩散 | **5个交易日绝对上限** | 第5天无新催化=全出，无例外 | 最严格，博末段 |
| Type 4 政策方向性 | **跟政策落地节点** | 重大文件发布当日开始减仓 | 政策+爆量科技=按Type 4非Type 3处理 |
| Type 4 政策落地性 | **1-3天** | 发布次日减50%；利好不涨=即日全出 | 落地=见光死 |
| Type 5 游资坐庄 | **≤12板** | 连板≥8板无条件减仓50%；连板≥10板全清 | 游资≠机构，时间是命 |
| 北交所标的 | **5天** | 第5天强制退出 | 流动性差，时间溢价高于主板 |
| 临时强势子方向（不在L1-L6主传导链） | **5个交易日** | 5天无新催化=全出，不等龙头-5% | 仓位≤8%，严格执行 |

**时间衰减触发标志**：
```
每个持仓进场时记录：
  入场日期：YYYY-MM-DD
  轮动类型：Type 1/2/3/4/5
  持仓上限到期：入场日 + 对应天数
  下一个催化验证节点：具体日期或"无"
→ 无催化剂节点 = 时间衰减按天数上限强制出场
→ 有催化剂节点 = 节点后3天内决定持有或出场
```

**T1→T2升级延长持仓（V1.2规则）**：
```
初始T1（5天上限）
→ D+3至D+7内出现具体产业数字（BOM价值量/订单金额/季报验证）
→ 升级为T2 → 持仓上限延长至"跟催化剂验证节点"
条件：①龙头从蹭热点→实际业务验证 ②≥2个子主题同日共振 ③有营收/订单数据支撑
依据：DeepSeek 2025年1月（D+4升级T2→行情持续40天）
```

## §2 TB出场梯 — 板块生命周期阶段模型

Track A用R倍数（公司自身盈利相对入场成本）。Track B不用R倍数——小票价格波动剧烈，R倍数要求2R才减仓会错过出场窗口。Track B用**板块生命周期阶段**作为出场节奏的坐标系。

### 板块生命周期四阶段

| 阶段 | 判定标准 | 炸板率 | 涨停家数 | 龙头状态 | Track B建议持仓比例 |
|------|---------|--------|---------|---------|----------------|
| **启动期** | 龙头首板或二板，全市场涨停30-60家 | <30% | 30-60家 | 首次涨停或连板中 | 40%（探针仓，观察传导） |
| **主升期** | 全市场≥60家涨停，龙头连板≥2，二线跟涨 | <30% | ≥60家 | 连续新高，无断板 | **100%（满仓，积极持有）** |
| **高潮期** | 涨停≥80家+连板≥5，末端扩散（L6）出现 | 30-50% | ≥80家 | 高位，量价背离迹象 | **50%（已开始减仓）** |
| **退潮期** | 龙头断板/量价背离/炸板率>50% | >50% | <60家 | 断板/回落/资金搬家 | **0%（全出）** |

### 分批退出时机

```
阶段切换判断：每个交易日收盘确认当前阶段（非实时）
动作规则：阶段降级时执行减仓，不等下一天

启动期 → 主升期（升级）：不减仓，按计划加至100%仓位
主升期 → 高潮期（降级第一步）：当日减仓50%（100%→50%）
  触发信号（满足任意一条）：
  ① 全市场涨停≥80家 + 连板≥5板（高潮信号）
  ② L6末端扩散首次出现（退潮预警）
  ③ 龙头量价背离：创新高+成交量较前5日萎缩>20%
  ④ 炸板率升入30-50%区间
  
高潮期 → 退潮期（降级第二步）：剩余50%全出
  触发信号（满足任意一条）：
  ① 龙头断板（Layer 1直接全出）
  ② 炸板率>50%
  ③ 主力净卖出>10亿（辅助：量价背离）
  ④ 涨停家数从>80回落至<60家 + 炸板率>30%
  ⑤ Layer 3时间止损到期

  Layer 1铁律触发：跳过出场梯，直接当日全出（不等分批）
```

### 出场梯决策树

```
每日收盘后判断流程（约60秒）：

1. Layer 1触发？
   是 → 当日全出，流程结束
   否 → 继续

2. Layer 2 S级触发？
   是 → 当日/次日全出，流程结束
   否 → 继续

3. 判断当前板块阶段：
   确定阶段标签（启动/主升/高潮/退潮）
   与昨日标签比较，是否降级？
   
4. 阶段降级？
   主升→高潮：减仓至50%
   高潮→退潮：全出剩余50%
   无降级：按现有仓位持有
   
5. Layer 3时间止损到期？
   是 → 触发清仓，不管阶段
   否 → 持有至下日
```

## §3 TB下跌分类 — 大盘跌了怎么处理

Track B持有的都是小票，弹性是双刃剑：大盘跌1%，小票可以跌3-5%。Track B的下跌分类必须比Track A更敏感。

### TB下跌分类（三类）

| 类型 | 判别标准 | 典型特征 | Track B动作 | 与Track A对比 |
|------|---------|---------|-----------|-------------|
| **A类 — 系统性** | 沪深300单日跌≥1.5% + Track B仓位无个股新闻 | 全线跌，无分化 | **即日减仓30-50%，不等观察** | Track A=Hold（A类down不动）；Track B更敏感 |
| **B类 — 板块内轮动** | 指数稳/微跌 + 主线板块回调 + 其他板块有新涨停 | 资金搬家信号，主线跌但市场活跃 | 检查资金搬家判断标准→决定是否换仓或持有 | Track A=Hold 1-2天观察 |
| **C类 — 个股补跌** | 指数稳 + 有公司新闻 + 跌<3% | 个股逻辑有变 | 分析是否改变龙头状态→视情况减仓 | Track A=评估减0-30%（相同逻辑） |

### A类下跌的Track B特殊处理

```
Track A: 沪深300跌≥1.5% = A类 = Hold（策略§3.3规则）
Track B: 沪深300跌≥1.5% = 立即判断是否触发主动减仓

原因：Track B持有小票（≤500亿），beta通常2-3x大盘
A类下跌时Track B仓位的实际损失是Track A的2-3倍
同样的大盘回撤，Track B止损价更容易被触发

A类下跌TB应对流程：
  Step 1: 确认是A类（沪深300≥1.5%，无个股新闻）
  Step 2: 检查龙头是否同日跌>5%
    是 → Layer 1铁律，全出
    否 → 继续
  Step 3: 检查F20状态
    呼气 → 减仓至30%，策略进入防御模式（§1.4关闭条件）
    中性 → 减仓30-50%，观察次日
    吸气 → 减仓20-30%，次日视量能决定是否恢复
  Step 4: 次日开盘看量能
    量能>1.5万亿+涨停>60家 → 逐步恢复仓位
    量能<1.5万亿或涨停<30家 → 维持减仓状态
```

### B类下跌的Track B处理（资金搬家判断）

```
判断是"板块内切换"还是"整体退潮"：

信号X：全市场涨停≥60家 + 新副线出现涨停
→ 判断：板块内切换，非退潮
→ 操作：旧仓减半，新方向按入场标准评估

信号Y：全市场涨停<60家 + 主线板块回调
→ 判断：整体分歧，近退潮
→ 操作：Track B全部减仓至50%，等待确认

参考：2026年5月27日 — PCB集体回调(-8%/-5%)但江海股份涨停
→ 全市场涨停数仍>60家 → 板块内切换，非退潮
（来源：ROTATION_STRATEGY §5.4）
```

### C类下跌的Track B处理（个股补跌）

```
C类特有问题：跟风票有新闻，但龙头无新闻
→ 首先判断：新闻是否影响龙头的催化剂？
  是 → 按Layer 1/2信号处理
  否 → 仅影响个股，不影响板块热度时，观察1天

Track B不对"个股利空"过度反应
  核心问题：这条新闻有没有改变【龙头状态】和【板块热度】？
  有 → 参考Layer 2处理
  没有 → 观察，不轻易减仓（跟风票本来就无独立逻辑）
```

### T1特有A类下跌豁免（审计C5）

```
事件冲击型（Type 1）的受益方入场，享有A类下跌豁免：
场景：DeepSeek当天，光模块跌停+AI应用涨停 → 受损方跌+受益方涨 → 大盘跌1.5%
→ A类下跌豁免不适用于T1受益方的入场决策
→ 仅Type 1特有，T2/T3/T4/T5不适用
```

## §4 TB风控触发器 — Portfolio层面

### 4.1 日级风控

| 触发条件 | 动作 | 与Track A协同 |
|---------|------|-------------|
| Track B当日亏损>3%（所有TB持仓加总） | 暂停当日所有Track B新建仓 | Track A不受影响 |
| 全Portfolio日亏>2.5% | 暂停所有新建仓（含TA/TB） | 策略§3.5规则 |
| 全Portfolio日亏>4% | 暂停所有新建仓+强制查D级 | 策略§3.5规则 |
| 单只Track B日亏>7% | 立即执行TB下跌分类 | 对应§3逻辑 |

### 4.2 周级风控

| 触发条件 | 动作 |
|---------|------|
| Track B当周亏损>5%（TB总仓位计算） | 下周前3天禁Track B新建仓 |
| 全Portfolio周亏>5% | 下周前3天禁新建（含TA/TB） |
| 全Portfolio周亏>8% | 全仓review，Track B减至50% |
| Track B连续3笔止损 | 强制暂停Track B 2个交易日，执行复盘 |

### 4.3 仓位层级风控

| 约束 | 上限 | 说明 |
|------|------|------|
| 单只Track B仓位 | ≤15%（北交所≤5%） | strategy.md §2.4b铁律 |
| Track B总仓位 | ≤30%（建议）/ ≤40%（绝对上限） | 牛市高峰期可临时接近上限，高潮期主动压至20% |
| Track B持仓只数 | ≤4只 | 超过4只 → 最弱conviction的止损掉 |
| 北交所标的总占比 | ≤10% | 流动性风险 |

```
Track B总仓位计算方式：
  所有标注为"Track B"的持仓市值 ÷ Portfolio总净值
  注意：同一标的如升级为Track A，从TB计数中移除
```

### 4.4 策略级开关

```
以下任一条件触发 → Track B全面停止新入场（执行已有止损）：
① 策略§1.4硬开关任一触发
  - 沪深300过去20日收益率 < -8%
  - 全市场成交额 < 1.5万亿连续5日
  - F20判定呼气期
  - 涨停家数连续5日 < 30家
  
② 紧急关闭触发（§1.4 V1.2新增）
  - 当日跌停家数 > 50家
  - 涨停家数48小时内降幅 > 40%

③ Portfolio Heat红线
  - Portfolio Heat > 15%（§3.4规则）
  → Track B仓位优先减仓（小票流动性好）
```

## §5 退出信号优先级体系

当多个退出信号同时触发，按以下顺序处理（不讨论，按序执行第一个触发的）：

### 完整优先级表（从高到低）

| 优先级 | 信号来源 | 信号名称 | 动作 | 执行时限 |
|--------|---------|---------|------|---------|
| **P0** | 元规则 | 退出信号与入场信号同日触发 | 退出优先，入场取消，无例外 | T+0 |
| **P1** | Layer 1铁律 | 主龙头跌>5% | Track B全部当日出场 | T+0当日 |
| **P1** | Layer 1铁律 | 监管限制措施 | 当日市价全出 | T+0立即 |
| **P1** | Layer 1铁律 | 批量澄清公告≥5家 | 按时机全出 | T+0盘前/盘中/次日 |
| **P2** | Layer 2 S级 | 监管+游资大额卖出叠加 | 即刻清仓 | T+0即刻 |

### 冲突处理规则

```
同时触发多个信号：执行优先级最高（P数字最小）的那个，其余暂不单独执行
例：P1 Layer 1铁律 + P7出场梯同时触发 → 执行P1（全出），P7自然完成

两个相同优先级信号：执行动作更严格的那个
例：P5"减仓>50%"和"减仓至半仓"同时触发 → 执行减仓>50%

退出信号 vs 入场信号（P0元规则）：
  退出信号触发，同日有吸引力的入场信号 → 退出优先，入场取消
  退出信号有效期48小时 → 48小时后新催化剂可重置入场判断
```

## §6 Track B与Track A止损的完全隔离

**核心原则：Track A和Track B是两套独立的止损系统，互不触发。**

### 隔离规则明细

| 场景 | Track A动作 | Track B动作 | 隔离原则 |
|------|-----------|-----------|---------|
| TB龙头跌>5% | **无影响** | Track B全出 | Layer 1只触发TB持仓 |
| TA持仓触及ATR止损 | 当日出场 | **无影响** | TA的ATR与TB无关 |
| 同板块TA+TB并存，龙头跌>5% | TA：按§3.3下跌分类处理（A类=Hold，有新闻=评估） | TB：立即全出 | 同板块不同规则 |
| Portfolio日亏>2.5% | 暂停新建仓 | 暂停新建仓 | 唯一共享的风控触发 |
| TA标的ATR收紧（棘轮） | 调整TA止损线 | **TB不调整** | 止损线各自独立 |

### 同板块TA+TB并存的处理

```
场景：AI算力链中，同时持有：
  Track A：中际旭创（龙头，15%仓位，A+级，ATR止损）
  Track B：某北交所小票（5%仓位，跟风票，龙头止损）

龙头（旭创）某日跌6%：
  TB动作：当日全出北交所小票（龙头-5%铁律）
  TA动作：判断旭创跌6%原因
    无新闻=A类下跌=Hold（TA规则§3.3）
    有新闻不改thesis=评估减仓（C类）
    thesis被证伪=3天清仓（D类）

→ 同一个龙头跌>5%，TB全出，TA按下跌分类处理
→ 两套规则不互相干扰
```

### 为什么隔离

Track A持有龙头，龙头跌5%可能是：
- A类系统性下跌（Hold）
- C类有新闻但thesis未变（评估减仓）
- 整体板块退潮信号

Track B持有的是跟风票，弹性2-3x，龙头跌5%对应跟风票可能跌10-15%。Track B没有独立thesis，thesis就是"龙头还在涨"。龙头一旦转向，跟风票立即失去全部支撑逻辑，没有任何理由等待分类。

## §7 操作备忘卡（每日收盘60秒检查）

```
Track B持仓日常检查流程：

【A. 铁律检查（任一触发=立即处理）】
□ 主龙头今日跌>5%？ → 全出
□ 有监管限制措施？ → 全出
□ 批量澄清公告≥5家？ → 开盘/次日全出

□ 全市场涨停家数：≥80/60/30/<30？
□ 炸板率：<30/30-50/>50/>60？
→ 根据答案判断当前板块阶段（启动/主升/高潮/退潮）

【C. 信号检查（分级减仓）】
□ 板块成交占比>40%（非T2）/ >48%（含T2）？
□ 核心游资席位出现大额净卖出？
□ 主线成交量较峰值缩减>30%？
→ 对应Layer 2分级操作

【D. 时间检查（防止抱尸）】
□ 每只TB持仓：今日是入场后第几天？
□ 对应轮动类型的持仓上限是多少天？
□ 距到期还有几天？
□ 是否有新催化剂延续持仓？

【E. 板块阶段切换判断（出场梯触发）】
□ 今日板块阶段vs昨日是否降级？
□ 降级了 → 执行对应减仓（主升→高潮减50%，高潮→退潮全出）
□ 没降级 → 按现有仓位持有
```
