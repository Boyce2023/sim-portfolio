# Track B PlayBook — 板块轮动赢家模式库 v1.0

> 编制日期：2026-05-28
> 数据来源：5场回测（bt_deepseek / bt_924 / bt_ai_may2026 / bt_hot_money / bt_style_switch）
> 状态说明：validated = ≥2次历史实例+正收益；observation = 1次实例或待验证
> Claude分析意见，非用户投资建议。

---

## 系统说明

### PlayBook的定位

PlayBook不是选股清单，是**决策加速器**。每次建仓前做一次Pattern匹配（<2分钟），匹配成功则：
1. 规则直接执行，不再重复分析
2. 仓位可按Pattern历史表现调整
3. 出场条件预设，不在盘中临时决策

未匹配到任何Pattern时：仍可按策略基础规则操作，但不享受仓位加成。

### Pattern ID体系

- `EP-XXX`：Event Patterns（事件驱动）
- `CP-XXX`：Chain Patterns（产业链传导）
- `PP-XXX`：Policy Patterns（政策驱动）
- `HP-XXX`：Hot Money Patterns（游资驱动）
- `SS-XXX`：Style Switch Patterns（风格切换）
- `AP-XXX`：Anti-Patterns（反面模式，陷阱）

## 第一部分：Event Patterns（事件驱动型）

---

### EP-001: 技术突破两极分化入场

**状态：VALIDATED（2实例：DeepSeek 2025-01-27 / 商业航天 2026-01）**

**模式名称**：重大技术事件 → 受益方T+0尾盘入场

**核心逻辑**：
外部技术事件（AI突破/产业颠覆性发布）同日造成"受益方大涨+受损方暴跌"的两极分化结构。两极分化确认后，受益方的初始波动是可以把握的，但超过5天的跟风仓位必须出清。

**触发条件（全部满足）**：
1. 当日受益方涨停≥3只 + 受损方同日下跌≥5%（两极分化信号）
2. 全市场成交额 ≥ 前5日均值×2（爆量确认市场震惊程度）
3. 受益方有可验证的业务关联（有合同/API接入/公告，不是"名字沾边"）
4. 事件发布≤3个交易日（越早入场成本越低）

**入场规则**：
- 时机：当日尾盘14:45-14:55，确认板块方向未反转后建仓
- 如遇春节/长假后首日：假期当最后2天完成"受益方vs受损方"分类，首个交易日相当于D+0
- 一字板密集的标的：等D+1或D+2回调入场（不追一字板）
- 优先级：P1（直接受益，有业务验证）> P2（产业受益，生态供应商）> P3（叙事受益，无直接合同）

**出场规则**（按优先级）：
- S+级退出（当日执行）：批量澄清/减持公告（盘前≥5家、盘中≥10家、盘后≥5家）
- A级退出（当日执行）：龙头板块指数单日跌幅 ≥5%
- B级退出（减仓50%）：机构ETF净流出 + 主力龙股炸板（涨停后开板）
- 时间止损：P3跟风仓5个交易日强制清仓；P1/P2核心仓10个交易日强制清仓

**仓位规则**：
- 单只 ≤ 15%（Track B硬顶）
- P3跟风票：≤8%
- 严格排除受损方（不做价格被打压的标的）

**历史表现**（从回测估算，非精确数据）：
- DeepSeek案例：P1/P2标的，D0至D8区间 +70%~+180%（超算/AI应用龙头）
- DeepSeek案例：S+退出规则（2月17日晚）完整规避2月18日14只跌停
- 商业航天案例：严格执行S+（1月15日监管三连发）→ 5%仓位×12%区间收益 = +0.6%组合贡献

**失败场景（Pattern不work的条件）**：
1. 事件发生在假期结束后已经过了3个交易日（成本已高，性价比下降）
2. 受益方没有任何一手来源验证业务关联（全是叙事，无合同/API）
3. 板块已出现≥5只涨停+2只明显蹭热点的"名字沾边"股 = 末端情绪，不入

**Type 1 → Type 2升级检查（DeepSeek教训）**：
若D3-D5出现"具体数字驱动的产业催化"（如平台接入合同/政府平台上线），且成交量维持前均量×200%以上，可将部分核心仓位升级为Type 2，拥挤度容忍度从40%放宽至48%，持仓上限仍10个交易日。

### EP-002: 假期信息缓冲入场

**状态：VALIDATED（1强实例：DeepSeek春节 2025-01 / 924国庆 2024-09）**

**模式名称**：重大事件跨越长假 → 开市首日等同D+0

**核心逻辑**：
重大催化事件发生在A股休市期间（春节/国庆），市场参与者有充分时间研究消化，假期结束首个交易日的爆发速度远快于普通事件（普通事件5天波次压缩至1-3天）。

**触发条件**：
1. 重大催化在A股休市期间发布（法定节假日≥3天）
2. 假期结束前2天已完成标的分类（受益方清单）
3. 首个交易日成交额 > 历史均量×150%（确认爆量）

**入场规则**：
- 假期最后2天：完成分类研究，准备建仓清单
- 首个交易日尾盘：对未涨停但方向正确的标的建仓（≤5%试仓）
- 第2个交易日：量能确认后加仓至目标仓位
- 已涨停的标的等D+1回调再入

**出场规则**：同EP-001

**失败场景**：
1. 假期内消息已在港股/美股充分price in → A股开市首日可能直接高开低走
2. 假期内出现反向消息（监管/政策否定）→ 取消入场计划

### CP-001: 光模块先行 → PCB主升接力

**状态：VALIDATED（实例：2026年5月AI算力链，含Rubin BOM催化）**

**模式名称**：算力链传导 — 光模块冲顶→PCB主升浪

**核心逻辑**：
AI算力链的资金传导存在可预测的层级顺序：光模块（L1）先行启动 → PCB/封装（L2/L3/L4）轮流接棒。历史Lag在2024年后已大幅压缩，实测2026年5月PCB主升浪在光模块启动后16个交易日触发（策略原预期4-8周已被证伪）。

**触发条件**：
1. 光模块龙头（旭创/新易盛）单日涨幅 ≥ 6% + 成交量放大
2. PCB龙头（鹏鼎/沪电/深南）跟涨1-2天（短暂联动）
3. 上游BOM催化剂出现（摩根士丹利/花旗等机构BOM拆解报告，或券商季度报告）

**入场规则**：
- 光模块启动当日（L1触发）：建仓PCB龙头（T+1入场，等L1启动确认）
- 仓位：PCB中盘100-500亿，≤15%；小盘<100亿，T+2至T+3入场，≤10%
- BOM催化剂出现当日（如Rubin报告）：D+0尾盘加仓PCB，视为新的入场机会
- CCL（生益科技类）：PCB大涨当日D+0至D+2跟进，≤10%

**出场规则**：
- 光模块主力净卖出 ≥10亿 → 减光模块50%，观察PCB是否维持（此时PCB可能即将爆发主升浪）
- L6末端扩散信号出现（超级电容/冷却液等末端品种首板）→ 减仓50%，第2次出现→清仓
- L6连续2-3天强势（变为独立主线而非退潮信号）→ 维持持仓，等真正的退出信号
- 龙头鹏鼎等跌 ≥5%：若原因是个股业绩（非板块逻辑证伪），看Rubin BOM催化是否仍有效；若催化仍有效，减仓但不全清
- 时间止损：无新BOM催化，持有5个交易日后重新评估是否继续

**仓位规则**：
- 光模块 ≤ 15%（龙头T+1）
- PCB ≤ 15%（分散在2-3只）
- CCL ≤ 10%
- 总产业链仓位 ≤ 35%（三层叠仓限制）

**历史表现**（2026年5月回测）：
- 5月6日光模块启动 → 5月22日Rubin BOM → PCB：鹏鼎涨停，胜宏+13.3%，深南+9%
- 生益科技（CCL）：5月26日涨停，5月7日持仓者获完整收益

**Lag参考（已更新）**：
- L1光模块领先：D+0启动（基准）
- L2 PCB短暂联动：D+1至D+2
- L2 PCB主升浪：光模块启动后1-4周（已从4-8周压缩）
- L5 CCL：PCB主升同日或D+2内跟进
- L6末端扩散（超级电容等）：主线回调当日出现 = 退潮信号

**失败场景**：
1. 光模块启动是政治/地缘因素（非算力需求），则传导链不成立
2. BOM催化来源是二手/转载，而非原始机构研究报告
3. 光模块Lag延长：若光模块启动后6周PCB主升浪仍未出现，信号过期
4. 个股龙头业绩爆雷（如鹏鼎一季报-31.85%）：触发§4.4但板块其他标的（沪电/深南/胜宏）未崩 = 个股分化，非系统退出

### CP-002: 龙头先飞 → 小票接棒

**状态：VALIDATED（实例：924行情科技链 2024-09 / DeepSeek产业链 2025-02）**

**模式名称**：板块龙头大涨 → T+3至T+5小市值建仓窗口

**核心逻辑**：
大市值龙头（>500亿）启动后，资金有1-5天的传导期才会流向中小市值标的。北交所最慢，在主板T+3至T+5才是最佳建仓窗口。

**触发条件**：
1. 板块龙头（>500亿）单日涨幅 ≥ 8% 或涨停
2. 中证1000 > 沪深300（小票领涨，F20吸气信号）
3. 全市场成交额 > 1.5万亿

**入场规则**：
- 龙头启动当日（T+0）：建仓龙头本身，≤15%
- T+1至T+2：建仓中盘（100-500亿），≤15%
- T+3至T+5：建仓小盘（<100亿）和北交所，≤10%
- 超过T+5不追（北交所尤其严格）

**出场规则**：
- 龙头跌 >5% → 当日全出（铁律，不看个股）
- 炸板率 >50% → 当日减仓
- 北交所：主板龙头跌>5%，同日清北交所（北交所±30%不等龙虎榜）
- 时间止损：北交所持仓≤5天强制复核

**历史表现**：
- 924行情：东方财富/券商启动→科技中小市值T+3建仓 → 北证50 927→1008 +53.1%
- DeepSeek：昇腾/信创龙头→小市值AI应用 T+1至T+2建仓

**失败场景**：
1. 龙头涨停但行业属于受损方（如DeepSeek中光模块龙头，不是受益方）
2. T+5后才发现机会（已错过，不追）
3. 北交所建仓时龙头已到第4板以上（Lag太大，风险大于收益）

### PP-001: 政治局会议拐点入场

**状态：VALIDATED（实例：924政治局 2024-09 / 2025-Q4四中全会预期）**

**模式名称**：政治局/央行级别政策 → D-3至D+2入场窗口

**核心逻辑**：
方向性政策（政治局会议超预期表述、央行超预期降准降息）代表政策拐点，这类催化剂的持续性（15-75天）远长于事件冲击型（5-15天）。但入场需等市场环境确认（硬开关解除后）。

**触发条件（全部满足）**：
1. 国务院/政治局/央行级别声明，有超预期表述（如"扩大内需""增加对股市的支持"等新增措辞）
2. 发布当日或次日全市场成交额 ≥ 历史均量×150%
3. §1.4硬开关：成交>1.5万亿连续2日+涨停>30家连续2日+新催化剂出现

**入场规则**：
- 政策当日：通常硬开关刚重启，等次日确认
- D+1至D+2：硬开关全部解除后入场（龙头涨幅门槛豁免条件：政治局级别+爆量+连续2日 → 龙头涨幅门槛从50%降至20%）
- 科技板块应归为Type 4（政策驱动），持仓跟政策节点，不用5天Type 3上限
- 北交所：主板D+3至T+5才入

**出场规则**：
- §6.1 ★★★信号（主力净卖出龙头>10亿+同日新方向涨停）→ Track B全部清仓
- §4.3炸板率 ≥ 50% → 当日减仓50%；≥60% → 当日清仓（比减仓更激进）
- §4.4龙头跌 >5% → 当日全出
- 政策验证节点（具体政策落地/数据发布后）：评估是否继续持有

**仓位规则**：Type 4允许持有到下个催化节点，无5天上限；仓位≤15%/只

**历史表现**：
- 924行情：927建仓→1009全出，科创50 +38.2%，北证50 +53.1%（高开清仓）
- 策略组合收益：约+8.1%（第一轮），+0.36%（第二轮）

**失败场景**：
1. 政策表述模糊，没有超预期内容（无实质）
2. 政策利好已被市场提前预期（9月涨幅已大，D+3后成交量萎缩）
3. 指数急涨当日硬开关刚触发但尚未稳定（当日可能是末端高潮，第二天反转）

### PP-002: 产业政策分级催化

**状态：VALIDATED（实例：工信部固态电池专项 2025-11 / 国家超算接入 2025-02）**

**模式名称**：部委专项政策 → BOM验证 → 精选标的

**核心逻辑**：
工信部/发改委等部委出台具体产业政策（有BOM拆解意义的），与季报/出货量数据共振，形成Type 2产业催化型行情。区别于政治局拐点型：这类行情持续更短（5-20天），但更精准（只影响产业链核心节点）。

**触发条件**：
1. 部委级别政策文件，内容含具体产品规格/技术标准/补贴金额
2. 板块龙头已开始涨停或大涨（验证市场认可度）
3. 全市场涨停家数 ≥ 60家

**入场规则**：
- 政策发布当日D+0：建仓有直接受益关系的龙头，≤15%
- 生益科技CCL类：D+0至D+2跟进
- 配合BOM拆解（机构报告确认受益量化）：D+0尾盘确认后加仓

**出场规则**：
- 龙头5日内净卖出>10亿 → 减仓50%
- 新的L6末端扩散（非直接受益的远端标的涨停）→ 减仓
- BOM验证被否定（季报数据不及预期）→ 当日清仓

**历史表现**：
- 固态电池/储能2025年11月：11月3-17日主升15天（Q4最长主线），严格按§6.1信号在11月18日退出，最大损失控制在-4%至-6%

## 第四部分：Hot Money Patterns（游资驱动型）

---

### HP-001: 梭哈魔咒减仓

**状态：VALIDATED（4/4实例：2007中国神华/2021TCL/2021三安光电/2023中科曙光）**

**模式名称**：超级游资超大单建仓 → 次日减仓

**核心逻辑**：
特定游资主力（如章盟主）的单笔超大单（>10亿）往往是该阶段顶部的领先信号。历史4次胜率100%，期望对组合贡献约+1.31%。核心逻辑：超大单买入 = 筹码过重 + 游资即将出货信号（不是"大资金看好"而是"大资金最后一搏"）。

**触发条件**：
- 龙虎榜出现特定游资（章盟主/陈小群级别）单笔 ≥ 10亿买入
- 当日或次日龙虎榜确认**无机构净买入**（纯游资独角戏）

**入场规则**：不是入场信号，是减仓/出场信号。

**出场规则**：
- 信号日（龙虎榜可见日，即T+1）：开盘减仓50%（若已持仓）
- 若龙虎榜确认同日有机构净买入 ≥1亿：降级为观察，不强制减仓
- 注意：龙虎榜T+1才可见（信号日的下一天才能确认），因此：
  - 信号日收盘前已有预估（盘中大单监测）→ 隔日开盘确认
  - 信号日次日开盘：确认龙虎榜无机构 → 减仓执行

**仓位规则**：减至50%（非全清，保留基础仓位）

**历史表现**：
- 4/4案例严格执行，胜率100%
- 期望值：0.75×(+2%) + 0.25×(-0.75%) = +1.31%对组合（每次触发）

**失败场景**：
1. 触发信号后股价继续上涨（约25%概率），会错过部分涨幅，但期望值仍为正
2. "超大单"来自机构而非游资（误判），需要次日龙虎榜确认游资属性

### HP-002: Type 5游资坐庄精准介入

**状态：VALIDATED（实例：商业航天2026-01，含失败教训）**

**模式名称**：游资板块连板早期建仓 → 连板≥8强制退出

**核心逻辑**：
Type 5游资主导的板块连板，在第1-3板窗口内可以用小仓位参与（5%上限），但监管介入风险极大，必须以S+信号（监管三连发）而非龙虎榜作为退出信号。

**触发条件（全部满足）**：
1. 板块当前连板高度 ≤ 3板（错过则完全不参与）
2. 龙虎榜无机构或机构极少（纯游资主导）
3. 全市场涨停 ≥ 60家（整体市场情绪正常）
4. 有基础产业催化（不是纯概念蹭热点）

**入场规则**：
- 第3板确认Type 5特征当日：建仓，仓位5%（取下限，信息滞后折价）
- 板块与其他产业链有重叠（如商业航天初期与军工重叠）：不额外加仓，维持5%

**出场规则（按优先级，龙虎榜排最后）**：
1. 优先级1：监管任何限制措施 → 盘中收到当日即执行，不等收盘
2. 优先级2：连板高度达第8板 → 无论信号，减仓50%；第10板全清
3. 优先级3：主板龙头跌 >5% → 当日全出
4. 优先级4：全市场涨停 <60家连续2日 → 当日减仓
5. 优先级5：龙虎榜核心游资净卖出 → T+1开盘全出（仅上述4条未触发时才作为主要信号）

**仓位规则**：5%（硬上限，不可加仓，信息滞后已折价）

**历史表现**：
- 商业航天2026年1月：5%仓位×12%区间收益（成功执行S+退出）= +0.6%组合贡献
- 若未执行S+（等到龙虎榜T+1）：一字跌停，无法出逃

**失败场景**：
1. 发现板块时已达第5板以上 → 完全不参与
2. 监管消息盘后发布（非盘中）且板块次日一字跌停 → 无法执行，接受损失
3. 纯游资无任何产业逻辑支撑 → 直接绕过，不参与

---

### HP-003: 龙虎榜T+1退出（标准场景）

**状态：VALIDATED（实例：多个Type 2标的，与HP-001不同场景）**

**模式名称**：龙虎榜核心游资净卖出 → T+1开盘减仓

**核心逻辑**：
龙虎榜T+1固有延迟（信号日次日才可见），在机构主导的Type 2行情中可接受，因为机构行情持续周期长（数周）。但在Type 5游资行情中此信号已失效（见HP-002）。

**适用场景**：Type 2（产业催化）+ 机构主导行情，游资部分参与者退出

**触发条件**：
- 龙虎榜：核心游资（单笔>5亿）净卖出
- 行情类型：Type 2（非Type 5）
- 持仓已有浮盈（非亏损减仓）

**出场规则**：
- 信号日+1的开盘：减仓50%
- 若同日有机构净买入：维持仓位，降级为观察

**历史表现**：在Type 2行情中，T+1可执行（有流动性），亏损可控

**失败场景**：
- Type 5游资行情（监管后跌停封死）→ 切换至HP-002的退出逻辑
- 北交所标的 → T+1可能已-30%跌停，此信号完全不适用（必须用主板龙头T+0信号）

## 第五部分：Style Switch Patterns（风格切换型）

---

### SS-001: 炸板率顶部信号清仓

**状态：VALIDATED（实例：924行情1008 炸板率69.8% / Q4行情11月18日骤降47%）**

**模式名称**：炸板率/涨停骤降 → 当日清仓，不等

**核心逻辑**：
炸板率（已涨停但当日开板的比例）是市场情绪最清晰的顶部信号，比指数高点和拥挤度信号更领先。炸板率 ≥ 60%（非50%）在历史上是"当日清仓"而非"当日减仓"的信号。涨停家数单日降幅 ≥ 40% 是等效信号。

**触发条件（任一满足）**：
- 条件A：炸板率 ≥ 60%（当日炸板数/当日曾涨停总数）
- 条件B：涨停家数单日降幅 ≥ 40%（如85家→45家，降幅47%触发）
- 条件C：炸板率 50-60%（较弱信号）→ 减仓50%，次日再评估

**出场规则**：
- 条件A或B触发：当日清仓Track B（§1.5退出优先，不分析，不等）
- 条件C触发：当日减仓50%，次日若继续50%以上炸板率 → 清仓剩余

**仓位规则**：出场完成前不新建任何仓位（§1.5元规则）

**历史表现**：
- 924行情1008：炸板率69.8%，当日减仓+次日（龙头跌>5%）清仓 → 完整规避1009创业板-10.59%
- Q4行情11月18日：涨停85→45家（降幅47%），当日出场 → 规避11月21日跌停71家

**失败场景**：
1. 炸板率50%是"分歧期"，若急于清仓可能错过后续行情（924教训：炸板率信号和拥挤度信号有重叠，精确阈值应用60%清仓而非50%）
2. 单日炸板率高但次日立即反弹（如Q4的"1天无人区"）→ 清仓后需要快速重建仓位，操作频率高

---

### SS-002: 科技→周期风格切换

**状态：VALIDATED（实例：924行情1010中字头接力 / Q4行情10月17日机器人接力）**

**模式名称**：科技龙头断板 → 识别新主线 → 等涨停>60家再入

**核心逻辑**：
科技板块退潮（龙头断板/跌>5%）与周期/防御板块接力，这个切换有规律可寻。铁律止损（龙头跌>5%）执行比风格切换识别更快，应先执行止损，再分析新主线方向。

**触发条件（§6.1的三个★★★）**：
1. ★★★：现主线龙头分化（连板高度下降+炸板率>30%+二线不跟）
2. ★★★：主力净卖出主线龙头>10亿 + 同日新方向涨停
3. ★★★：全市场涨停家数从>60回落至30-60

**出场规则**：
- 3个★★★信号出现 → Track B全部清仓（无条件）
- 2个★★★信号 → 减仓50%
- 1个★★信号 → 减仓，等第二个信号确认

**重入规则（清仓后）**：
- 新主线确认（涨停>60家+有明确催化剂）：T+0当日重建仓位
- 无人区压缩到3天（2024年后规律）：最多等3个交易日，不要等7-15天

**历史表现**：
- 924行情1009-1010：铁律先行（计算机-10.22%触发龙头跌>5%清仓），比风格切换识别（1010中字头确认）早2天完成出场，完整规避
- Q4行情：11月18日→21日，提前3天识别，严格执行者损失-4%至-6% vs 未执行者-10%至-15%

**失败场景**：
1. 风格切换是假信号（如924行情1017清仓后1020立即反弹，需要重建仓位，有操作代价）
2. 无人区比3天长（宏观环境变化时），等待重入成本高
3. 新主线不在Track B的<500亿市值范围内（中字头大市值不符合Track B定义）

---

### SS-003: 无人区快速重建

**状态：VALIDATED（实例：924行情1010-1014 3天 / Q4行情11月21-24日 3天）**

**模式名称**：无人区结束 → 涨停>60家+新催化剂确认 → T+0重建

**核心逻辑**：
2024年后无人区持续时间已压缩至3-7天（原估计7-15天严重高估），快速V型反弹是新常态。等待太久会错过第二波主升浪入场。

**触发条件**：
1. 无人区已持续 ≥ 2个交易日
2. 反弹日涨停家数 ≥ 60家（重启信号）
3. 有明确的新催化剂或新主线名称（不是"昨天的板块反弹"）

**入场规则**：
- 反弹首日（涨停>60家）：T+0当日可建仓，按新主线Type分类执行
- 若前一日清仓仓位，可全额重建（不设冷却期）

**历史表现**：
- 924行情：1009清仓→1014（3天后）军工/中字头信号→1018科技再爆发，3天无人区
- Q4：11月21日→11月24日（3天），AI/华为概念接力反弹

**失败场景**：
1. 反弹是假信号（仅当日涨停>60，次日回落）→ 建仓后需用§6.1信号再次确认
2. 宏观风险未解除（如贸易摩擦/监管政策还在发酵）→ 无人区可能延长至7天以上

## 第六部分：Anti-Patterns（反面模式/陷阱库）

反面模式同样是必读内容。遇到以下信号时，立即触发"不参与"决策，无需进一步分析。

---

### AP-001: 蹭热点妖股陷阱

**触发场景**：板块正在爆发，某只个股涨停，但与板块主题无实际业务关联（仅名字沾边、主营业务无交叉）。

**历史案例**：杭钢股份（钢铁+地产）在DeepSeek行情中9连板+112% → 一字跌停（2025年2月）；多家短信公司蹭AI概念。

**识别规则**：
- 无一手来源验证业务关联（无合同、无公告、无API接入证明）
- 主营业务与当前热点行业无交叉（查年报主营收入构成）
- 股票名称沾边但实体业务无关

**操作**：**不参与，无例外。**已持有者立即清仓（不以成本为理由持有）。

**代价估算**：错过假信号的"涨幅"（如杭钢+112%），但规避崩盘（一字跌停）。期望值为正。

---

### AP-002: 追末端扩散标的

**触发场景**：主线行情进入尾声，末端产品（与主线关联极远的概念）开始涨停。追入此类标的即追末端。

**历史案例**：
- DeepSeek行情末期：杭钢股份（钢铁）蹭热点，+112%后崩塌
- AI算力链末期：超级电容（宏达/江海）在光模块/PCB主线退潮时涨停（L6扩散信号）
- 商业航天末期：非航天概念的蹭热点股

**L6末端扩散识别**：
- 主线已运行 ≥ 5个交易日
- 出现与主线产业链无直接关系的标的涨停（超级电容/冷却液/碳纤维等与算力无关）
- 该类标的是"第6层以上"的传导链末端

**操作**：不追末端扩散标的。L6信号出现 = 主线退潮信号（第1-2次），减仓主线仓位50%。注意：L6连续强势2-3天可能演变为独立主线，届时等重新确认后再考虑。

---

### AP-003: 龙虎榜滞后致命陷阱

**触发场景**：Type 5游资行情或北交所标的，以龙虎榜游资卖出作为退出信号，结果T+1已是一字跌停。

**历史案例**：商业航天2026年1月15日，监管盘中发出三连发，游资陈小群1月16日才在龙虎榜出现（T+1），但1月16日全线一字跌停，无法出逃。

**识别规则**：当持仓标的属于以下类型时，龙虎榜退出信号不可作为主要退出依据：
1. Type 5游资主导行情（连板高度≥3板，无机构）
2. 北交所标的（±30%涨跌停）
3. 任何监管已有预警的板块

**操作**：在上述场景中，出场依据为：监管信号T+0 → 炸板率 → 龙头跌>5% → 时间止损，龙虎榜仅作兜底。

---

### AP-004: 过早拥挤度退出

**触发场景**：行情处于Type 1阶段，触发40%拥挤度规则后急于减仓，但实际上行情后来演化出Type 2特征，持续8天不崩。

**历史案例**：DeepSeek行情D2（2月6日）成交额>40%，按T1规则应减仓，但D4-D8超算主升浪继续，并行科技再+180%。

**识别规则**：
- 拥挤度40%触发，但同日/次日出现"具体数字驱动的产业催化"（平台接入合同/国家级平台上线）
- 催化剂有一手来源（官方公告，非媒体解读）

**操作**：当T1→T2升级条件满足时（见EP-001升级检查），拥挤度容忍度从40%放宽至48%，不过早减仓。注意：每日检查，不可事后追认。

---

### AP-005: 无人区等待过久

**触发场景**：清仓后按旧规则（7-15天无人区预期）等待，结果3天后市场反弹，错过了重建窗口。

**历史案例**：924行情1009清仓后，策略预期7-15天无人区，但实际1014（3天后）军工接力，1018再次爆发；Q4同样3天反弹。

**识别规则**：2024年后A股无人区已系统性压缩至3-7天（融资余额高、政策密集、AI应用对信息消化加速）。

**操作**：清仓后最多等3个交易日即开始重新检查入场条件（涨停>60家+新催化剂），而非等7天。

---

### AP-006: 连板高度过高才参与Type 5

**触发场景**：发现Type 5游资行情时，连板已到第4-5板甚至更高，还想追入。

**历史案例**：商业航天2026年1月，第1-3板时还可以入，第4板以上（1月8日）已超V1.1入场窗口，风险/收益比恶化。

**识别规则**：
- 发现板块时连板已达第5板以上 = 直接绕过，不参与
- 游资行情只在第1-3板窗口有合理期望收益

**操作**：发现时已≥5板，当记录观察，不建仓。这不是"踏空"，是正确的纪律执行。

---

### AP-007: 假政策误判为PP-001

**触发场景**：市场有传言/预期政策发布，提前建仓，但政策未如期发布或内容低于预期，导致建仓后被套。

**历史案例**：Q4四中全会预期在10月27日后逐步证伪，导致部分标的高位被套。

**识别规则**：
- 催化剂是"预期"而非"已发布"：降低仓位至50%（如PP-001建议的≤15%降为≤8%）
- 政策发布后内容与预期一致：可按正常仓位建仓
- 政策发布后内容低于预期：当日减仓

**操作**：政策"预期"阶段不满仓，等待实际发布确认后再加仓到目标仓位。

## 第七部分：Pattern匹配协议

### 建仓前匹配流程（<2分钟）

```
Step 1：识别当前机会的主要特征
  → 是什么类型的催化剂？（事件/产业链/政策/游资/风格切换）
  → 选择对应Pattern分类（EP/CP/PP/HP/SS）

Step 2：逐条对照Pattern触发条件
  → 满足≥70%条件 = 初步匹配
  → 检查Anti-Pattern：是否触发任何AP？触发即取消匹配

Step 3：匹配打分（0-100分）
  见下表

Step 4：打分结果 → 操作决策
  ≥80分：强匹配，按Pattern规则执行，仓位可加成
  60-79分：一般匹配，按Pattern基础规则执行，不加仓
  40-59分：弱匹配，降仓位50%，严格止损
  <40分：无匹配，按策略基础规则操作，标准仓位
```

### 匹配打分规则（0-100分）

| 维度 | 权重 | 满分条件 |
|------|------|---------|
| 触发条件满足数/总条件数 | 40% | 100%条件满足=40分 |
| 有历史实例 | 20% | validated=20分；observation=10分 |
| 催化剂一手来源可验证 | 20% | 官方公告/机构原文=20分；二手媒体=10分 |
| 市场环境匹配（吸气期/呼气期）| 10% | 吸气期=10分；中性=5分；呼气期=0分 |
| 无Anti-Pattern触发 | 10% | 无AP=10分；有AP=直接0分（取消匹配）|

**示例**：
- DeepSeek EP-001：触发条件4/4（40分）+ validated（20分）+ 新闻报道=一手（20分）+ 吸气期（10分）+ 无AP（10分）= 100分 ✓
- 蹭热点标的 EP-001：AP-001触发 → 0分，不参与

### 匹配后的行为调整

| 匹配分 | 仓位调整 | 持有期参考 | 止损严格度 |
|--------|---------|-----------|----------|
| ≥80分 | 基础仓位×1.2（不超过硬顶）| 按Pattern出场规则 | 标准（铁律不放宽）|
| 60-79分 | 基础仓位×1.0 | 按Pattern出场规则 | 标准 |
| 40-59分 | 基础仓位×0.5 | 缩短至50%持有期 | 严格（提前止损线）|
| <40分 | 基础仓位×1.0（策略基础规则）| 策略基础规则 | 标准 |

### 无Pattern匹配时的默认行为

无匹配时，使用策略基础规则：
1. 按行情类型（Type 1/2/3/4/5）分类
2. 入场时机参考§3.4催化剂入场时机表
3. 出场使用§4.1-§4.5退出信号
4. 仓位使用标准上限（≤15%/只）
5. 不享受PlayBook的仓位加成

## 第八部分：Pattern演化规则

### 新Pattern加入条件

1. **实例数 ≥ 2次**：必须有至少2次历史实例（含本次）
2. **正收益**：两次实例均产生正收益（组合层面）
3. **机制清晰**：能用1-2句话说清楚为什么有效（不是统计噪音）
4. **触发条件可操作**：条件必须在实时交易中可观测（不依赖事后数据）
5. **提案流程**：将新Pattern写入observation_pool，观察第2次实例后升为validated

### 失效Pattern淘汰规则

1. **连续2次失败** → 标记为`dormant`（休眠）
2. **失败定义**：触发Pattern后按规则操作，结果亏损 ≥ 3%（组合层面）
3. **dormant条件**：dormant Pattern不执行，仅记录观察。若第3次出现后盈利，可重新激活
4. **废弃条件**：累计5次失败（不论是否连续），标记为`deprecated`，从匹配流程中移除

### Parameter动态校准规则

| 参数 | 校准触发 | 更新方式 |
|------|---------|---------|
| 产业链Lag | 每次CP Pattern执行后，记录实际Lag | 取最近5次Lag的中位数更新 |
| 无人区时长 | 每次SS-002/SS-003后，记录实际天数 | 最近4次中位数更新 |
| 炸板率阈值 | 每季度回顾，看是否有系统性漂移 | 由用户确认后更新 |
| 拥挤度容忍度T1/T2 | 每次Type 1/2行情结束后复盘 | 提案修改，用户确认 |
