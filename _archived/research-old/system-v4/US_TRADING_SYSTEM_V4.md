## §2 入场时机（什么时候买）

### 2.1 三路径决策树

**Step 1：查VIX**

| VIX水位 | 路径 | 含义 |
|--------|------|------|
| **<18** | Momentum路径 | 趋势市，追momentum，不等完美回调 |
| **18-25** | 催化剂路径 | 必须有具体事件才入场，纯momentum减半 |
| **>25** | 逆向路径 | 只做quality逆向+极端恐惧+多重催化剂 |
| **>35** | 危机模式 | 只做宏观对冲（GLD/能源/防御），个股多头≤30% |

**Step 2：GAP规则（任何情况下适用）**

| Gap幅度 | 规则 | 案例依据 |
|--------|------|---------|
| **>15%** | 等3-5个交易日，回调到gap 50%位置再入场 | META +12%开盘→20天后-12.2% |
| **5-15%** | 当天建半仓，次日确认加满 | PEAD漂移空间，分批控成本 |
| **<5%** | 一次性建仓至目标仓位 | 市场仍有分歧，趋势未充分定价 |

### 2.2 Momentum路径（VIX<18）

| Momentum分 | 入场条件 |
|-----------|---------|
| ≥28/35 | 任何3%回调或5日横盘即入，不等更大回调 |
| 20-27/35 | 5天横盘后突破前高入场 |
| <15/35 | 必须>15%回调+基本面支撑 |

> 铁律：美股强势股最大回调通常3-7%（AMD 11笔 +215%累计，每次3-7%回调后创新高）。等10%回调=系统性踏空。

**美股momentum持续数季度，趋势跟踪优于等回调。** 铁律：每次3-7%回调是入场机会，不是止损信号。

### 2.3 催化剂路径（VIX 18-25）

**财报入场决策树**

```
Pre-earnings:
  近30天内财报 + A级+ + 股价回调>5% → DIP BUY（仓位≤8%）
  近30天内已涨>20% → 不追（run-up已消化预期）

Post-earnings:
  周期股（PE<20x）+ 三连击（Revenue beat + EPS beat + Guidance上调）
    → T+3~T+5买入（等sell-the-news消化）持有60-90天（PEAD最强）

  成长股（PE 20-40x）
    → Revenue beat + Guidance上调 → T+1买入，持有8-10天蜜月期
    → 仅EPS beat（cost cut）+ Guidance平/下调 → 不操作

  高PE（>40x）AI/科技股
    → Guidance强力上调 → 当天半仓，T+1加满，持有8-10天（L7蜜月期）
    → Beat但Guidance平/下调 → 当日减仓/做空窗口（NVDA模式）

  任何Miss → 不接飞刀，等T+3稳定后re-evaluate
```

**FOMC/宏观数据入场**：事件后24h方向确认再入场，不提前猜。
**产品发布（WWDC等）**：事件前1-2周建仓，事件当天减仓50%。
**地缘事件**：风险上升期提前1-5天入场（军事调动可见时）；停火当天立即入quality。
**S&P指数调整**：公告次日（T+1）买入；effective date前1天（T-2）清仓。
**M&A并购**：不追目标公司（gap通常>15%，已在>15%排除区）。收购方：公告后T+3评估（市场初始反应通常过度）。

### 2.4 逆向路径（VIX>25）

极端恐惧（VIX 30+）+ 多重催化剂（停火/政策转向） → 全力买quality（GOOGL/AMZN/AMD）。

**历史验证**：Apr 8 2026停火，VIX从31→21，Agent-10: 91%WR，+18.2%均值。这是全系统最高胜率setup。

### 2.5 禁止入场清单

1. 追>15% gap当天
2. 高PE（>50x）beat后加仓（NVDA模式=卖出，不是买入）
3. VIX>25做空quality compounders（Agent-09最贵教训：-48%）
4. 财报大gap>10%当天不减仓
5. 等10%+大回调才入强势股（美股强势股最大回调通常3-7%，等10%=系统性踏空）
6. 地缘事件爆发当天才建仓（已price in）
7. 财报Miss后T+0接飞刀（Miss惩罚是Beat奖励的6倍）

### 7.3 四种Regime状态

**RISK-ON** (VIX<18，信号均未触发)
- 正常交易，momentum入场
- 多头sizing无约束，做空独立catalyst即可
- 历史：Dec 2025均值VIX 15.5 → 100%WR

**CAUTIOUS** (VIX 18-25，或1个信号触发)
- 暂停新建仓，现有持仓维持
- 新机会等2个交易日确认方向后再操作
- 历史：Nov 2025 VIX 19.9 → 53%WR，未及时切换代价

**RISK-OFF** (VIX>25，或2+信号同时触发)
- **核心规则1: 停止所有做空**（24h内关闭所有现有空头）
- **核心规则2: 多头减仓至60%**（卖出最低conviction持仓）
- 只持有S/A级quality names
- 历史：Mar 2026 VIX>25 → 做空全输，做多quality有机会

**EXTREME FEAR** (VIX>35)
- 立即评估逆向做多quality机会
- 不追空（VIX>35=历史12个月后S&P 100%正收益）
- 等VIX 5日delta出现-15%信号→执行逆向建仓
- 历史：Mar 27 VIX=31 + Apr 8停火 → Agent-10入场 91%WR

### 7.4 Regime Detection决策树（30秒执行）


## §9 ABCD下跌分类（美股版）

> **CRITICAL**: ABCD分类之前，禁止任何卖出动作。先分类，后行动。60秒内完成分类。

### 9.1 五类型分类表

| 类型 | 名称 | 判别标准 | 行动 | 止损规则 |
|------|------|---------|------|---------|
| **A** | 大盘系统性下跌 | SPY/SOX/KWEB跌≥**2.5%**且无个股新闻。注意: 大盘跌但**板块指数稳**→个股可能是C类 | **Hold**（不操作，不恐慌，不补仓） | 只有同时触发组合级止损才处理 |
| **B** | 板块轮动噪音 | 指数稳或涨；无新闻；下跌<3% | **Hold**，观察1-2天 | 持续3天且无基本面变化仍持有 |
| **C** | 叙事切换（非破坏性） | 有新闻但不涉及核心thesis；市场情绪切换 | **评估**：减仓0-30%（按信心等级），写下重新评估结论再操作 | 24h内给出C→D升级结论或维持C |
| **C+** | 待确认叙事变化 | 单一信源/传闻/预期打压，无硬数据 | **观察**：24h内主动核实；核实前不操作 | 核实后升级C或D；超过48h视为D |
| **D** | Thesis被证伪 | 财报核心指标miss≥2个季度/重大客户流失公告/政策文件否决/管理层不明原因离职 | **立即清仓**，48h内执行，不等反弹 | 无例外。D类不存在"再等等看" |

**美股A类阈值=2.5%**。美股波动率天然更高，1.5%在美股是正常噪音，不触发A类判断。

### 9.2 60秒分类流程

```
1. 看SPY/SOX/KWEB：跌≥2.5%？ → 是 = A类 → Hold，不操作
2. 有没有今天的新闻？ → 没有 = B类 → Hold 1-2天观察
3. 新闻改变了thesis核心逻辑吗？
   → 没改变 = C类 → 评估，写结论，可减0-30%
   → 改变了但只有传闻/单一信源 = C+类 → 24h核实，暂不操作
   → 有硬数据支撑（财报/公告/政策文件）= D类 → 48h内清仓
记住：只有D类是必须卖的。
```

### 9.3 C与D的高发混淆场景（美股版）

| 场景 | 分类 | 理由 |
|------|------|------|
| 财报beat但市场叙事切换（META案例） | C类 | 公司基本面未变，市场重定价叙事 |
| 竞争对手发布更好产品 | C+→D | 需核实对份额/GPM的具体影响 |
| 大客户单季度减少采购 | C+类 | 单季度可能是库存调整，≥2季度升D |
| 核心技术/专利被否 | D类 | 硬壁垒消失，thesis破坏 |
| DOJ/SEC正式调查公告 | D类 | 业务基础受冲击，直接D |
| 传闻监管打压（无正式文件） | C+类 | 等文件落地 |
| 正式监管文件落地（负面） | D类 | 政策确定，无讨论空间 |
| 管理层CEO替换（内部提拔） | B-C类 | 评估新任背景 |
| 管理层创始人离职（无交代） | D类 | 创始人vision是核心资产时直接D |
| 财报in-line但股价-9%（HSAI案例） | **C类，不是D类** | in-line非miss，止损前先分类 |
| 指引下调但收入beat | C类 | 需评估下调幅度，不自动升D |

## §10 行为铁律 L10-L15（美股语境完整版）

> 这6条是在正确框架下最常见的执行失败来源。每条均有具体诊断信号和触发检查。

---

### L10 — 关注点漂移：看别人涨了就想换

- **定义**: 持有NVDA未动，看到AMD/META大涨 → 萌生"换到AMD"的想法
- **诊断信号**: 卖出原因是"别的股票涨了它没涨"（不是thesis变化，不是催化剂失效）
- **铁律**: 卖出前必须回答两个问题：①thesis变了吗？②我是因为看了别的股票才想卖吗？如果②是"是"→关闭Bloomberg/行情软件，等1小时后再决定
- **美股语境**: 美股sector rotation快，FAANG/AI个股日内互相拉锯是常态，不是信号
- **Why**: 相对表现焦虑，和追涨是同一情绪的两面。holding the winner是美股最难执行的动作

---

### L11 — 催化剂前离场：在最关键的时候撤退

- **定义**: 有明确财报日期（如NVDA 5月28日），在财报前3天因"已涨了不少"而减仓
- **诊断信号**: 建仓时写了催化剂日期；催化剂未到就开始考虑减仓；唯一理由是"涨了"
- **铁律**: 催化剂前默认持有。减仓需要给出"不是催化剂原因"的充分理由（如ABCD分类D类，或已大幅超过目标价50%+）
- **美股特例**: 财报当天high PE AI股（>40x）+ 已涨>20%pre-earnings → 可减至50%（NVDA蜜月期模式）。这是规则内的，不是违反L11
- **Why**: 从赔率角度：强势股催化剂前持有期望值 > 未持有。短期获利欲望压过赔率思维

---

### L12 — 赢小输大反置：小赚就跑，大亏死扛

- **定义**: AMD涨10%全卖了；HSAI亏20%继续扛
- **诊断信号**: 收盘时扫描持仓——盈利方向仓位是否在缩小，亏损方向是否在扩大或维持
- **铁律**: 卖出时问自己"如果这是一笔新投资（$0基础），我会以当前价格买吗？"是→持有；不是→这才是真正的卖出信号
- **美股案例**: HSAI Q1 in-line但-9%，如果新投资你会在这个价格买吗？→ 需要重新评估，不是机械持有
- **Why**: 前景理论（亏损厌恶）。美股T+0（可以随时卖）使这个偏差更容易爆发，必须用机械规则对抗

---

### L13 — 追涨三联动：FOMO驱动的买入（三行强制检查）

- **定义**: 任何由"今天才知道它涨了"驱动的买入决策
- **铁律（三行检查，缺一不可）**:
  - ①买的理由是什么（不是"涨了"）— 能说出具体thesis
  - ②跌20%我割不割（说不出→不买）— 测试conviction深度
  - ③是不是今天才知道它的（是→不买）— 防止信息链末端买入
- **第③条最关键（美股语境）**: NVDA今天涨8% → 今天才知道 = 别人早买了，你在接出货。AMD明天财报但今天才看到 = 信息链末端
- **例外（美股）**: 底部放量涨停 + 真实催化剂 + 个人已研究（只是今天触发了setup）= 不违反L13。关键是"研究过"vs"今天才注意到"
- **Why**: FOMO（错失恐惧），在美股这个信息速度更快的市场更致命

---

### L14 — 仓位-Conviction倒挂：最大仓位是最不确定的

- **定义**: GOOGL（C级，几乎不了解驱动力）持8%；NVDA（A级，深度研究）持4%
- **诊断信号**: 说不出完整thesis的标的持仓最重；研究最深的标的持仓最轻
- **铁律（美股版）**: 每次建仓前先定Conviction等级（S/A/B/C/T）→对应仓位上限（25%/15%/10%/8%/5%）→ 按分数上限建仓。说不出thesis = C级，$7,500-$12,000（5-8%）
- **验证**: v3实盘Day 1-5：SRUUF(C级)8.1% > FPS(B+级)4.6%，完全倒置
- **Why**: 美股买入时没有强制scoring框架，凭感觉/仓位倒置

---

### L15 — Thesis错配不自知：涨了不代表你的理由对了

- **定义**: 买AAPL因为"AI功能扩展"，实际涨幅驱动是服务业务margin扩张（与你的thesis不同）
- **诊断信号**: 说不清楚"它现在为什么在涨，和我买的理由是同一个吗？"
- **铁律（强制一步，每笔买前执行）**: 搜索"[TICKER] stock why is it up today/this week"或查看最近catalyst → 对比搜索结果和自己的买入理由 → 不同=不买
- **美股案例**: 买某AI股因为"数据中心需求"，实际上涨驱动是股票回购公告——thesis对了方向，错了驱动力，持仓逻辑基础不牢
- **Why**: 自我确认偏误（confirmation bias）。美股ETF/机构主导，真实驱动力必须通过成交量/新闻验证，而非假设

## §14 执行强制链（确保方法论被执行）

### 14.1 问题根源

AI知道规则但不执行——v3.0所有规则写清楚但5天运行中几乎没有主动执行。原因：每次决策是独立的生成过程，规则存在于输出文本中但不存在于下一个决策节点的context里。

**修复原则：所有修正机制必须是外部强制（硬编码检查表、数值门槛），而不是"提醒AI注意"。**

### 14.2 SSOT文件路径（美股版）

| 文件 | 用途 | 读取规则 |
|------|------|---------|
| `portfolio_state.json` → `us_positions` | 美股持仓唯一真相源 | 每session必读，任何价格/仓位计算只从此文件读 |
| `portfolio_state.json` → `us_cash` | 美股现金余额 | 建仓前先查此值 |
| `pending_actions.json` → `market="us"` | 美股待执行操作 | 每session开始先读，先执行再选新标的 |
| `watchlist_config.json` → `us_candidates` | 美股候选做多池 | D1研究时读 |
| `watchlist_config.json` → `us_short_candidates` | 美股空头候选池 | 每周三做空扫描时读 |
| `daily-reviews/YYYY-MM-DD.md` | 每日美股复盘 | W4（04:00+1 BJT）写完整版 |

**SSOT原则**: 所有价格、仓位、P&L只从`portfolio_state.json`读取。禁止从memory或旧文件推算。

### 14.3 pending_actions.json机制

**用途**：记录所有未执行的方法论要求，每次session开始必须先清零，再选新标的。

**写入时机**：需要后续rebalance/加仓/减仓时；做空扫描发现候选时；Regime切换后需要调整时。

**格式**：
```json
{
  "pending": [
    {
      "type": "rebalance/short/stop_loss/regime_adjustment",
      "action": "具体动作描述",
      "ticker": "XXXX",
      "market": "us",
      "reason": "方法论依据",
      "deadline": "YYYY-MM-DD",
      "priority": "high/medium/low"
    }
  ]
}
```

### 14.4 每session开始强制检查清单（按序执行，不可跳过）

```
□ 1. Regime Check（30秒，3行）：
     VIX: ___ → Risk-On/Cautious/Risk-Off
     10Y收益率: ___% → 方向: 上行/下行/稳定
     SPY RSI(14): ___ → 超买/超卖/中性

□ 2. 读pending_actions.json（filter: market="us"）→ 先执行上次未完成的方法论动作（优先于选新标的）

□ 3. 空头状态检查：
     当前空头持仓数: ___
     若=0 → 必须寻找1笔做空机会，或明确记录"今日无空头机会+原因"

□ 4. Conviction-Sizing倒挂检查：
     列所有美股持仓+Conviction等级+当前仓位%
     检查是否有低Conviction仓位>高Conviction仓位

□ 5. 总仓位检查：
     多头: ___/6只 | 空头: ___/3只 | 总: ___/9只
     现金: ___% （≥15%/$22,500？）
```

### 14.5 每笔新建仓强制检查表（无此表=不能交易）

```
□ 标的:             方向（多/空）:
□ 5维评分总分: ___/50 → 等级: S/A/B/C/T/排除
□ Bear case downside: ___% （>35%且非A+=排除；25-35%=T级+止损）
□ 初始仓位%: ___ | 最大仓位%: ___ | 对应$: ___
□ 止损线: ___（EOD check，非实时）
□ 催化剂+日期: ___（必须30天内，A级例外）
□ 目标价: ___ → 分批减仓计划: 50%处减1/3, 100%处减1/3, 剩余trailing
□ 槽位检查: 多头___/6 | 空头___/3 | 总___/9
□ Sizing倒挂检查: 新仓Conviction ≥ 所有比它小的现有仓位的Conviction？
□ 最低建仓金额 ≥ $7,500？
□ Thesis一句话: ___
□ 写入pending_actions.json（如需后续操作）
```

### 14.6 做空配额强制机制

- 每日开盘前（BJT 22:00）：空头持仓=0 → 必须执行"每日空头扫描"（5分钟）
- 每周三：执行全套空头扫描SOP（5步），即使0笔执行也必须写记录
- 每月末：复盘空头持仓占比，持续<10%需要分析原因
- 空头记录写入daily-reviews："空头状态: {n}笔 | {ticker} | 今日无空头：{原因}"

### 14.7 Regime检查强制机制

- FOMC/CPI/NFP当天：事件发布后次日开盘前强制执行24h SOP（§7.5）
- VIX单日变化>15%：当天必须执行Regime Check，写日志
- 连续3笔空头全输：检查VIX和市场方向，评估是否Regime shift

### 14.8 违反记录与自纠正

每次违反以下规则之一，在daily-reviews中写明：
- [VIOLATION] 项目 | 原因 | 补救动作

强制规则清单（违反即记录）：
1. 开仓无5维评分表
2. Bear case>35%仍建仓（非T级）
3. VIX>25建新空头
4. 做空没有thesis+catalyst+日期
5. 未读pending_actions.json就选新标的
6. 总持仓超过9只
7. 单笔建仓<$7,500

- 周期股blowout让winner跑60-90天，不过早了结
- 分批减仓：目标50%减1/3，100%减1/3，剩余trailing

**做空**

### DON'T（绝对禁止）

**入场**

**仓位**
