# 退出规则框架 v4.0
**Agent-14 | 美股交易系统v4.0 | 2026-05-22**

---

## 一、核心结论（一句话）

退出比入场更难：止损保本，trailing stop锁利，时间止损防资本占用，催化剂后必须主动决策——四类规则缺一不可。

---

## 二、止损方式选择矩阵

| 信心等级 | 止损类型 | 触发阈值 | 执行方式 | 依据 |
|---------|---------|---------|---------|------|
| **S级** | 催化剂止损 | 催化剂失败=当日全出，不等价格 | 一次全退 | S级是纯催化剂驱动，催化剂失败thesis已证伪 |
| **S级备用** | 硬止损 | -7%（价格触及立即执行） | 一次全退，无条件 | strategy.md铁律：S级-7%硬规则 |
| **A级** | Trailing stop | ATR×2回撤，或高点-12%（取较宽） | 分批：先减1/3，再减1/3，剩余跑trailing | A级需要时间发酵（60天），固定止损会错杀；AAPL-2.5%→+13.4%验证 |
| **B级** | Trailing stop | ATR×1.5回撤，或高点-10% | 分批减仓 | B级给30天，中等conviction，允许一定波动 |
| **C级** | 固定止损 | -8%（硬止损，不等ABCD分类） | 一次全退 | C级低conviction，fast fail，资本效率优先 |
| **T级** | 固定止损 | -8% | 一次全退 | 试仓性质，快进快出，失败就退 |
| **所有空头** | VIX止损 | VIX>25=立即cover全部空头 | 一次全退，无例外 | H2 Agent-09教训：VIX>25开仓的4笔空头全亏-48%；空头在恐慌中损失最惨 |

**Trailing Stop计算方法：**
- ATR×2（A级）：日ATR过去14天均值，触发线=当前高点 - 2×ATR
- 固定%（B级）：从持仓期间最高收盘价回撤10%触发
- 实操简化：每日收盘后更新高水位线，次日以高水位线计算

## 三、三种退出类型详细规则

### 3.1 止损退出

**触发后无条件执行，不等ABCD分类（S级和C/T级），不等次日。**

| 等级 | 止损阈值 | 执行细节 |
|------|---------|---------|
| S级 | -7%（从入场价） | 硬止损，价格触及立即全退 |
| A级 | Trailing -12% | 从持仓期最高收盘价计算，分批减；若单日暴跌>15%则一次全退 |
| B级 | Trailing -10% | 从持仓期最高收盘价计算，分批减 |
| C级 | -8%（从入场价） | 硬止损，一次全退 |
| T级 | -8%（从入场价） | 硬止损，一次全退 |

**一次全退的强制场景（无视等级）：**
1. D类下跌（thesis被硬数据证伪）
2. VIX>25且持有空头
3. 催化剂明确失败（S级）
4. 单日跌幅>15%（止损已无意义，防穿仓）

---

### 3.2 目标价退出（分批锁利）

**基本逻辑：让winner跑，但分批锁定收益，避免"拿满又还回去"。**

```
入场后：
├─ 距目标价50%处 → 减仓1/3（锁利第一批）
├─ 距目标价100%处 → 再减1/3（锁利第二批）
└─ 剩余1/3 → 转trailing stop，让剩余仓位跑trend
```

**具体示例（A级，目标价+30%）：**
- 入场$100，目标$130
- 涨至$115（距目标50%）→ 卖1/3
- 涨至$130（目标价） → 再卖1/3
- 剩余1/3设高点-12% trailing stop

**等级差异：**
- S级：催化剂当天减至50%，催化剂后3日内全退（见3.3）
- C/T级：达到目标价直接全退（不做trailing，低conviction不值得拿）

---

### 3.3 催化剂退出（最重要，最复杂）

**核心原则：催化剂落地当天必须主动决策，不能被动持有。**

#### 财报类催化剂（Earnings）

```
财报结果出来后：
Q1: PE水位如何？
├─ PE<20x周期股
│   └─ Blowout beat（Revenue+EPS+Guidance全上） → Hold，PEAD效应持续60-90天
│       普通beat → 减1/3，其余trailing
│       Miss → ABCD分类 → D类全退
│
├─ PE 20-40x成长股
│   └─ 三连击（Revenue+EPS+Guidance全上） → 减1/3锁利，其余持有30天
│       Revenue beat + Guidance平 → 维持，等1周再评估
│       Miss → 立即ABCD分类 → C类观察，D类全退
│
└─ PE>40x高成长（AI/tech）
    └─ 强力Guidance大幅上调 → 持有8-10天蜜月期，Day 10主动减半
        Beat但Guidance平/下调 → 当日减至50%，蜜月期不超过5天
        任何Miss → 当日全退
```

**大gap开盘规则（META模式）：**
- Gap >10%：当日减仓1/2（gap IS the move，拿住=还回去）
- Gap >15%：当日全退，等3-5天回调再评估重新入场
- 依据：META Jan 29 H2，开盘+12%，20天后-12.2%

#### 非财报催化剂（政策/产品发布/事件）

```
催化剂后当天：
├─ 超预期实现 → 减1/3锁利，剩余设trailing stop
├─ 符合预期 → 维持，更新下一个催化剂日期
├─ 低于预期（但thesis完整）→ 观察24h → C类处理
└─ 催化剂失败（thesis核心假设被推翻）→ S级全退 / 其他等级48h内清仓
```

**S级催化剂退出协议（strategy.md规则）：**
1. 催化剂当天：减至50%
2. 催化剂后Day 1：评估残余thesis，明确下一催化剂
3. 催化剂后Day 3：全退（无论盈亏）
4. 持仓不超过10个交易日

## 四、时间止损（防资本长期占用）

**资本是有成本的。无法有效持有的仓位必须释放。**

| 等级 | 时间止损 | 触发条件 | 执行 |
|------|---------|---------|------|
| T级 | 10个交易日 | 无催化剂落地 | 全退 |
| C级 | 20个交易日 | 无明显价格推进 | 全退 |
| B级 | 30个交易日 | 价格横盘，无新catalyst | 减半，其余再给15天 |
| A级 | 60个交易日 | 横盘无推进 | 评估thesis是否失效，必要时清仓 |
| S级 | 10个交易日（硬规则） | 任何情况 | 全退，不等catalyst |

**时间止损操作标准：**
- 计时从首次建仓日开始
- 加仓不重置计时器（防止用加仓规避时间止损）
- 到期前3天必须写重新评估memo，说清楚是否延期以及原因

**PLTR蜜月期验证：** H1数据显示高PE AI股财报后8-10天开始fade，因此A级高PE AI股应将时间止损缩短至15天（特殊规则）。

## 五、Optimal Holding Period（回测锚点）

| 交易类型 | 最优持仓期 | 依据 | 操作含义 |
|---------|---------|------|---------|
| **Earnings催化（周期股blowout）** | 60-90天 | PEAD效应集中在前2个月；MU H2+46.9% | PEAD trade不应早退，设90天时间止损 |
| **Earnings催化（高PE AI股）** | 8-10天 | PLTR蜜月期H1数据；NVDA beat后-9% | Day 8主动减半，Day 10全退 |
| **大gap开盘后** | 0天 | META +12%开盘→-12.2% over 20天 | Gap IS the move，当日减仓 |
| **Trend/Momentum** | 30-60天 | AMD H2 +68.6% over 20天（20天卖太早）；Agent-15 avg +157.5% | 不要用20天固定持仓期限制trend trade |
| **做空** | 15-20天 | 空头利润集中在前3周；COIN H2 5笔都在3周内锁利 | 空头超过20天的每日评估是否到期 |
| **T级试仓** | 10天最长 | 低conviction，快进快出是设计原则 | 不要让T级变成事实上的A级持仓 |

## 六、退出决策树（可直接嵌入strategy.md）

```
每笔持仓每日检查：

Step 1: 触发止损了吗？
├─ S级: 今日价格触发-7%？→ 立即全退（不走后续步骤）
├─ C/T级: 今日价格触发-8%？→ 立即全退
├─ A/B级: 今日收盘触发trailing stop？→ 分批减仓
└─ 全部空头: VIX>25？→ 立即cover全部空头
→ 未触发 → Step 2

Step 2: 今天有催化剂落地吗？
├─ 有财报
│   ├─ 周期股+Blowout three-hit → 继续持有，更新trailing stop
│   ├─ 高PE AI股+任何beat → 检查蜜月期天数（D≤8 hold; D>8 减半）
│   ├─ Gap>10% → 当日减半；Gap>15% → 当日全退
│   └─ Miss → ABCD分类 → D类全退，C类观察
├─ 有非财报催化剂
│   ├─ 超预期 → 减1/3锁利
│   ├─ 失败（S级）→ 当日全退
│   └─ 失败（其他）→ ABCD分类
└─ 无催化剂 → Step 3

Step 3: 目标价检查
├─ 距目标50% → 减1/3
├─ 距目标100% → 再减1/3，剩余转trailing
└─ 未到目标 → Step 4

Step 4: 时间止损检查
├─ T级: 持仓≥10天且无catalyst？→ 全退
├─ C级: 持仓≥20天？→ 全退
├─ B级: 持仓≥30天且横盘？→ 减半
├─ A级: 持仓≥60天且横盘？→ 重新评估thesis
└─ S级: 持仓≥10天？→ 无条件全退

Step 5: 无操作，更新高水位线，明日继续
```

## 七、特殊场景规则

### 7.1 D类下跌（Thesis证伪）
- 一次全退，48h内执行，无等级之分，无例外
- 不等止损线，不等回调，不管当前盈亏
- 定义：财报核心指标miss≥2季度 / 重大客户流失公告 / 政策文件否决 / 管理层不明原因离职

### 7.2 加仓后退出计算
- 止损阈值以**加权平均成本价**计算，不以初始入场价
- 时间止损以**首次建仓日**计算，不以加仓日

### 7.3 做空专属退出规则
- VIX>25：立即cover全部空头，无条件（H2 Agent-09教训）
- 空头盈利>20%：自动减半，剩余设trailing stop（空头反弹风险极高）
- 持仓>20天：每日重新评估thesis是否仍有效
- 大盘系统性反弹（指数单日+2%）：空头必须ABCD分类，A类则cover

### 7.4 同一标的多笔建仓
- 分笔记录entry date和entry price
- 止损触发时按最早一笔入场日期开始计算时间止损
- 分批建仓不改变信心等级（建仓时已确定等级）
