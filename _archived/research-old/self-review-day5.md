- H1 Agent 14 (Nov 3-14): **40%胜率，全场最差**
- 原因：FOMC鹰派信号出现后仍在做多科技 → 方向完全反了
- Agent 14的唯一两个赢家都是空头（AMD -15.5%, PLTR -13%）
- 教训：regime change后24h内必须翻转方向

**在模拟盘的映射：**
- 当前无regime shift检测机制
- 如果下周FOMC/CPI突然转鹰，我会继续按原方向操作直到被打止损
- 这是最贵的错误类型——不是个股选错，是方向选错

**修正：**
- 每个FOMC/CPI/NFP后，强制执行"regime check"：
  1. 对比事件前后的CME fed fund futures
  2. 看VIX变化（>3点 = regime signal）
  3. 看2Y-10Y利差变化
  4. 结论：regime unchanged / shifted dovish / shifted hawkish
- 如果shifted → 24h内调整方向（减多加空 或 减空加多）
