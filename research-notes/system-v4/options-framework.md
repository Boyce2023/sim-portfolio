# 期权使用框架 v4.0
> 生成日期: 2026-05-22 | Agent-16 | 美股交易系统v4.0

---

## 定位声明（一句话）

期权是精准工具，不是策略主轴：做空用Put替代裸空，催化剂用Call加杠杆，高IV用Short Put收权利金。总仓位上限10%（$15K），同时最多2笔。

---

## 全局Sizing规则（优先读）

| 规则 | 数值 |
|------|------|
| 期权总仓位上限（按权利金） | $15K（组合10%） |
| 单笔权利金上限 | $4.5K（组合3%） |
| 同时最多持有 | 2笔 |
| 期权预算独立 | 不计入conviction sizing |
| 到期前强制平仓 | 到期前1天，无例外 |

## 策略一：买Put = 做空首选工具

### 适用场景
做空有明确catalyst但不想承担裸空的无限风险。LULU Put +26.1%是最高收益率期权trade，逻辑清晰：downside thesis确定 + catalyst明确 + 最大亏损封顶。

### 进场条件（全部满足才执行）
- [ ] 空头评分 ≥ 28（符合系统做空标准）
- [ ] 有明确负面catalyst，且catalyst在30天内
- [ ] 期权总仓位加上本笔后 ≤ $15K
- [ ] 当前持有期权 < 2笔

### 执行参数
- **到期日**：catalyst日期 + 7-14天（留缓冲，不卡催化剂当天）
- **行权价**：平值（ATM），delta 0.4-0.6
- **权利金**：≤ $3K（组合2%）
- **买入时机**：catalyst前3-7天（避免买太早被theta decay侵蚀）

### 退出规则（按顺序判断）
1. **盈利50%止盈**：权利金价值翻1.5倍 → 立即平仓，不贪
2. **Catalyst落地次日平仓**：无论盈亏，事件结束后次日强制平仓
3. **到期前5天评估**：如果还未达到50%目标，评估roll vs close（大概率close认赔）
4. **止损线**：权利金亏损50% → 减半（保留一半等待反转机会）
5. **到期前1天**：无论任何情况，强制平仓，永不让期权归零

### 优势与边界
- 优势：最大亏损 = 权利金（$3K封顶），VIX spike时put反而升值
- 边界：不因"感觉会跌"用put，必须有thesis + catalyst，缺一不可

## 策略二：卖Put = 高IV时收权利金

### 适用场景
IV处于高位时，卖out-of-money put收权利金，同时设定一个愿意接货的价格。TSLA Short Put +13.1%，theta decay对卖方有利。

### 进场条件（全部满足才执行）
- [ ] IV Rank > 50%（当前IV高于历史中位）
- [ ] 30天内无已知重大负面catalyst（特别是earnings，IV crush会反向）
- [ ] 愿意以行权价接货（这是硬前提，不是optional）
- [ ] 接货后名义资金占用 ≤ 组合10%（$15K）
- [ ] 期权总仓位加上本笔后 ≤ $15K

### 执行参数
- **到期日**：30-45天（DTE 30-45，theta decay甜蜜区间）
- **行权价**：OTM，delta 0.2-0.3（近期支撑位下方1-2档）
- **权利金目标**：≥ 行权价的1%/月（否则收益不够补偿风险）
- **避开**：earnings前2周内不卖put（IV自然抬升，方向不确定）

### 退出规则（按顺序判断）
1. **权利金收到80%止盈**：主动买回，不等到期
2. **标的跌破行权价**：立即评估是否接货或提前买回（亏损平仓）
3. **接货后处理预案**：接货 → 按正股持仓规则处理（检查bear case，超35%立即卖出）
4. **到期前1天**：如仍在OTM且权利金不足$50，允许持有至到期；否则买回

### 风险提示
卖put的主要风险是标的暴跌后被迫接货，接货后名义亏损可能远超权利金收入。进场前必须真正愿意在行权价持有该标的，否则不操作。

## 策略三：买Call = 催化剂方向加杠杆

### 适用场景
正股仓位已到上限，有高conviction催化剂（A级+），不想错过upside时用call加杠杆。MSFT Call +6.1%，事件驱动逻辑成立时有效。

### 进场条件（全部满足才执行）
- [ ] 正股已持仓且接近或已达仓位上限
- [ ] 有明确正面catalyst，且catalyst在7天内（不是30天，时间要近）
- [ ] Conviction等级 A级或以上（说不出thesis一句话 = 不操作）
- [ ] 期权总仓位加上本笔后 ≤ $15K

### 执行参数
- **买入时机**：catalyst前3-5天（避免买太早theta大量侵蚀）
- **到期日**：catalyst日期 + 7天（不需要太长，事件后无条件退出）
- **行权价**：轻度OTM，delta 0.3-0.5（事件敏感度与权利金的平衡点）
- **权利金**：≤ $3K（组合2%）

### 退出规则（严格执行，无例外）
1. **Catalyst次日无条件平仓**：不管结果好坏，事件结束后次日强制平仓
2. **Catalyst当天大涨50%+**：可当天盘中部分止盈，剩余次日平
3. **到期前1天**：无论任何情况，强制平仓
4. 没有"等反弹"，没有"再等一天" — 持有超过catalyst后是赌博，不是投资

## 期权到期管理（三步检查）

**到期前14天（Theta加速区进入）**
- 评估：当前盈亏情况 + 剩余时间价值
- 如果还没盈利且无catalyst即将触发 → 大概率提前close认赔

**到期前5天**
- 做决定：Roll（换期）vs Close（平仓）
- Roll条件：thesis未变 + 新到期日有catalyst支撑 + 流动性好
- Close条件：其他所有情况

**到期前1天（硬规则）**
- 无条件平仓，永远不让期权到期归零
- 哪怕权利金只剩$50，也要主动平仓（心理习惯培养）

## DO / DON'T 快速参考

**DO**
- 空头thesis + catalyst 30天内 → 买put替代裸空
- IV Rank > 50% + 愿意接货 → 卖put收权利金
- 正股满仓 + 明确catalyst 7天内 → 买call少量加杠杆
- 到期前1天无论盈亏 → 平仓

**DON'T**
- 不做Straddle/Strangle（AAPL 10年胜率41%，排除）
- 不做多腿复杂结构（Ratio Spread/蝶式等）
- 不因"感觉会跌/涨"操作期权，必须有thesis + catalyst
- 不在earnings前一天才买call/put
- 不同时持有超过2笔期权
- 不让期权自然到期归零
- 不在catalyst后继续持有call/put（无论结果如何，次日平仓）
