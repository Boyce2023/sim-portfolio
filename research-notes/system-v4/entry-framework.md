# 入场时机框架 — 美股交易系统v4.0
**Agent-12 | 2026-05-22**

---

## 核心原则（一句话）

美股趋势延续性强，**入场延迟的代价远高于入场稍早的代价**。等"完美回调"在强势股中是系统性踏空，不是风控。

---

## 一、GAP规则量化

所有催化剂事件后的gap，遵循三档规则：

| Gap幅度 | 规则 | 理由 |
|--------|------|------|
| **>15%** | 不追。等3-5个交易日，回调到gap 50%位置再入场 | Gap IS the move（META +12%开盘→20天后-12.2%，大gap即顶部信号） |
| **5-15%** | 当天建半仓，次日确认方向加满 | 中等gap有PEAD漂移空间，分批入场控制成本 |
| **<5%** | 正常入场，一次性建仓至目标仓位 | 小gap意味着市场仍有分歧，趋势未充分定价 |

**胜率验证**：H2 Lesson 14/15 — META +12%开盘gap不追规则验证，Day 1追入者-12.2%；H1 L1规则（>15% gap等3-5天）全程有效。

## 二、催化剂分类与最优入场窗口

### 2.1 财报（Earnings）

**完整决策树：**

```
Pre-earnings:
├─ 近30天内有财报 + conviction A级以上 + 股价回调>5% (非thesis破坏)
│   → DIP BUY（L3规则）。仓位≤8%（L15单笔earnings上限）
│   → 逻辑：市场hedging导致的非基本面下跌，期望值正
│   → 胜率验证：H1 L3 pre-earnings dip buy系统性有效
└─ 近30天内已涨>20% → 不追（pre-earnings run-up已消化预期）

Post-earnings：
Q1: 这是周期股还是高PE成长股？
├─ 周期股（PE<20x）+ Blowout三连击（Revenue beat + EPS beat + Guidance上调）
│   → 财报后T+3~T+5买入（等短期sell-the-news消化）
│   → 持有60-90天（PEAD效应在低估值周期股最强）
│   → 最优案例：MU Dec 2025，+46.9%（H2 Agent 03，Dec 100% WR核心）
│
├─ 成长股（PE 20-40x）
│   Q2: Revenue beat还是仅EPS beat?
│   ├─ Revenue beat + Guidance上调 → T+1买入，持有至蜜月期结束（8-10天）
│   └─ 仅EPS beat（cost cut驱动）+ Guidance平/下调 → 不操作
│
└─ 高PE成长股（PE>40x，市场darling）
    Q3: Guidance方向?
    ├─ 强力上调 → 当天买半仓，T+1确认加满，持有8-10天（L7蜜月期）
    └─ 平或下调 → 即使beat也是卖出/做空窗口（NVDA模式）
        → 胜率验证：H2 NVDA beat +85% YoY仍-9%；H1 L4 NVDA空头财报后+8.6%

Miss情况：
└─ 任何miss → 不接飞刀，等T+3稳定后re-evaluate
   → Miss惩罚-5.6%是beat奖励+0.9%的6倍，不做逆向接刀
```

**最优入场天数**：
- 周期股三连击：T+3~T+5（让市场先消化sell-the-news）
- 成长股Revenue beat：T+1（方向确认后立入）
- 高PE beat/Guidance强：当天半仓，T+1加满
- Miss后重评：T+3观察后决策

### 2.2 政策/FOMC

**规则**：事件后24小时方向确认再入场，不提前猜。

| 情景 | 入场时机 | 操作 |
|------|---------|------|
| 鸽派（降息/暗示宽松） | 声明后2-4小时确认涨势 | 做多成长/科技，减空头 |
| 鹰派（缩表/dot plot上调） | 声明后次日确认跌势 | 做空高久期名字，做多美元 |
| 预期内（市场已price in） | 不操作，等次日数据 | 无催化剂=无入场理由 |

**胜率验证**：H2 Dec 2025 FOMC"鹰派降息"后做空高久期，100% WR（25笔全赢）。关键是等方向确认，不提前押注。

**最优入场天数**：FOMC后T+0（声明后2-4h）或T+1开盘确认。

### 2.3 产品发布（WWDC/CES/GPU Day等）

**规则**：事件前1-2周买入预期，事件当天减仓50%。

| 阶段 | 操作 | 逻辑 |
|------|------|------|
| 事件前2周 | 建仓，预期交易 | 市场未充分定价，入场窗口 |
| 事件前1天 | 持有，不追加 | 期待已接近顶部 |
| 事件当天 | 减仓50% | "买预期卖事实"规律 |
| 事件后3天 | 观察新增信息 | 超预期细节才加回来 |

当前日历应用：WWDC 2026（6月8日）→ AAPL/鹏鼎当前持仓，维持至6月7日，6月8日减仓一半。

---

### 2.4 M&A（并购公告）

**规则**：不追目标公司（gap通常>15%）。观察收购方。

| 角色 | 操作 | 理由 |
|------|------|------|
| 目标公司 | 不追（gap通常20-40%，属于>15%不追规则） | 套利空间已被并购套利基金吃掉 |
| 收购方 | 看整合逻辑，公告后T+3评估 | 初始反应往往过度（市场担心整合风险），3天后情绪稳定再判断 |

---

### 2.5 指数调整（S&P 500入选）

**最优交易窗口（数据验证）**：

| 时间点 | 操作 | 收益来源 |
|--------|------|---------|
| 公告次日（T+1） | 买入 | 被动基金被迫买入效应尚未开始 |
| Effective date前1天（T-2） | 清仓 | 被动买入高峰，兑现溢价 |

平均溢价窗口：公告后到effective date约7-10个交易日，超额收益约3-8%（来自指数基金强制买入）。Effective date之后通常回落。

---

### 2.6 地缘事件

**规则**：事件前入场（L2），不是事件当天（已被price in）。

```
地缘风险上升信号（新闻/情报泄漏阶段）
├─ 卫星图/官员声明/军事调动可见 → 此时入场
│   → 做多：能源(XOM/COP)、军工(LMT/RTX)、黄金(GLD)
│   └─ 做空：高beta科技、运输、旅游
│
└─ 事件当天爆发 → 不追。观望。
   → 此时GAP规则接管（>15%单日跌或涨，等3-5天）

停火/解决信号出现 → 立即做多quality
   → 胜率验证：H2 Apr 8伊朗停火，+88% WR，Apr月均+14.7%
   → 最优操作：停火公告当天买入quality（GOOGL +25.4%，AMZN +24.3%）
```

**最优入场天数**：地缘风险上升期→事件明确前1-5天入场；停火/缓和→当天入场。

---

### 2.7 宏观数据（CPI/NFP/PPI）

**规则**：数据公布后确认方向再行动（不提前猜）。但预期偏差较大时可以提前建小仓。

| 数据偏差 | 操作 |
|---------|------|
| CPI大超预期（通胀）→ 数据后1h确认市场方向 | 做空高久期成长；若FOMC转鹰，追加空头 |
| CPI大低于预期（通缩）→ 数据后1h确认 | 做多成长/REITs；利率敏感股优先 |
| 数据在预期区间 | 不操作，无超额alpha |

**最优入场天数**：数据公布后T+0（1-2小时内）或T+1开盘确认。

## 三、Regime-Dependent入场规则

**VIX是核心环境过滤器，入场逻辑随VIX切换：**

| VIX水位 | 环境判断 | 入场策略 |
|--------|---------|---------|
| **<18** | 低波动、趋势市 | Momentum入场：突破新高买入，不等回调。Momentum评分≥25/35的名字，3%回调即入 |
| **18-25** | 中等波动 | 催化剂入场：必须有具体事件（财报/政策/产品发布）才入场，纯momentum操作减半 |
| **>25** | 高波动、恐惧市 | 逆向入场：只做quality逆向（恐惧极值+多重催化剂）。参考AMD模式：极端恐惧+停火信号→全力买 |
| **>35** | 危机模式 | 只做宏观对冲（GLD/能源/防御）。个股多头仓位降至30%以下 |

**胜率验证**：
- VIX<18对应H2 Dec（100% WR），Apr下旬（90%+ WR）——趋势清晰，追momentum最优
- VIX>25对应H2 Mar（50% WR）——Agent 09在高VIX做空质量股，最大亏损月
- VIX>25+停火信号（Apr 8）→ 逆向做多质量股，88% WR

## 四、Momentum入场细则（VIX<18环境）

基于agent-momentum.md的35分评分框架：

| Momentum分 | 入场条件 | 等待阈值 |
|-----------|---------|---------|
| ≥ 28/35 | 任何3%回调或5日横盘即入 | 不等"理想"入场价 |
| 20-27/35 | 5天横盘后突破前高入场 | 或等新一轮盈利exceed确认 |
| <15/35 | 必须>15%回调+基本面支撑 | 不做纯momentum |

**反"等回调"铁律**：美股强势股回调3-5%，不等10%+。历史：AMD每次3-7%回调后创新高，等大回调者全程踏空（11笔交易，+215%累计）。

## 五、完整入场决策树（实战直用版）

```
步骤1：VIX检查
├─ VIX<18 → 进入Momentum路径
├─ VIX 18-25 → 进入催化剂路径
└─ VIX>25 → 进入逆向路径

【Momentum路径（VIX<18）】
├─ Momentum评分≥25/35？
│   ├─ 是 → 回调3%+即入，不等更大回调
│   └─ 否 → 需要具体催化剂才入场
└─ GAP规则检查（见第二节）

【催化剂路径（VIX 18-25）】
├─ 财报 → 见第二节完整决策树
├─ FOMC/宏观 → T+0确认后入（不提前猜）
├─ 产品发布 → 事件前1-2周建仓，当天减半
├─ 地缘 → 风险上升期提前入，解决当天入
└─ 指数调整 → T+1买，effective前T-2清

【逆向路径（VIX>25）】
├─ 极端恐惧（VIX 30+）+ 多重催化剂（停火/政策转向）
│   → 全力买quality（GOOGL/AMZN/AMD）
│   → 胜率最高setup（H2 Apr 88% WR）
└─ 仅VIX高、无催化剂 → 不操作，等信号
   → H2 Agent 09教训：VIX高时做空quality = 最惨亏损

步骤2：GAP检查（任何情况下适用）
├─ >15% gap → 等3-5天回调到50%位置
├─ 5-15% gap → 当天半仓，T+1加满
└─ <5% gap → 一次性建满

步骤3：仓位上限（来自strategy.md，不在此重复定义）
└─ S≤40% / A≤25% / B≤15% / C≤8% / T≤8%
```

## 七、禁止行为清单（Anti-DO）

1. **追>15% gap当天** — 历史一致亏损
2. **高PE（>50x）beat后加仓** — NVDA模式，beat=卖出信号
3. **VIX>25做空quality compounders** — Agent 09最贵教训
4. **财报后gap>10%持仓不动** — Gap IS the move，当天减50%
5. **等10%+大回调才入强势股** — 美股强势股最大回调通常3-7%
6. **地缘事件当天才建仓** — 已被price in，没有alpha
7. **Miss后T+0接飞刀** — 等T+3再评估
