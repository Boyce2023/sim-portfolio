## 二、Rotation领先信号分析

### 已观察到的有效领先信号

**1. NVDA vs SMH价格背离（最强信号）**

- 规则：SMH创新高 而 NVDA未确认（落后/负偏差），是"反常信号"
- 当前数据（2026年5月初）：SMH较2025年10月底高点+30%，而NVDA仍较同期低-1%
- 历史规律（据io-fund.com分析）："每一次SMH创新高而NVDA未确认，都预示着当前趋势正在消耗动能，为反转布局。"
- 置信度：medium（来自分析师，非一手数据）

**2. 技术指标：PPO负背离 + RSI高位（>70持续6个月+）**

- 来源：StockCharts分析师Tom Bowley
- 信号：月线PPO负背离 + RSI长期超买 → 半导体整体见顶预警
- 局限：这是sector-level信号，不能精确捕捉rotation方向

**3. Volume deceleration（成交量减速）**

- 观察：NVDA反弹时成交量萎缩（"三浪结构 + 成交量下降"）
- 信号：低conviction反弹，上涨持续性存疑
- 置信度：low（无具体数量数据）

**4. 期权市场（NVDA为最多hedged标的）**

- NVDA进入2026年是"市场上最多持仓、最拥挤、最多对冲的标的"
- 拥挤 → 资金边际移向洼地（CPU股、内存股）
- 信号：NVDA期权put/call比率监控（当前数据待补充）

**5. NCNR订单（半导体供应链领先信号）**

- 内存厂商No-Cancel-No-Return订单启动 → 领先内存价格周期约9个月
- 历史上：NCNR启动 → 内存股受益（MU、HBM供应商）→ NVDA配件短缺不影响NVDA但影响资金分配

**6. 新闻/事件型信号**

- DeepSeek型：算法效率突破 → 引发"GPU需求是否过头"质疑 → 先卖NVDA再分析（Jevons Paradox往往后来被证明是过度反应）
- 出口管制型：China ban → 直接收入影响 → 需要评估charge规模
- 竞争型：AMD/AVGO hyperscaler合同 → NVDA市场份额焦虑 → 资金预防性rotation

## 三、Rotation持续时间分析

| 触发类型 | 典型持续时间 | 案例 |
|---------|------------|------|
| 事件驱动（单一催化剂：DeepSeek、财报miss） | 1-2周 → 快速回归NVDA | R3（财报-6%）、R4（DeepSeek-17%后1-2周内止跌） |
| 政策/监管（出口管制、tariff） | 数周至2个月，受谈判进展主导 | R6（H20禁令）；R1（tariff期间） |
| 结构性轮动（AI spending下沉供应链） | 2-3个月，可能更长 | R5（2025年Q1-Q2 memory cycle）；R9（2026年当前进行中） |
| 设备/foundry type | 2-4周，随季报结果修正 | R7（2026年2月） |

**核心结论：没有"标准duration"。持续时间由以下决定：**
1. 有没有对立催化剂（NVDA beat-and-raise → rotation立刻逆转）
2. Rotation标的的基本面是否真正支撑（AMD/MU有earnings支撑 vs 纯sentiment flight）
3. 市场整体risk environment（risk-off全部跌，rotation意义弱化）

## 四、典型Rotation顺序

根据2024-2026年数据观察，GPU-to-Stack rotation的典型路径：

```
第一波（即时反应，1-5日）：
NVDA承压/下跌 → AMD小幅受益（最直接替代品，叙事最清晰）

第二波（1-4周，需基本面支撑）：
→ Memory（MU/HBM）：GPU越多，HBM需求越多；且memory有独立周期
→ Networking/Optics（MRVL/ANET/LITE/FN）：数据中心互联瓶颈受益

第三波（1-3个月，结构性）：
→ CPU/EPYC（AMD CPU + INTC）：agentic AI工作负载转向inference
→ Semicap设备（AMAT/LRCX/KLAC）：长周期capex受益，与GPU主导权无关
→ Foundry（TSM）：无论谁的GPU都要台积电制造
```

**注意**：这个顺序不是铁律，主要受以下变量干扰：
- 各标的earnings时间节点（beat → 资金单独涌入）
- 宏观风险情绪（tariff/geopolitical → 全面risk-off）
- 个股特定事件（AVGO与Google/Meta的ASIC大单 → 独立行情）

## 五、当前状态评估（2026年5月27日）

### Rotation正在发生的证据

1. **价格分歧**：AMD +105% YTD vs NVDA +17% YTD（截至5月中旬）；AMD trailing 1-month +70.4% vs NVDA +13.4%
2. **SMH背离**：SMH高于2025年10月底+30%，NVDA仍低于该水平-1%
3. **资金去向**：AMD（EPYC + MI400）、MU（HBM4 + DRAM超级周期）、INTC（苹果foundry + agentic AI）
4. **结构性叙事**：AI从训练转向agentic推理 → CPU需求增量；ASIC分流（AVGO MTIA/Google TPU）→ NVDA市占从90%→68%
5. **挤兑迹象**：NVDA被描述为"最拥挤交易"，拥挤交易动能先于基本面衰竭

### 反驳/风险

1. NVDA即将发布财报（2026年5月）：一旦beat-and-raise（Blackwell强势），rotation可能"数日内逆转"
2. 当前rotation的标的（AMD特别是）的数字尚未完全经过earnings验证
3. MU UBS目标价大幅上调（$1,625 from $535）可能是sentiment极值信号而非进入机会

### 结论

**当前（2026年5月）存在真实的NVDA→AMD/MU/CPU rotation，且持续时间已超过1个月，具有结构性叙事支撑（agentic AI）而非纯事件驱动。但关键翻转条件是NVDA 2026年5月财报。**

## 七、对Rotation Detection Protocol的输入建议

### 检测规则草案

1. **主信号**：SMH vs NVDA 30日相对强弱背离（SMH>0, NVDA<0）→ 发出rotation alert
2. **确认信号**：AMD或MU 5日RSI > 65 同时 NVDA 5日RSI < 50 → rotation活跃
3. **新闻触发器**：出口管制/算法突破/竞争合同 → 立即评估NVDA revenue impact vs sentiment impact（后者往往反应过度）
4. **持续时间阈值**：rotation > 3周且有earnings支撑 → 视为结构性，可建仓rotation标的
5. **退出信号**：NVDA财报后beat-and-raise + Blackwell/Rubin roadmap强化 → rotation逆转概率 > 70%

### 目标标的优先级（rotation发生时）

| 优先级 | 标的 | 理由 |
|------|------|------|
| P1 | AMD | 最直接GPU替代叙事；有EPYC CPU + MI系列双催化剂 |
| P1 | MU | HBM/DRAM超级周期独立逻辑；不完全依赖NVDA输赢 |
| P2 | AVGO | ASIC定制芯片长期份额提升；但已有部分priced in |
| P2 | INTC | agentic AI + Apple foundry催化剂；但基本面转型风险高 |
| P3 | AMAT/LRCX | 长周期设备需求；rotation弹性低于直接AI芯片股 |
| P3 | ANET/MRVL | networking受益但周期与GPU更紧密，rotation beta较低 |
