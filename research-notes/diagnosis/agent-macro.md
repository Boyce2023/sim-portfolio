# Agent-8: 宏观变量传导链分析
> 美股交易系统v4.0重建 | 2026-05-22

## 核心结论

两次regime shift error的共同特征：**在宏观信号已经发出后，继续持有对该信号方向错误的板块敞口**。H1 Agent-14和H2 Agent-09的亏损不是选股错了，是regime框架空缺——没有一个强制检查点要求"宏观环境变了，持仓要重新归零检验"。

---

## 五条传导链（数据验证版）

### 链1：利率上升 → 科技估值压缩

**机制**：高增长科技股DCF敏感于折现率，10Y yield每+50bp，理论上压低15-20x PE倍数的成长股约10-15%。

**yf实测**：
- TNX在2025-12-18（FOMC鹰派，dot plot 4→1次降息）后急升，对应H2 Jan WR仅55%——是全年第二差月份
- 2026-05-15至05-19：TNX从4.59飙至4.67（+8bp，三周新高），与当前市场小型tech震荡同步
- 当前TNX水位：4.59（05-21），逼近年内高位；**持续高于4.5%的周期里做多QQQ/XLK需要显著的earnings支撑才能compensate**

**操作触发**：TNX单周+20bp以上 → 立即减少高PE成长股（forward PE>30x）敞口；TNX突破4.7% → 禁止新建成长股多头，转向value/financials

**H2验证**：Dec FOMC鹰派降息后科技板块承压，但H2 Dec整体100%WR——说明做的是金融/能源方向而非盲目做多科技

---

### 链2：美元走强 → 跨国公司earnings miss

**机制**：S&P500约40%收入来自海外。DXY每+5%，跨国公司EPS平均受损约2-3%（因货币换算损耗）。强美元同时压制大宗商品价格（以USD计价）。

**2025实测**：
- 关税宣布后，美元先涨后反转跌（市场定价滞胀风险），DXY 2025全年走弱
- J.P. Morgan预测EUR/USD升至1.22（2026-03），意味着美元持续偏弱
- H2 Jan（Trump就职+关税初始）55%WR：关税宣布 → 美元强 → 跨国大盘股遭抛 → 短期错配

**操作触发**：
- DXY单周+3%以上 → 减持海外收入占比>50%的跨国公司（AAPL/MSFT/GOOGL海外收入均>40%）
- DXY持续走弱 → 做多国际EM ETF（EEM/VWO）和大宗商品（COPX）
- 关税升级消息 → **先看美元方向**，再决定是减持跨国股还是加仓本土受益股（KR/WMT/BJ）

---

### 链3：油价飙升 → XLE涨 / XLY跌（反向对冲结构）

**机制**：油价对XLE相关性约74%（历史数据）。油价每+10%，消费者可支配收入缩减约0.3-0.5%（汽油支出+输入性通胀），直接压制XLY/AMZN/HD等。

**yf实测（CL=F）**：
- 2026-04-13：$99.08，同期伊朗-霍尔木兹危机解除前夕
- 2026-04-17：$83.85（-15%，停火消息），对应H2 Apr V型反弹开始（88%WR最强月）
- 2026-05-15至05-18：$105→$108，油价反弹（OPEC+延续减产）
- H2 Mar（伊朗危机）：50%WR，油价高位 → XLE虽涨但整体策略WR最差 → Agent-09在高油价regime下无法有效配置其他板块

**操作触发**：
- 地缘冲突导致油价单周+10% → 立即做多XLE/OXY，同时减持XLY/AMZN/TSLA
- 停火/和谈消息 → 减XLE，转XLY（消费复苏预期）
- 油价在$80-100区间震荡 → 中性，无强烈板块信号
- 油价持续>$100 → 关注通胀二次抬头，TNX可能随之上行，引发链1效应

### 链4：金价上涨 → 避险情绪高涨（不是单纯做多GLD）

**机制**：金价上涨通常由两个驱动：(a) 实际利率下行（名义利率不变但通胀预期上升）；(b) 避险需求（地缘/金融危机）。两种驱动对股票的含义截然不同。

**yf实测（GC=F）**：
- 2026-04-13：$4,742，达到历史高位区间
- 2026-04-17：$4,857.60（阶段顶），对应伊朗危机高峰
- 2026-04-28至05-04：$4,591→$4,519（金价回落），同期停火+V型反弹启动（88%WR）
- 2026-05-15至05-19：$4,555→$4,506（持续下行），同期TNX走高 → 实际利率上行压金价

**区分两类金价上涨**：

| 驱动类型 | 金价特征 | 伴随信号 | 正确操作 |
|---------|---------|---------|---------|
| 避险型 | 急涨+波动大 | VIX同步飙升，债券收益率下行 | 做多GLD+TLT，减持成长股 |
| 实际利率型 | 缓涨+波动小 | VIX温和，TNX温和下行 | 做多GLD+TIPS，保留科技 |

**H1验证**：Jackson Hole后GLD +7.4%（市场定价"鲍威尔转鸽"，实际利率预期下行驱动）。GLD 20笔trades +78.7%累计 — 说明金价上涨本身是可交易信号，关键是识别驱动类型。

**操作触发**：
- 金价单周+3%且VIX>25 → 避险型，加仓GLD，全面减持进攻板块
- 金价单周+3%且VIX<18 → 实际利率驱动，加仓GLD但保留科技多头

### 链5：关税升级 → 供应链板块分化（本土受益 vs 进口依赖受损）

**机制**：关税是最复杂的宏观信号，因为同一事件同时产生赢家和输家，且市场定价速度参差不齐。

**H2实测**：
- Jan Trump就职（关税初始）：55%WR — 市场消化不确定性，跨国股承压，本土股小幅受益
- May 中美贸易休战：77%WR — 最强月份之一 — 不确定性消除 = 全面修复

**板块分化矩阵**：

| 关税方向 | 受损板块 | 受益板块 | 关键ETF |
|---------|---------|---------|---------|
| 对华加关税 | 苹果供应链/零售商(WMT/TGT) | 本土制造/半导体设备 | AMAT/LRCX |
| 对华宣布豁免 | 无明显受损 | 科技/零售全面反弹 | QQQ/XRT |
| 全球关税升级 | 跨国大盘股/EM | 本土服务业/医疗 | XLV/XHB |
| 贸易休战 | 无 | 所有板块V型修复 | SPY做多 |

**操作触发**：
- 新关税宣布 → 24h内检查持仓的"进口依赖度"，减持高风险，加仓本土受益
- 贸易谈判积极信号 → 先买最受压的跨国股（risk-on先行）

## FOMC/CPI/NFP后24h强制检查清单

发布后**次日开盘前**必须完成：

1. **TNX水位**：是否突破4.5%（警戒）或4.7%（红线）？单日变化幅度>15bp?
2. **金价+VIX同步性**：两者同涨 = 纯避险；金价涨而VIX稳 = 实际利率驱动
3. **持仓PE分布**：平均forward PE>25x的持仓，若TNX>4.5%，触发"减仓科技"规则
4. **美元方向（DXY）**：单周+3%以上 → 检查跨国敞口
5. **油价**：$80以下 = 中性；$80-100 = 能源中性；$100以上 = 能源多+消费空

## Regime Shift Detection：防止Agent-09级别错误

两次最贵错误的共同pattern：**宏观regime已切换，持仓还停在上一个regime的配置里**。

**Regime快速分类（2×2矩阵）**：

| | 利率上行 | 利率下行 |
|---|---|---|
| **风险偏好高（VIX<18）** | 金融/能源 Value | 科技/成长/EM |
| **风险偏好低（VIX>25）** | 现金/GLD/TLT | GLD/防御/TLT |

**Regime切换触发条件（任一满足即重新定性）**：
- VIX单周从<18跳至>25（或反向）
- TNX单周变化>25bp
- 油价单周变化>10%
- FOMC点阵图意外改变（cuts数量变化>=2次）

**切换后强制动作**：持仓全面归零重新审视，按新regime配置从头构建，不允许"继续持有旧仓位等反弹"。
