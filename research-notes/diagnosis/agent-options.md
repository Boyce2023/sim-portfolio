## 四大策略适用场景分析

### 1. 买Put（做空替代）— 首选工具
**适用场景**：
- 有明确空头thesis但不想裸空（风险无限）
- 财报前预期miss，catalyst明确且30天内
- 技术形态破位，短期方向性下行bet

**H1验证**：LULU put +26.1%，Agent-02期间（5/0全胜）

**参数建议**：
- 到期日：30-45天（theta decay可控）
- 行权价：平值或轻微虚值（delta 0.4-0.6）
- 高conviction空头才用，不是"感觉会跌"

**最大亏损**：权利金（可控，比裸空好）

### 2. 卖Put（收权利金）— 次选，需要高IV
**适用场景**：
- 想以更低价格买入某标的（愿意接货）
- IV Rank > 50%（高于历史中位）
- 短期无重大催化剂，预期横盘或小涨

**统计基础**：卖put策略理论胜率70-85%（premium随时间衰减对卖方有利）

**参数建议**：
- 30天到期，delta 0.30以下（OTM保护）
- 避开earnings前2周（IV crush可能反向）
- 行权价选近期支撑位下方

**风险**：标的暴跌时被强制接货+账面亏损

### 3. 买Call（catalyst加杠杆）— 情景专用
**适用场景**：
- 有明确催化剂（WWDC/财报/监管批准）且方向性判断高confidence
- 正股仓位已满（比如AAPL已到上限），想额外加杠杆
- 催化剂30天内，事件驱动逻辑清晰

**H1验证**：MSFT call +6.1%

**参数建议**：
- 催化剂前3-5天买入（避免买太早theta decay侵蚀）
- 催化剂当天或前后卖出（不持仓过catalyst后）
- delta 0.5-0.7（平值，最大事件敏感度）

**风险**：方向判断错误全损权利金（≤2-3%组合占比可接受）

### 4. Straddle（跨式）— 模拟盘不推荐
**原因**：
- Long straddle历史胜率差：AAPL earnings straddle 10年胜率41%，年化-1.31%；FB straddle仅27%胜率
- 需要精确判断波动幅度，模拟盘信息优势不在这里
- IV在earnings前自然抬升（"IV crush"效应），买straddle实际上是买贵的波动率

**结论**：模拟盘明确排除straddle，这是专业做市商的工具。

## 期权 vs 正股：什么时候用期权替代？

| 场景 | 正股 | 期权（买put/call） |
|------|------|-------------------|
| 多头长期持有（A/S级） | 正股，享受复利 | 不用期权（theta侵蚀长期头寸） |
| 空头thesis（30-60天） | 不裸空（风险无限） | **买put是首选** |
| 催化剂方向性bet | 正常建仓 | **买call加杠杆**（仓位已满时） |
| 高IV收权利金 | 不操作 | **卖put**（愿意接货时） |
| 对冲持仓下行风险 | 持有 | 保护性put（成本高，一般不用） |

**杠杆 vs 时间损耗权衡**：
- 期权杠杆倍数：买平值call/put约5-10倍
- Theta decay每天侵蚀：30天期权每天约损失权利金的1/30
- 结论：只有在catalyst明确且30天内，杠杆收益才能跑赢theta decay

## $150K模拟盘期权配置建议

**期权仓位上限：总组合10-15%（约$15-22K）**

理由：

## DO / DON'T清单

### DO
- 空头thesis有catalyst → 买put替代裸空
- IV高位（IV Rank > 50%）+ 愿意接货 → 卖put收权利金
- 正股仓位已满 + 有catalyst → 买call少量加杠杆
- 期权到期前1周无对位 → 直接平仓，不等到期
- 每笔期权买入前写进场检查表（同正股要求）
- 买put时delta选0.4-0.6，期限30-45天

### DON'T
- 不做straddle/strangle（胜率差，需要精确判断波动幅度）
- 不做ratio spread / 蝶式等多腿复杂策略（Greeks管理要求高）
- 不在期权上用超过3%单笔（权利金损失可接受）
- 不在earnings前一天才买call/put（IV Crush时机太难把握）
- 不因"感觉会跌"就买put，必须有thesis + catalyst
- 不持有期权超过catalyst事件后（不管结果如何，catalyst日次日平仓）
- 不同时持有3笔以上期权（复杂度管理）

## 操作SOP（期权进场三步）

**Step 1 — 前提确认（缺一不允许）**
- [ ] 有明确方向性thesis（一句话说清楚）
- [ ] catalyst在30天内
- [ ] 期权占比不超过总组合15%

**Step 2 — 参数选择**
- [ ] 到期日：catalyst日期 + 7-14天（留缓冲）
- [ ] 行权价：买put/call选平值，卖put选OTM 1-2档
- [ ] 权利金 ≤ 组合的2-3%

**Step 3 — 出场纪律**
- [ ] Catalyst事件结束后次日无条件平仓
- [ ] 剩余7天到期时无论盈亏平仓
- [ ] 止损线：权利金损失50%时减半
