# A股发现系统 v2.0 — Anti-Cocoon Discovery System

> 从US §10移植+A股本土化重构。28-agent研究支撑。
> 核心转变: v1.0的手动板块扫描 → v2.0的**信号驱动发现** + **量化反茧房** + **F20市场状态耦合**

---

## §1 核心原则

**信息茧房问题**: 只扫已知名字 → 错过PCB材料链(生益电子+67%)/MLCC(风华高科+54%)/存储(江波龙+89%)。成功本身强化茧房: 已持仓赚钱 → 更多时间看持仓 → 搜索算法锁定 → 认知围墙越来越厚。

**解法**: 用thesis-agnostic的SIGNAL驱动发现，不用thesis驱动发现。

**三条铁律**:
1. 板块集中不是问题，板块盲区才是。在正确方向集中可以赚大钱，但必须是"看过所有板块后选择集中"
2. 能力圈 = 分析方法(供应链穿透/催化剂识别/量价配合)，不是行业标签。"不是我的行业"不是合法rejection
3. Discovery排在F20 Market Breathing之后、持仓分析之前。零发现也必须跑scanner并记录

---

## §2 F20 Market Breathing — 主控开关

> 所有scanner在F20呼气期降功率运行。这是A股与美股Discovery System的最大结构差异。

| 市场状态 | 判定标准 | Scanner功率 | 适配策略 |
|---------|---------|------------|---------|
| **吸气期** | 中证1000跑赢沪深300 + 成交>20日均量130% + 北向连续3日净流入 | **100%全功率** | 动量优先/小票/高弹性/快进快出 |
| **中性期** | 无明确方向信号 | **70%** | 龙头+事件驱动，Scanner正常但降低PRIORITY swap频率 |
| **呼气期** | 沪深300跑赢中证1000 + 缩量 + 北向持续净流出 | **30%仅S3+S4运行** | 防御/减仓/确定性高的绩优股。新发现进MONITOR不进INVESTIGATE |

**状态更新**: 每周一更新，3个交易日以上的一致方向才算有效状态转换。

## §3 六大信号扫描器

> 从US的5个scanner重新设计为6个，适配A股微结构。
> 核心差异: S2量价背离(替代Volume Breakout, A股放量突破IC为负); S5政策催化(替代52W New Highs, A股新高有反转效应); S6涨停质量(A股特有)。

### S1: PEAD / 盈余惊喜 (Post-Earnings Announcement Drift)

**原理**: A股盈余公告后高开(AOG)是最强快信号，RankIC +4.05%, ICIR 3.88, 月胜率88.9%。信息摩擦产生的短期alpha。

**扫描规则**:
```
筛选条件:
  - 业绩预告/快报/正式财报发布后5个交易日内
  - 归母净利同比增速 > 30% (扭亏 > 预增, 权重更高)
  - 排除已在portfolio + watchlist的标的

优先级排序:
  P1: 扭亏为盈 (0→正, 信号最强)
  P2: 加速增长 (Q-on-Q增速提升)
  P3: 超预期幅度 > 20% (实际 vs Wind一致预期)
```

**信号有效窗口**: 5-13个交易日（公告后漂移）。超过15天衰减明显。
**频率**: 每周一扫描 + 财报季(1/4/7/10月)加密到每日。
**数据源**: 东方财富业绩预告/快报API。

---

### S2: 量价背离 (Volume-Price Divergence)

**原理**: ★★★**A股反常识** — 放量突破在A股统计上是负信号(IC -3%~-5%)。真正有效的是: 缩量上涨(筹码锁定→低成本拉升) + 放量回调后的缩量企稳(洗盘结束信号)。

**扫描规则**:
```
正向信号(发现候选):
  - 连续5日上涨但成交量逐日递减 (缩量上涨, 筹码高度集中)
  - 放量下跌3日后缩量止跌 + 量能缩至放量日的30%以下 (洗盘结束)
  
负向信号(排除/预警):
  - 高位放量滞涨 (天量见天价)
  - 连续3日放量上涨 + 换手率>15% → 短期见顶概率>60%
```

**与US S2的本质差异**:
- US: Volume Breakout = Nokia pattern = 正信号
- A股: Volume Breakout = IC为负 = 反信号
- 原因: A股散户占比高(交易量60%+), 放量意味着散户涌入 = 接盘

**信号有效窗口**: 3-8个交易日。
**频率**: 每周一。
**数据源**: 东方财富push2 API量价数据。

---

### S3: 机构共振 (Institutional Convergence)

**原理**: 单一机构信号噪声大。三方共振(机构调研 + 基金加仓 + 北向/社保)才有信号价值。北向资金是确认信号而非发现信号。

**扫描规则**:
```
三方共振检测:
  信号A: 机构调研 — 过去30天接待机构数>20家 (互动易/深交所公告)
  信号B: 基金加仓 — 季报显示>5家基金新进/加仓 (季度更新)
  信号C: 北向/社保 — 北向持股比例月增>1ppt OR 社保基金新进

共振等级:
  3/3共振 → PRIORITY (直接进入4-Gate)
  2/3共振 → INVESTIGATE (15min deep look)
  1/3 → MONITOR (加入expanded watchlist)
```

**关键发现(北向资金agent)**:
- 龙虎榜仅有游资 → 后续平均下跌
- 仅有机构 → 后续平均上涨
- **游资+机构共振(前五中2家以上知名游资+机构同时出现) → 最强信号**

**信号有效窗口**: 15-30个交易日（比量价信号慢）。
**频率**: 每周一(北向日频数据) + 每季(基金季报)。
**数据源**: 东方财富机构调研/北向数据 + 季报持仓变动。

### S4: Anti-Sector (板块盲区)

**原理**: 与US §10相同——专门扫描零持仓/零研究的板块。但A股用申万二级行业(31个)替代GICS。

**扫描规则**:
```
每周一检查:
  1. 统计当前portfolio + watchlist的行业分布
  2. 标记零覆盖的申万二级行业
  3. 对每个零覆盖行业:
     - 取过去20日涨幅TOP 3标的
     - 检查是否有S1/S5/S6信号共振
     - 有 → INVESTIGATE; 无 → 下周再查

强制升级:
  零覆盖行业中出现3+标的在Anti-Portfolio TOP 50 → 板块盲区警报
```

**11个GICS一级行业A股版**:
能源 / 材料 / 工业 / 可选消费 / 必需消费 / 医疗保健 / 金融 / 信息技术 / 通信服务 / 公用事业 / 房地产

**频率**: 每周一。
**数据源**: 申万行业分类 + 东方财富板块行情。

### S5: 政策催化 (Policy Catalyst)

**原理**: ★★★**A股特有** — 替代US的52W New Highs(A股新高是反转信号)。A股政策驱动行情的5阶段模型:

```
阶段1: 政策发布 (T+0)          → 概念股涨停潮, 龙头混乱
阶段2: 主线确认 (T+1~T+2)      → 资金集中到真受益标的
阶段3: 机构跟进 (T+3~T+7)      → 最优入场窗口★
阶段4: 卖方覆盖 (T+7~T+15)     → 共识形成, alpha衰减
阶段5: 散户涌入 (T+15+)        → 拥挤, 准备退出
```

**扫描规则**:
```
政策源监控:
  - 国务院/发改委/工信部/科技部/财政部/央行/证监会 官网
  - 新华社 / 人民日报 政策定调
  - 地方政府产业政策(重点: 上海/深圳/北京/合肥)

筛选条件:
  - 新政策文件 → 识别最直接受益标的(不超过5个)
  - 优先: 供给侧政策(产能/牌照/准入限制) > 需求侧补贴
  - T+3~T+7为最优扫描窗口(阶段3)
  
排除:
  - 已进入阶段5(散户涌入, 成交量>20日均量300%, 换手率>20%)
  - 纯概念无实质受益(检查政策文件是否点名具体技术/产品)
```

**信号有效窗口**: T+3~T+15（最优T+3~T+7）。
**频率**: 实时(政策发布后立即触发) + 每周一回顾。
**数据源**: 政府官网 + 新华社 + 东方财富政策板块。

---

### S6: 涨停质量 (Limit-Up Quality)

**原理**: ★★★**A股独有** — 涨停板是A股最强的短期信号源。但不是所有涨停都有效:
- **缩量封板**: 次日成功率91.7%（供给被锁死）
- **放量封板**: 次日成功率显著更低（筹码松动）
- **首板次日溢价**: 10%涨停时代平均+8.0%, 20%涨停时代降至+3.2%

**扫描规则**:
```
首板筛选:
  - 当日涨停 + 非ST/非次新(<60日)
  - 成交量 < 前5日均量 × 2 (缩量封板优先)
  - 封板时间 < 14:00 (早封优于晚封)
  - 炸板次数 ≤ 1 (稳定封住)

二板确认龙头(赵老哥定律):
  - 首板筛选通过 + 次日继续涨停
  - 题材唯一性: 该主题方向涨停股≤3只(不是普涨)
  - 龙虎榜: 前五中有知名游资+机构共现 → 信号升级

信号分级:
  缩量封板 + 早封(<14:00) + 二板确认 → PRIORITY
  放量封板 OR 尾盘封板 → MONITOR only
  炸板2次以上 → EXCLUDE
```

**注意**: 此scanner对执行成本极度敏感。滑点从0.05%→0.30%时累计收益从151.6%→18.4%。模拟盘可跑，但仓位严格限制在C级(≤8%)。

**信号有效窗口**: 1-3个交易日（极快衰减）。
**频率**: 每日收盘后(T+0日扫描, T+1执行)。吸气期每日跑; 呼气期停止。
**数据源**: 东方财富涨停数据 + 龙虎榜。

### 信号衰减规律

A股信号衰减显著快于美股:
- **极快信号(1-3天)**: S6涨停质量, S1的AOG(公告高开)
- **快信号(3-8天)**: S2量价背离
- **中速信号(5-13天)**: S1 PEAD漂移, S5政策催化
- **慢信号(15-30天)**: S3机构共振

**操作含义**: A股scanner产出的INVESTIGATE/PRIORITY，必须在信号窗口内完成评估。超过窗口的自动降级:
- PRIORITY超过信号窗口 → 降为MONITOR
- INVESTIGATE超过信号窗口 → 移除

## §5 Anti-Portfolio Protocol

> 每周五执行。核心问题: "这周市场涨最多的50只票里，有多少我连名字都没听过？"

### 执行步骤

```
Step 1: 取全市场滚动4周涨幅 TOP 50
  - A股: 排除ST/退市预警/日均成交<¥5000万的
  - 排除: portfolio + watchlist + 30天内研究过的名字

Step 2: 分类剩余标的
  APG_known: 研究过但主动排除的 → 可接受(有rejection reason记录)
  APG_blind: 从未听说/研究过的 → 真正的茧房信号★

Step 3: APG_blind分析
  来自同一行业 ≥3只 → 板块盲区警报, 触发S4全扫描
  个别标的 → 检查驱动力(政策/财报/主题), 值得追踪的加入MONITOR

Step 4: 补充数据源(季度更新)
  - 北向资金持仓变动TOP 30 (排除已知名字)
  - 融资余额增幅TOP 20
  - 涨停主题统计: 过去4周涨停次数最多的主题方向
```

### 本周实测数据(2026-05-19~23)

| 分类 | 数量 | 代表标的 | 发现 |
|------|------|---------|------|
| APG_blind | 11/30 | 生益电子/风华高科/江波龙 | PCB材料链+MLCC完全在覆盖盲区 |
| APG_known | 8/30 | 巨化/紫金/宁德 | 研究过但未建仓 |
| 已覆盖 | 11/30 | 新易盛/沪电/鹏鼎 | 持仓标的 |

**APG_blind率 = 37%** → 黄灯(30-50%区间), 主要盲区: 被动元件/存储芯片/PCB材料链

## §6 Signal Aggregation Rules

> 同一标的触发多个scanner → 信号升级

| Hits | Level | Action | 时间预算 | 信号窗口 |
|------|-------|--------|---------|---------|
| 1 | MONITOR | 加入expanded watchlist | 0 min | 记录日期, 窗口到期自动移除 |
| 2 | INVESTIGATE | 15min deep look + SABCD初评 | 15 min | 较短scanner的窗口为准 |
| 3+ | PRIORITY | vs最弱conviction持仓对比 → 4-Gate | 30 min | 立即执行 |

### 4-Gate评估(PRIORITY标的)

```
Gate 1: F20市场状态 — 呼气期暂停所有新建仓
Gate 2: Bear Case二元过滤 — ≤40%通过, >40%=D排除
Gate 3: SABCD Conviction评级 — thesis强度+催化剂明确度+竞争壁垒
Gate 4: 仓位容量 — 现金≥目标仓位 × 1.5(预留止损空间)
```

### 信号共振加权(A股特有)

某些scanner组合的共振信号强度远超简单叠加:

| 组合 | 共振含义 | 处理 |
|------|---------|------|
| S1(PEAD) + S3(机构共振) | 业绩爆发+机构提前布局 | 直接PRIORITY |
| S5(政策) + S6(涨停) | 政策驱动+市场确认 | PRIORITY, 但注意阶段判断 |
| S2(缩量上涨) + S3(机构加仓) | 筹码锁定+主力入场 | PRIORITY |
| S1(PEAD) + S6(涨停) | 业绩公告+涨停确认 | PRIORITY, 极短窗口(1-3天) |
| S4(盲区) + S5(政策) | 全新板块+政策驱动 | INVESTIGATE (需更多时间理解) |

## §7 Override Protocol

> PRIORITY通过4-Gate后，与最弱conviction持仓比较

```
Discovery Grade > Weakest Holding Grade → displacement swap
同级 → Discovery有近期catalyst(30天内) → swap; 否则MONITOR

约束:
  吸气期: 每周 ≤2次 Override swap
  中性期: 每周 ≤1次
  呼气期: 禁止Override, 所有PRIORITY进MONITOR等吸气

执行后写入portfolio_state.json备注: "Discovery Override from S{X}+S{Y}"
```

## §8 PRIORITY Rejection Protocol

> 每个PRIORITY标的必须二选一: INVESTIGATE 或 REJECT(写rejection reason)

### 合法Rejection

```
✓ "信号为一次性事件(并购/拆分/减持), 非持续趋势"
✓ "流动性不足(日均成交 < ¥5000万)"
✓ "已在expanded watchlist, 上周已investigate"
✓ "Bear case >40%, D级排除"
✓ "F20呼气期, 进MONITOR等吸气"
✓ "涨停板已进入阶段5(散户涌入, 换手率>20%)"
```

### 非法Rejection

```
✗ "不是我的行业" ← 你的edge是分析方法，不是行业标签
✗ "没时间" ← Discovery排第一，时间不够砍后面的
✗ "感觉不熟" ← 不熟才是Discovery的意义
✗ "传统行业没意思" ← 巨化涨了85%你说没意思？
✗ "这票我不喜欢" ← 无具体理由的感觉不算
✗ "估值太高了" ← 需要具体数字, 不是感觉
```

## §9 Anti-Cocoon Dashboard

> 8个可量化指标 + Cocoon Index综合评分。每周五sizing reconciliation后检查。

### 8个指标

| # | 指标 | 定义 | 计算 | 绿灯 | 黄灯 | 红灯 |
|---|------|------|------|------|------|------|
| M1 | Discovery Rate | 每周净新增观察标的 | watchlist新增数/周 | ≥3个 | 1-2个 | 0个(连续2周) |
| M2 | Pipeline Freshness | 观察池30天新标的占比 | N(30d内新增)/N(总) | ≥20% | 10-20% | <10% |
| M3 | Sector Coverage | GICS一级行业覆盖度 | N(覆盖行业)/11 | ≥7/11 | 5-6/11 | ≤4/11 |
| M4 | Anti-Portfolio Gap | TOP50中完全不认识的比例 | APG_blind/50 | <30% | 30-50% | >50% |
| M5 | Rejection Quality | 被拒标的后续是否真的不涨 | N(justified)/N(total) | >60% | 40-60% | <40% |
| M6 | Source Diversity | 信息源类型多样性 | 6类信源覆盖比 | ≥4类 | 2-3类 | ≤1类 |
| M7 | Contrarian Score | 与市场共识偏离度 | 持仓F15评分均值 | 0.3-0.7 | <0.3或>0.8 | <0.2 |
| M8 | Attention Balance | 各行业研究时间分布 | 30天内活跃行业数/11 | ≥6/11 | 4-5/11 | ≤3/11 |

### M6 Source Diversity六类信源

| Type | 来源类型 | 举例 |
|------|---------|------|
| A | 一手来源 | 公司公告/交易所公告/管理层电话会 |
| B | 卖方研究 | 券商研报/晨会纪要 (占比不超过40%) |
| C | 独立媒体 | 财经媒体原创报道/行业自媒体 |
| D | 社交/散户 | 雪球/东方财富股吧/Twitter |
| E | 另类数据 | 产业链数据/高频数据/卫星/专利 |
| F | 对立方来源 | 做空报告/空头论坛/bear case专项搜索 |

### Cocoon Index计算

```
Cocoon Index (CI) = 100 - Σ(红灯罚分) - Σ(黄灯罚分)

权重分配:
指标              权重   红灯罚分  黄灯罚分
─────────────────────────────────────────
M1 Discovery Rate  15%    -15      -8
M2 Pipeline Fresh  10%    -10      -5
M3 Sector Coverage 15%    -15      -8
```

## §11 历史回溯验证

> 中际旭创2024 + 寒武纪2023的S级机会信号回测，验证scanner有效性

### 寒武纪688256 (2023年1月→6月, +394%)

**信号时间线**:
| 日期 | 事件 | 哪个Scanner能捕获 | 领先天数 |
|------|------|------------------|---------|
| 2022-11-30 | ChatGPT发布 | S5政策催化(全球事件) | **90天** |
| 2023-01-30 | 同花顺创建ChatGPT概念板块 | S5政策催化 | 53天 |
| 2023-02-02 | 成交量放大4倍(92,852手) | S2量价背离(底部放量) | **46天** |
| 2023-02-07 | 成交量突破15万手 | S6涨停质量前兆 | 37天 |
| 2023-02-13 | +10.61%大阳线 | S6涨停质量 | 32天 |
| 2023-03-17 | 第一个+20%涨停 | S6涨停质量(确认) | 0天(已晚) |

**核心教训**:
1. **A股认知时差**: ChatGPT发布→A股概念板块创建: 61天。持续关注全球AI事件可领先60天
2. **成交量放大先于涨停43天**: 两只标的(寒武纪+中际旭创)均在第一个涨停前43天出现量能4倍放大
3. **只扫涨停榜已经晚了**: 真正的发现窗口在量价异动期，不是涨停期
4. **唯一性溢价**: 寒武纪当时是A股唯一AI芯片股，稀缺性标的在主题行情中被追到极端估值
5. **Bear case全部无效**: 持续亏损/客户集中/商业化困难——在事件驱动+稀缺性溢价下全部被市场忽略

### 中际旭创300308 (2023年2月→2024年, +570%)

**信号指纹与寒武纪高度一致**:
- 底部横盘6个月 → 放量突破 → 涨停确认
- 两只股票3月22日**同日**放量涨停
- 旭创成交量异动信号**更早更明显**(创业板流动性更好)

### S级机会信号指纹(通用模式)

```
指纹1: 全球事件触发 + A股1-2个月认知时差 → 最优发现窗口
指纹2: 底部横盘≥6个月 + 成交量4倍放大 → S2 scanner核心信号
指纹3: 唯一性标的(A股仅此一只XX概念) → 估值无上限
指纹4: 国内大厂事件(文心一言/通义千问等)为"发令枪" → S5确认信号
指纹5: 成交量先于价格30-45天异动 → 最佳入场窗口
```

```
S6已内置此逻辑
```

### 邓晓峰跨行业对比法(整合到S4)

```
每个投资决策同时看2-3个可比行业

S4 Anti-Sector的理论基础
```

## §13 能力圈 vs 茧房的边界

> 区分"真正的能力圈"和"伪装成能力圈的茧房"

| 维度 | 真正的能力圈 | 伪装成能力圈的茧房 |
|------|------------|-----------------|
| 校准度 | 70%置信度预测约70%正确 | 90%置信度实际准确率远低 |
| 边界清晰度 | 能明确说出"边界在哪，为什么" | 用"不熟悉"替代"不研究" |
| 扩展方式 | 基于知识积累向外延伸 | 视扩展为威胁 |
| 近期表现敏感性 | 边界认知稳定 | 牛市后自动"扩大"能力圈 |
| 否定能力 | 有具体理由排除 | 靠感觉排除，无法复盘 |

**操作规则**: 能力圈是加速研究的工具(熟悉领域可更快达到相同置信度)，不是不研究某些领域的借口。

**冯柳框架**: 
- 熟悉行业 → 选有分歧的公司(出奇制胜)
- 不熟悉行业 → 只选共识认可的龙头(守正不出奇，降低认知错误)
