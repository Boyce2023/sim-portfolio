---

## 一、方法论说明与重大警告

**本文档有严重的事后偏差，使用前必须理解以下局限：**

1. **事后选股偏差（最严重）**: 两份回测的股票池均在已知结果后选定，实际可执行胜率估计为 60-70%，而非文中的 90%+。
2. **主策略来源**: 主策略来自用户真实交易经验和用户与 Claude 的互动纠偏，见 `strategy.md`。本文仅为辅助参考。
3. **权重层级**: 互动纠偏 > 实盘记录 > 回测模式。回测模式只能验证或细化现有规则，不能创建新规则。
4. **用途限制**: 本文数据用于评估现有规则的一致性，以及提供板块和季节性的参考方向。禁止将回测胜率作为未来预测依据。

## 四、季节性规律参考（基于 2024 数据）

| 时段 | 表观胜率 | 说明 |
|------|----------|------|
| 1-4月 | ~100% | 年初效应 + 政策窗口期 |
| 5-7月 | 较高（>85%） | 正常运行区间 |
| 8月 | 55.6%（年内最弱中期） | 暑期流动性收缩 |
| 9-10月 | 强（刺激政策窗口） | 但追涨后事件消退快（见下方 Sep-24 教训） |
| 11月 | 85%，3次亏损 | 开始分化 |
| 12月 | 53.8-75%（全年最弱） | 年底结账，资金离场 |

**核心结论**: 12 月系统性最弱，两年均验证。表现最好的月份集中在 Q1。

## 五、回测验证的现有规则（仅作一致性确认）

以下规则已在 `strategy.md` / 互动纠偏中存在，回测数据提供一致性支持：

| 规则 | 回测支持强度 | 说明 |
|------|-------------|------|
| 事件当天不追涨 | 极强（两年均验证） | 德赛西威/赤峰同时出现最强最弱列表是直接证据 |
| 成交量突破 >1.5x 为核心入场信号 | 强 | 两年回测中最稳定的入场特征 |
| 板块连涨 3 日后降仓位 | 强（2024 Agent 24 验证） | 9月刺激后板块快速见顶 |
| 事件前量能积累 > 事件后追涨 | 强（Sep-24 刺激为教训来源） | 政策公布后追涨的胜率显著低于预期 |
| 12 月系统性降仓 | 强（两年 12 月均最弱） | 胜率 65-78% vs 其他月份 85-100% |
| 板块轮动优先序: 资源>PCB>半导体 | 中（样本支持但有时段依赖） | 仅供参考，不固化 |
| L5 产能过剩过滤 | 中（间接验证） | 药明/天齐/赣锋被过滤后实际走弱，与回测排除一致 |
| 恐慌性洗盘（3x+ 放量） = 反向买入信号 | 中（需基本面支撑为前提） | 回测中数次底部入场来自大幅放量下跌后 |

**未被回测数据充分验证的规则**: 以互动纠偏为准，回测沉默不代表规则无效。

## 六、局限性总结

1. **事后偏差无法消除**: 两份回测的股票池选择均知道结果，这是根本性缺陷，无法通过统计修正完全消除。
2. **时段代表性有限**: 2021 和 2024 的市场背景差异极大（2021 牛市尾声，2024 政策驱动反弹），两年数据覆盖的市场状态不全面。
3. **执行成本未计**: 回测不含冲击成本、手续费、实际滑点，高换手策略的实际收益会显著低于回测。
4. **正确率 != 盈亏比**: 高胜率回测可能对应低盈亏比策略，实际期望值取决于两者乘积，不能只看胜率。
5. **本文数据权重**: 低于用户实盘记录和互动纠偏。策略更新以互动纠偏为驱动，非本文。
