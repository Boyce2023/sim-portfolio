# Claude模拟盘交易策略 v1.0

## 账户规模
- A股: ¥1,000,000
- 美股: $150,000
- 周期: 2026-05-19 → 2026-06-18 (1个月)

## 策略哲学

供给侧优先 + 催化剂驱动 + 严格风控。不做纯概念炒作，不追涨停，不猜政策。

核心原则来自用户10步流程和240天A股复盘教训：
1. 每笔交易必须有明确的thesis和催化剂日期
2. Bear case downside > 20% 不建仓
3. 事件落地当天不动手，等1周资金验证
4. 退出看"2周无催化"，不主观设时间
5. 先淘汰再推荐——研究宽度是为了缩窄

## 仓位管理

### 单只上限
| 信心等级 | 仓位占比 | 触发条件 |
|---------|---------|---------|
| 核心持仓 (A级) | 10-15% | 自主研究完成+催化剂明确+bear case <15% |
| 重点持仓 (B级) | 5-10% | thesis清晰+有催化剂 |
| 观察仓 (C级) | 2-5% | 初步thesis+验证中 |
| 交易性仓位 | 2-5% | 短期事件驱动 |

### 组合约束
- 单只上限: 15% (A股) / 15% (美股)
- 单板块上限: 30%
- 现金下限: 20% (保持灵活性)
- 同日新建仓不超过3只
- 计划外加仓需先记录thesis再执行（杜绝"先买后研究"）

### 止损规则
- 每只必须有硬止损（入场即设）
- 组合级止损: 单日亏损>3% → 暂停新建仓1天
- 单只止损后不在同一交易日回买

## 出场规则

1. **D类下跌(thesis被证伪)**: 立即止损
2. **止损触发**: 无条件执行
3. **目标达成**: 分批出(50%目标/50% trailing)
4. **2周无催化**: 评估是否退出
5. **组合再平衡**: 单只超15%→减仓至12%

## 每日流程

### 08:00 BJT (A股开盘前)
1. 检查今天是否交易日 (市场日历)
2. US市场昨日收盘review + 对A股的映射
3. A股持仓逐只检查止损/目标/催化剂
4. 决定当日A股操作（或不操作）

### 15:30 BJT (A股收盘后)
1. 更新A股持仓价格
2. 记录当日操作和原因
3. 准备US市场pre-flight

### 21:30 BJT (US开盘)
1. 检查US持仓状态
2. 执行预定操作
3. 盘中不追涨（除非catalyst当日）

### 05:00 BJT (US收盘后)
1. 更新US持仓价格
2. 记录当日操作
3. 更新portfolio_state.json
