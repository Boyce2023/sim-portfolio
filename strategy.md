## §0 Critical Rules（永不跳过，5条）

> 这5条是唯一"违反立即造成不可逆组合损害"的规则。每条有可量化硬边界。

**C1 — 唯一真相源**: 所有价格/仓位/P&L从`portfolio_state.json`的A股section读取，禁止从记忆或旧文件推算。

**C2 — 仓位硬上限**: S≤40%（同时最多1只，-7%硬止损）/ A+≤35% / A≤25% / A-≤20% / B+≤15% / B≤15% / C/T≤8%；单只硬cap 35%（S级例外可至40%）；单板块≤40%；现金≥15%（加仓前须≥20%）。

**C3 — 下跌先分类再操作**: ABCD分类完成前，禁止任何卖出。60秒内完成：A=大盘跌≥1.5%→Hold / B=无新闻跌<3%→Hold / C=有新闻但thesis在→评估减0-30% / C+=传闻→24h核实 / D=硬数据证伪→48h清仓。

**C4 — 无thesis不建仓**: 进场检查表（标的/信心等级/thesis一句话/催化剂+日期/止损价/R/R比≥1.5）必须写完才能执行交易。

**C5 — Bear case硬门槛**: A股>40%不建仓（硬规则，无例外）。

### §3.3 仓位管理

#### §3.3.1 信心等级与仓位（v3.0分级制）

> **核心变更(v6.2)**: 单只硬cap从15%→35%（S级可至40%）。引入+/-修饰符增强区分度。Conviction决定仓位，不人为设低cap限制。参考巴菲特集中持仓理念。

| 等级 | 建仓目标仓位 | 上限（硬约束） | 建仓条件 | 止损宽度 |
|------|------------|------------|---------|---------|
| **S级（全仓短线）** | 30% | **40%** | S1-S5全部满足（见§3.3.6），供应链不可替代+催化剂确认+技术底部 | **-7%（硬止损，无商量余地）** |
| **A+级（准S级核心）** | 25% | **35%** | 几乎满足S但差一个条件，完整研究+强催化剂+bear case极浅 | 15-20% |
| **A级（核心持仓）** | 20% | **25%** | 完整研究完成 + 催化剂明确（≤30天）+ bear case可控 | 15-20% |
| **A-级（核心观察）** | 15% | **20%** | 好thesis但有一个明确担忧，催化剂待验证 | 12-15% |
| **B+级（重点持仓）** | 12% | **15%** | 接近A但缺催化剂或安全边际不足 | 10-15% |
| **B级（标准持仓）** | 10% | **15%** | thesis清晰 + 有明确催化剂 + 研究充分 | 10-15% |
| **C级（观察仓/Scout）** | 5% | **8%** | 初步thesis + 验证中，快速止损是保留子弹的机制 | 7-10% |
| **T级（事件交易仓）** | 4% | **8%** | 短期事件驱动，有明确退出条件，催化剂触发后快速离场 | -5%（更紧） |

**数量约束**: A+/A/A- 合计≤4个，S级全portfolio≤2个。

**A级升降级规则**:
- 升级(C→B→A): 催化剂落地且符合预期 → 可申请升1级，加仓至新等级目标仓位；需有明确理由写入daily-review
- 降级(A→B→C): 催化剂推迟>14天 → A降B；叙事变化(C类下跌) → B降C；bear case扩大超20% → 强制降至C或清仓
- 清仓: 任何等级遭遇D类下跌（thesis被证伪）→ 当日清仓，无商量余地

#### §3.3.2 组合级硬约束

| 约束 | 上限/下限 | 违反时行动 |
|------|---------|---------|
| 单只仓位 | **≤35%**（S级可至40%，按SABCT等级分级执行） | 超额减至对应等级目标值（S级30% / A+级25% / A级20% / B级12% / C级6%） |
| S级总数 | 最多同时持有1只S级仓位 | 新S级建仓前必须退出现有S级 |
| 单板块仓位 | ≤40% | 新增该板块需先平仓 |
| 单一主题仓位 | ≤30% | 同一叙事标的超过3只时，对比选最优2只，其余清仓 |
| 现金比例 | **≥15%（加仓前须≥20%）** | 集中度提高后单只质量更高，现金可适度降低，但加仓门槛是20% |
| 单日新建仓 | ≤3只 | 第4只等次日 |
| 最大持仓数量 | ≤8只 | A股分析成本更高，集中优于分散 |
| C级标的总仓位 | ≤25% | 新C级标的先减旧的 |

#### §3.3.4 加仓规则

**可以加仓（全部满足）**:
1. 当前持仓 < 该等级上限（A+<35%，A<25%，A-<20%，B<15%，C<8%）
2. 现金充足（加仓后现金 ≥ 20%）
3. thesis确认（未被证伪）
4. 下跌类型 ≠ D（D类不加仓）
5. 催化剂仍在30天内（或有新催化剂）

**禁止加仓（任一满足即禁止）**:
- 价格低于成本且drawdown_class = D
- 已超过等级上限
- 当日新建仓已达3只限额（加仓不计入此限额）
- 现金加仓后低于20%
- 同板块仓位已超板块上限（40%）

### §3.4 选股框架

#### §3.4.1 A股选股框架（v5.0，原v3.0重写+v5.0五步法升级）

> **核心变更**: 旧版第一步"供给侧优先"改为"市场在信什么"。原因：A股行情由叙事和情绪驱动，用价值框架选A股是系统性错误（用户直接纠正）。

**新五层信号体系（替代旧版6条排序）**:

| 层级 | 信号类型 | 权重 | 说明 |
|------|---------|------|------|
| **L1: 市场在信什么** | 主题共鸣度 | 40% | 资金正在追什么叙事？主题处于上升期？这是第一问。**注**: A股共识信号弱（散户主导，卖方滞后），看资金流向非研报数量(E13) |
| **L2: 催化剂前置性** | 催化剂是否还在前面 | 30% | 催化剂已兑现=不买；催化剂在前面=涨停也可买 |
| **L3: 龙头地位** | 行业内绝对领先 | 20% | A股资金追最先进/最领先，不追估值洼地 |
| **L4: 动量信号** | 游资/机构资金行为 | 7% | 换手率/主力净流入/涨停板连板数 |
| **L5: 基本面下限** | 不是空壳/不是诈骗 | 3% | 仅作为排除条件，不作为正向打分 |

**A股评分制（97分制，L5为独立门槛）**:
- L5先行: L5不通过（空壳/诈骗/退市风险）= 直接排除，不进入评分
- L1×40 + L2×30 + L3×20 + L4×7 = 满分97
- ≥60分: 可建仓（C级scout，8%上限）
- ≥75分: 正常建仓（B级，15%上限）
- ≥90分: 快速建仓（A级，25%上限）。**S级需额外通过S1-S5检查表（§3.3.6），L1-L5评分无法直达S级**
- <60分: 不建仓，写排除理由

**Grade Adjustment Rules (v6.2, 基于8天实盘数据验证)**:

> 在L1-L5基础评分确定base grade后，以下三个modifier可升降1档(如B→B+或B→B-)。

| Modifier | 条件 | 效果 | 实证 |
|----------|------|------|------|
| **M1: 毛利率门槛** | 毛利率<20% | **降1档** | 工业富联(6.6%毛利)全链涨它反跌，+0.3%。高毛利=护城河proxy |
| **M2: 回调入场奖励** | 板块/个股回调≥5%当日入场，thesis未变 | **升1档** | 鹏鼎回调-6.32%入场→+31%；安集回踩305入场→+7% |
| **M3: 瓶颈节点加分** | F2/F13确认供应链物理瓶颈(同级供应商≤2家) | **升1档** | 晶方(CIS封装唯一)+37%，鹏鼎(iPhone PCB独家)+31%，新易盛(1.6T唯一批量)+6.4% |

- Modifier叠加上限: 最多升2档（如C→B+），不能跳过A直接到S
- M1为强制降级（毛利率是硬数据），M2/M3为可选升级（需人工确认）

**A股选股步骤（v5.0五步法）** [来源: 互动纠偏★★★ 2026-05-22确认框架]:
1. **Step 1: 市场呼吸** — 今天资金在流向哪个主题？扫描B1-B5指标，确认市场温度和当前主题。低温(<8000亿)不追新仓 [互动★★★#4 | 回测验证: momentum信号选股胜率>90%]
2. **Step 2: 小票信号** — 同一主题下小盘股(<50亿)是否已出现2+只连板？龙虎榜游资是否扎堆？小票先飞=龙头布局窗口打开 [互动★★#7 | 回测验证: 两段式结构在2021/2024均成立]
3. **Step 3: 瓶颈节点** — 该标的是否为行业龙头？供应链中是否占据不可替代的瓶颈节点？A股资金追最先进，不追估值洼地 [互动★★★#2]
4. **Step 4: 催化剂时间线** — 该主题是否有具体日期的催化剂尚未兑现？催化剂在前面=涨停也可买；已兑现=见光死风险 [互动★★★#3 | 回测验证: 事件前布局>事件后追涨]
5. **Step 5: 估值透支检查** — 当前PE已经反映了几年预期？主题成熟度在哪个阶段（极早期/早期/部分定价/充分定价）？充分定价=不追 [互动★★]

**A股Bear Case规则（v3.0分级，替代旧版二元硬规则）**:

> Why: A股bear case天然偏高，高弹性成长股bear case永远>20%，旧版规则会系统性过滤所有A股alpha来源。

| Bear Case | A级（高conviction） | B级（研究充分） | C级（试探性） | T级（事件交易） |
|-----------|------------------|--------------|------------|--------------|
| ≤15% | 建仓，可到上限25% | 建仓，正常15% | 建仓，8%内 | 建仓，8%内 |
| 15-25% | 建仓，需明确止损点+写下bear case | 建仓，减半size（≤8%），严格止损 | 建仓，最高C级（≤5%） | 可建仓（≤4%） |
| 25-40% | 仅A级conviction可建，初始仓位减半（≤12%），必须有事件驱动催化剂 | 不建仓 | 不建仓 | 可建试单（≤2%），有明确止损 |
| >40% | **不建仓**（硬规则，无例外） | 不建仓 | 不建仓 | 不建仓 |

**补充规则（防止机械套用）**:
- 有强催化剂（确定日期）的标的，bear case可放宽5%
- 市场处于强势主题期（板块ETF连续资金流入），bear case阈值+5%
- 已在持仓的标的：bear case每2周重评一次（或thesis重大变化时立即重评）。bear case恶化超过原建仓时档位→触发降级机制（A→B→C→清仓）

#### §3.4.3 A股专用方法论

**市场呼吸5个量化指标（每日W1执行前扫描，需web-access skill获取）**:

**B1: 情绪温度计（关键决策依据）**:
```
沪市成交额（过去5日均值）:
  · <8000亿 = 市场低温，不追高，守底仓，不新建仓
  · 8000-12000亿 = 正常，按策略操作
  · >12000亿 = 过热，减仓等回调
TMT板块成交占比:
  · >45% = 拥挤度过高，即便看多也不加仓
两融余额:
  · 破2.1万亿 = 牛市加速信号
北向资金:
  · 连续5日净流入 = 外资确认主题
```

**B2: 板块轮动热度**:
- 今日换手率TOP20个股所在板块 → 识别当日领涨板块
- 同一板块连续3日领涨 = 过热信号，新进资金收益有限

**B3: 主力资金净流入（板块级）**:
- 板块连续3日主力净流入>0 + 换手率放大 → 机构跟进确认
- 板块主力净流出但个股上涨 → 拉高出货信号，不追

**B4: 游资主导度**（来源：龙虎榜）:
- 游资营业部≥3家 → 游资主导（切技术框架，不用基本面）
- 机构席位≥2家 → 机构主导（可用基本面框架）
- 游资席位占买入总额>60% = 纯妖股，不参与（E8+E16规则）

**B5: 催化剂密度日历**:
- 未来30天有3+个明确催化剂节点 = 市场有说法，可以部署
- 催化剂间隔>3周 = 动能可能衰减，轻仓或等

**板块轮动两段式结构（用户验证的核心规律）**:

```
第一段：游资/小票先飞
  信号: 小盘股（市值<50亿）集中涨停 + 龙虎榜游资营业部扎堆
  含义: 主题刚被发现，游资试水，机构尚未进场
  操作: 这是布局窗口——不追妖股，找该主题的真龙头买入

第二段：机构/龙头后动
  信号: 该主题ETF/大盘股放量 + 主力净流入转正
  含义: 机构确认主题，进入快速拉升阶段
  操作: 龙头仍可买，但不是第一波了
```

**关键陷阱**: 龙头不动≠主题死（E11规则）。龙头滞后于小票是正常节奏，不要因龙头还没动就否认主题。

**游资先飞的4个具体信号（满足3条=布局龙头黄金窗口）**:
1. 同一主题下有2只以上小票连板（非单个妖股）
2. 龙虎榜同一天多只小盘股出现相同游资营业部（抱团主题）
3. 东财股吧/雪球主题词频比前7日均值>3倍
4. 该主题真龙头涨幅<小票（机构还没跟进）

**三阶段操作规则**:

| 阶段 | 特征 | 操作 |

**A股进场时机规则**:

| 场景 | 规则 | 原理 |

**"催化剂在前面"判断标准（纠正等回调伪逻辑）**:

**错误逻辑**: "涨停了所以等回调" — 这是用价格行为否定thesis

**正确框架**:
```
问题: 催化剂在前面还是后面？

如果在前面:
  → 涨停也买（更多买家还会来）
  → "等回调到¥XX"这句话不要说

如果已兑现/即将兑现:
  → 才该等或出场（见光死风险）
  → 不是因为价格涨了多少

案例: 德赛西威5/21涨停 → FSD Q3获批还在前面 → 不该等
反例: 某标的政策已出+股价大涨30% → 催化剂基本兑现 → 此时才叫"等"
```

#### §3.5.2 ABCD下跌分类——下跌先分类再行动

> **CRITICAL**: ABCD分类之前，禁止任何卖出动作。先分类，后行动。60秒内完成分类。
> **摘要见§0 C3**。完整版在此处。

| 类型 | 名称 | 判别标准 | 行动 | 止损规则 |
|------|------|---------|------|---------|
| **A** | 大盘系统性下跌 | 沪深300同步跌≥1.5%且无个股新闻。注意: 大盘跌但**板块指数稳**→个股可能是C类 | **Hold**（不操作，不恐慌，不补仓） | 只有同时触发组合级止损才处理 |
| **B** | 板块轮动噪音 | 指数稳或涨；无新闻；下跌<3% | **Hold**，观察1-2天 | 持续3天且无基本面变化仍持有 |
| **C** | 叙事切换（非破坏性） | 有新闻但不涉及核心thesis；市场情绪切换 | **评估**：减仓0-30%（按信心等级），写下重新评估结论再操作 | 24h内给出C→D升级结论或维持C |
| **C+** | 待确认叙事变化 | 单一信源/传闻/预期打压，无硬数据 | **观察**：24h内主动核实；核实前不操作 | 核实后升级C或D；超过48h视为D |
| **D** | Thesis被证伪 | 财报核心指标miss≥2个季度/重大客户流失公告/政策文件否决/管理层不明原因离职 | **立即清仓**，48h内执行，不等反弹 | 无例外。D类不存在"再等等看" |

**60秒分类流程**:
```
1. 看沪深300：跌≥1.5%？ → 是 = A类 → Hold，不操作
2. 有没有今天的新闻？ → 没有 = B类 → Hold 1-2天观察
3. 新闻改变了thesis核心逻辑吗？
   → 没改变 = C类 → 评估，写结论，可减0-30%
   → 改变了但只有传闻/单一信源 = C+类 → 24h核实，暂不操作
   → 有硬数据支撑（财报/公告/政策文件）= D类 → 48h内清仓
记住：只有D类是必须卖的。
```

**C与D的高发混淆场景**:

| 场景 | 分类 | 理由 |
|------|------|------|
| 财报beat但市场叙事切换 | C类 | 公司基本面未变，市场重定价叙事 |
| 竞争对手发布更好产品 | C+→D | 需核实对份额/GPM的具体影响 |
| 大客户单季度减少采购 | C+类 | 单季度可能是库存调整，≥2季度升D |
| 核心技术/专利被否 | D类 | 硬壁垒消失，thesis破坏 |
| 传闻监管打压（无正式文件）| C+类 | 等文件落地；A股尤其高发 |
| 正式监管文件落地（负面）| D类 | 政策确定，无讨论空间 |
| 管理层CEO替换（内部提拔）| B-C类 | 评估新任背景 |
| 管理层创始人离职（无交代）| D类 | 创始人vision是核心资产时直接D |
| 财报in-line但股价-9%（HSAI案例）| **C类，不是D类** | in-line非miss，止损前先分类 |

#### L10-L15: 新增行为铁律（v3.0）

> **摘要见§1.2 D7-D9**。完整定义（含Why和验证案例）在此处。

---

**L10 — 关注点漂移：看别人涨了就想换**

- **定义**: 持有标的X未动，看到Y/Z涨了 → 萌生"换到Y/Z"的想法
- **诊断信号**: 卖出原因是"别的股票涨了它没涨"（而不是thesis变化）
- **铁律**: 卖出前必须回答①thesis变了吗？②我是因为看了别的股票才想卖吗？如果②是"是"→关闭行情软件，等1小时后再决定
- **验证案例**: 肇民05-21早盘，机器人板块其他股涨，肇民没动→卖出→下午肇民+2.86%
- **Why**: 相对表现焦虑，和追涨是同一情绪的两面

---

**L11 — 催化剂前离场：在最关键的时候撤退**

- **定义**: 有明确催化剂日期，在催化剂前1-3天因为"已经涨了不少"而减仓
- **诊断信号**: 建仓时写了催化剂日期；催化剂未到就开始考虑减仓
- **铁律**: 催化剂前默认持有。减仓需要给出"不是催化剂原因"的充分理由（如ABCD分类为D类，或已大幅超过目标价）
- **Why**: 从赔率角度：催化剂前持有期望值 > 未持有。短期获利欲望压过赔率思维

---

**L12 — 赢小输大反置：小赚就跑，大亏死扛**

- **定义**: 持仓小幅盈利时快速获利了结；持仓大幅亏损时扛着不止损
- **诊断信号**: 盈利方向仓位越来越小；亏损方向仓位维持或扩大
- **铁律**: 卖出时问自己"如果这是一笔新投资，我会买吗？"是→持有；不是→这才是真正的卖出信号
- **验证案例**: 蓝思赚了¥6870快速清；HSAI亏$757扛着（部分合理因为有thesis）
- **Why**: 前景理论（亏损厌恶）

---

**L13 — 追涨三联动：FOMO驱动的系统性错误（三行检查表，强制）**

- **定义**: 任何由"今天才知道它涨了"驱动的买入决策
- **铁律（三行检查表，强制，缺一不可）**:
  - ①买的理由是什么（不是"涨了"）
  - ②跌20%我割不割（说不出→不买）
  - ③是不是今天才知道它的（是→不买）
- **第③条最关键**: 今天才知道 = 信息链末端 = 别人早就知道 = 你在买入别人的出货
- **例外**: 底部放量涨停+真实催化剂（如德赛西威从¥96反弹，是底部启动不是高位追涨）
- **Why**: FOMO（错失恐惧）驱动的系统性错误

---

**L14 — 仓位-Conviction倒挂：最大仓位是最不确定的**

- **定义**: 仓位大小与研究深度/信心等级呈反比（A级标的仓位最小，C级标的仓位最大）
- **诊断信号**: 说不出thesis的标的持仓最重；研究最深的标的持仓最轻
- **铁律**: 每次建仓前先定信心等级（A/B/C）→对应仓位上限（25%/15%/8%）→按上限建仓。说不出thesis = C级，C级上限8%
- **验证案例**: 德赛电池1200股无thesis vs 思源100股最高conviction（完全倒置）
- **Why**: 买入时没有强制sizing决策框架，凭感觉

**L15 — Thesis错配不自知：涨了不代表你的理由对了**

- **定义**: 买入理由（thesis）与标的真实驱动力不一致，短期涨了而强化错误thesis
- **诊断信号**: 说不清楚"它现在为什么涨，和我买的理由是同一个吗？"
- **铁律（强制一步，买前执行）**: 搜索"[标的名] 最近为什么涨"→对比搜索结果和自己的买入理由→不同=不买
- **验证案例**: 圣泉605589——用户因"钠电硬碳"买入，实际涨幅驱动是PPO树脂/AI PCB（H1+51%全靠PPO），钠电收入<3%。Thesis完全错配，当日全清
- **Why**: 自我确认偏误（confirmation bias）

### §3.9 If-Then预承诺格式（v2.0升级版）

**核心规则: 没有预承诺 = 没有操作。禁止临场发挥。**

#### 标准格式（v2.0含行为防护）

```markdown
## [标的] If-Then 预承诺表
建仓日期: XXXX-XX-XX
建仓价格: ¥X.XX
催化剂日期: XXXX-XX-XX
Conviction级别: A/B/C
止损线: ¥X.XX（成本价 -X%）
目标价1: ¥X.XX
目标价2: ¥X.XX（trailing stop触发后）

### 催化剂前（现在到XX/XX）
- If 股价下跌5-10%（A/B类下跌）: 持有，不止损
- If 股价下跌>12%（需判断C/D类）: 执行ABCD分类，D类立即止损
- If 出现更好的买入机会冲动: 先问"今天才知道它的吗？是→不动"（L13）

### 催化剂后（XX/XX当天及后续）
- If [超预期结果]: [具体操作，如"加仓至X%"]
- If [In-line结果]: [具体操作，如"减仓25%，保留底仓等下一催化"]
- If [Miss/负面结果]: [具体操作，如"执行ABCD分类，D类全清"]

### 时间止损（无催化剂期）
- If 14个交易日无新催化 AND 处于B/C类下跌: 重新评估是否继续持有
- If 14个交易日无新催化 AND 仓位亏损>-8%: 清仓

### 行为防护条款
- 卖出前必须确认: "我卖出的理由不是因为看了别的股票"（L10）
- 加仓前必须确认: "不是因为今天涨停才想加"（L13）
- 催化剂前不减仓默认持有（L11）
```

### §3.13 组合级风控触发阈值

> **摘要见§2.3**。完整阈值表在此处。

#### 日内触发

| 触发条件 | 立即行动 |
|---------|---------|
| 单日组合亏损 >2.5% | 暂停新建仓（当日剩余时间）|
| 单日组合亏损 >4% | 暂停新建仓 + 检查D类标的是否需要止损 |
| 单只持仓单日亏损 >7% | 强制ABCD分类，C+及以上减仓50% |
| 单只持仓单日亏损 >10% | 强制分类，D类立即清仓，C类减仓至C级上限以下 |

#### 周度触发

| 触发条件 | 行动 |
|---------|------|
| 周度组合亏损 >5% | 下周前3个交易日禁止**新建仓**（加仓现有持仓不受此限，但需满足§3.3.4加仓规则），专注持仓review |
| 周度组合亏损 >8% | 全仓review，所有B/C级标的重新评级，降级的立即减仓 |
| 连续2周亏损（合计 >6%） | 强制策略复盘（找根因，不是找解释）|

#### 月度触发

| 触发条件 | 行动 |
|---------|------|
| 月度亏损 >10% | 仓位上限临时降至70%（现金下限30%）持续1个月 |
| 月度亏损 >15% | 仓位上限降至50%，持续2个月；必须做完整复盘才能解除 |

#### 赢家模式（继续复制）

| 模式 | 代表案例 | 关键要素 |
|------|---------|---------|
| **W1: 事件驱动+明确时间窗口** | 蓝思/鹏鼎/恒瑞 | 催化剂越具体、时间越近，入场确定性越高 |

| **W3: 纪律性止盈执行干净** | 蓝思+10%止盈 | 止盈预设在买入时定好，不是涨了再想 |
| **W4: 逆势加仓核心持仓** | 思源05-21大跌日加仓 | 大跌日加核心仓，小涨日不追 |

| 模式 | 代表案例 | 改进 |
|------|---------|------|
| **L1: 无催化剂的赛道配置** | 安集纯赛道配置 | 短期模拟盘中，纯赛道配置不适用 |
