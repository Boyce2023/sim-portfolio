## §2 进攻系统

### §2.1 建仓五步法

```
Step 1: Thesis — 一句话为什么买（"涨了"不是thesis）
Step 2: Catalyst — 催化剂具体日期。说不出=不布局
Step 3: Stop — ATR动态止损(§3.1)算止损价
Step 4: Size — §2.3表查等级硬顶，§2.6风险预算参考
Step 5: Scale — 分批建仓(§2.7)。A股T+1，不一次性全仓
→ 五步过了才买。不等回调，用sizing+scaling控风险
```

### §2.2 统一SABCD评级系统（v8.1）

**评级 = Conviction强度。** 止损管实际风险，评级管conviction。Bear case不决定评级上限。

| 等级 | Conviction标准 | 仓位硬顶 | ATR K值 | 最大硬止损 |
|------|---------------|---------|---------|-----------|
| **S** | 多维信号命中 + ≤4分析师 + PEG极端 + 三层thesis | 50% | 3.5 | -20% |
| **A+** | 三层thesis验证 + 催化剂<30天 + 竞争壁垒确认 | 35% | 3.0 | -18% |
| **A** | 强thesis + 催化剂明确 + risk/reward好 | 25% | 3.0 | -15% |
| **A-** | 好thesis + 一个不确定性（时机/执行/宏观） | 20% | 2.5 | -15% |
| **B+** | 不错thesis + 催化剂稍远或稍模糊 | 15% | 2.0 | -12% |
| **B** | 边际thesis，观望为主 | 12% | 2.0 | -10% |
| **B-** | 弱thesis，最小仓位 | 10% | 1.5 | -10% |
| **C** | **观察池（0%）** — 有潜力但催化剂不明确 | — | — | — |
| **D** | **排除/清仓** — bear case>40%或thesis证伪或板块结束 | — | — | — |

**Bear Case的唯一角色——二元过滤器**:
- ≤40%: **通过**，进入conviction评级。bear case 25%的票照样可以给A+，只要conviction够强
- >40%: **D级排除**，不浪费时间
- Bear case不再决定S/A+/A之间的等级差异。不确定性是alpha来源，不是降级理由

**深度研究的正确用法**: 确认thesis是否成立，不是搜集降级弹药。问"这个机会真不真"，不是"能找到多少问题"。中际旭创和寒武纪在起飞前都有一堆bear case——但thesis是对的，催化剂兑现了。

**共识信号（F15 A股修正v2）**: 卖方全看多 ≠ 排除。看价格折让：
- 股价≥共识目标价 → 真拥挤，降优先级
- 股价低于目标15%+ → 共识方向对，alpha在择时
- 结构性供给变化刚开始 → 共识可能低估，优先研究

**S级特别条件**: ①多维度信号同时命中 ②分析师覆盖≤4人 ③业绩beat频率>80% ④PEG<0.6或行业绝对最低 ⑤已持有≥5天。历史标杆：中际旭创(2024)、寒武纪(2023)。

**数量约束**: S≤1。A+/A/A-合计≤3。C级不占持仓名额。

**升降级规则**:
- 升级: 催化剂兑现+符合预期 → 升1级，加仓至新等级。写理由
- 降级: 催化剂推迟>14天 → 降1级；thesis被证伪 → D级
- 板块结束信号(§4.3) → 同板块全部降为C或D
- D级 → 48h内清仓，无例外

## §3 防守系统

### §3.1 三层ATR止损

**Layer 1 — 初始止损**: 止损 = Entry - K × ATR(14)。cap at最大硬止损。

| 等级 | K值 | 最大硬止损 |
|------|-----|-----------|
| S/A+/A | 3.0-3.5 | -15%~-20% |
| A- | 2.5 | -15% |
| B+/B | 2.0 | -10%~-12% |
| B- | 1.5 | -10% |

**Layer 2 — 棘轮锁利**:

| 阶段 | 止损调整 |
|------|---------|
| +1R | 移至保本 |
| +2R | high - 2×ATR |
| +3R | high - 1.5×ATR |
| +50%+ | high - 1×ATR |

止损只上移不下移。

**Layer 3 — 时间衰减**:

| 条件 | 动作 |
|------|------|
| 10天未达+1R | 收紧0.5×ATR |
| 每多5天 | 再紧0.5×ATR |
| 催化剂过期未兑现 | 3天内清仓（举证义务反转） |
| 20天无催化兑现 | 强制重评 |

**执行铁律**: 触及→当日执行。不分类不犹豫不等反弹。

## §4 持仓管理

### §4.1 加仓规则

全部满足: ①持仓<等级硬顶 ②现金后≥20% ③Thesis未证伪 ④催化剂仍在30天内 ⑤板块后≤35%

### §4.2 R倍数出场梯

1R = Entry - 止损价。出场按R倍数分级：

| 阶段 | 操作 | 止损 |
|------|------|------|
| +2R | 卖25% | 保本 |
| +3R | 再卖25% | high-1.5×ATR |
| +4R+ | Trail 50% | high-1×ATR |
| 催化剂兑现 | 3天决策窗口 | — |

目标价: T1≥+2R, T2≥+4R。达不到+2R=risk/reward不够→降级。
A股T+1: 大涨日(+1R以上)不卖。收盘前30分决策，次日执行。

### §4.3 A股出场规则

| 场景 | 行动 |
|------|------|
| **板块龙头单日跌>5%** | **板块结束→同板块启动出场** |
| 催化剂已兑现 | 3天内决定退出（见光死是A股最高频死法） |
| 14天无新催化 | 重评；亏>-8%→清仓 |
| 龙虎榜游资变"卖出" | 主题第一阶段结束 |
| 大涨日 | 不减仓（T+1卖了买不回来） |

**板块结束判断**: 龙头股（板块涨幅最大/最先启动的）单日大跌>5%当天=行情结束。后面是波动不是趋势。同板块次日开盘评估出场。

### §4.4 业绩节奏检查（学自美股F21）

持仓发布财报后24h必填:

| 类型 | 判别 | 操作 |
|------|------|------|
| 扩张型 | 营收+净利beat + 毛利改善 | 升级加仓 |
| 维持型 | Beat但增速平 | Hold不加 |
| 恶化型 | Beat但增速放缓/毛利降 | 降1级+收紧止损 |
| Miss | 营收或净利miss | 3天出场倒计时 |

A股中报(8月)+年报(3-4月)。间隔期用行业数据proxy。

### §4.5 轮动检测（学自美股§7，每周一10分钟）

   1000>300+2%=吸气 / 300>1000+2%=呼气

## §7 错误速查

| 错误 | 对策 |
|------|------|
| 低PE选A股 | A股低PE=陷阱。买催化剂+龙头 |
| 板块热5天才追 | 第1-2天入。第5天=尾声 |
| 先买后研究 | thesis未完不下单 |
| 概念无业绩 | >10%营收才建仓 |
| 高位追涨停 | 底部+催化剂=OK; 高位=不追 |
| 游资股基本面止损 | 游资主导→技术框架 |
| **等"5天无新高"** | **龙头跌>5%当天=板块结束，不等5天** |
| 深研越多降级越多 | 研究确认thesis，不是搜集降级弹药。10个bear case不比2个更应该降级 |
| 共识看多=排除 | 看价格折让，不看分析师人头数（F15修正） |
| bear case定等级 | bear case只做二元过滤(>40%排除)，不影响S/A/B之间的conviction评级 |

A股规则: T+1(大涨不卖) / 涨停=确认信号 / 海外映射30分钟内 / 板块龙头跌>5%=结束

## §10 Pain/Reward Architecture — A股版

> 六层双向行为反馈系统。从美股V6.2§11适配，参数针对A股重新校准。
> Pain先行Victory跟随——不亏钱比赚钱更重要(λ=2.25)。

### PART A — PAIN SYSTEM

**L1 Pain Memory（痛觉记忆）**

触发: ATR止损 / D级48h清仓。板块结束纪律性出场和催化剂兑现出场**不触发**。

3问post-mortem（5-10min，止损当天收盘完成）:
1. 哪里判断错了？（不是"市场不好"——是**我**错在哪）
2. 有没有提前出现的信号我忽略了？
3. 下次同类情况的if-then是什么？

Pattern Tags: `round_trip` / `fomo_rotation` / `sector_end_missed` / `catalyst_deferred` / `t1_panic_sell` / `sizing_emotion` / `thesis_drift` / `fomo_entry`

文件: `pain_memory.md`（rolling 5条）。建仓前扫描match→必须回答"这次有什么不同"。

**L2 Circuit Breaker（熔断器）**

| 状态 | 触发条件 | 效果 | 恢复 |
|------|---------|------|------|
| 🟢 GREEN | 默认 | 正常 | — |
| 🟡 YELLOW | 周组合跌>5% 或 连续止损≥2 | sizing×0.5, 每日新建≤1只 | 连续2周无止损 |
| 🔴 RED | 周组合跌>8% 或 连续止损≥3 | 禁止新建仓, 仅执行止损 | 1周无止损→🟡 |

板块结束信号纪律性出场**不计入**连续止损（正确执行不惩罚）。

**L3 Conviction Credit（信念信用）**

| 情形 | 处罚 | 解除 |
|------|------|------|
| A+止损 | 30天内不可新建A+ | 到期自动 |
| 同一Pattern Tag第2次 | 该类操作降1档(30天) | 到期自动 |
| Grade Accuracy<40%(≥5笔) | 全线降1档 | 准确率恢复>60% |

### PART B — VICTORY PROTOCOL

**L4 Victory Memory（胜利记忆）**

触发: 盈利出场 且 R≥+2R 或 催化剂兑现 或 Trade Score≥A级。

3问Victory Memo（5min）:
1. 哪里做对了？（具体操作，不是"运气好"）
2. 有没有更好的exit时机？（MFE检查）
3. 这个setup可以复制吗？复制条件是什么？

文件: `victory_memory.md`（rolling 5条）。建仓前match→sizing可至等级上限（不超硬顶）。

**L5 Conviction Amplifier（信念放大器）— 仅🟢GREEN生效**

| 状态 | 触发 | Sizing |
|------|------|--------|
| ⚪ NORMAL | 默认 | ×1.0 |
| 🔵 ELEVATED | A-grade≥60%(rolling 10) + R期望值>0.5 | ×1.15 |
| 🟣 PEAK | A-grade≥75% + 期望值>1.0 + 连续盈利≥4 | ×1.25 |

Phase 1保守倍数（×1.15/×1.25）。≥20笔后Phase 2升级（×1.25/×1.5）。

**L6 PlayBook + R-Multiple + MFE**

- PlayBook(`playbook_astock.json`): 从零积累，≥2实例=validated。A股模式独立于美股
- R-Multiple: 1R = Entry - ATR止损。Rolling 20笔追踪win_rate/期望值
- MFE: T+1下用**次日开盘价**替代日内最高。Capture<40%=处置效应
- Anti-Disposition: 持仓review隐藏成本价，只看thesis/催化剂/止损线

### PART C — 集成

**Scorecard（每session开头<2min）**: CB状态 + CA状态 + 最近Pain/Victory + Grade限制 + PlayBook applicable

**集成触点**: Session开头(scorecard) / 止损后(post-mortem 5-10min) / 盈利后(VM 5min) / 建仓前(双向match) / 每周一(hold-review+Anti-Disposition) / 每周五(CB/CA更新)

**FOMO专项**: Round Trip(3天内买卖盈亏<3%)→计入连续止损+tag`fomo_rotation`。同周第2次→🟡。L3同pattern 2次→降1档30天。建仓前L13三问+pain match: "这次setup哪里不同于上次FOMO？"
