- 通用: 足够好就行动。不因不完美而排除，选股力≠执行力，执行是短板。

## 3. 仓位管理

### 3.1 信心等级与仓位（v3.0分级制）

> **核心变更**: 旧版单只15%死规则废除。新版分级制：A级高conviction可到25%。理由：旧规则导致70%现金长期空转，高conviction机会被平等对待。

| 等级 | 建仓目标仓位 | 上限（硬约束） | 建仓条件 | 止损宽度（A股 / 美股） |
|------|------------|------------|---------|-------------------|
| **A级（核心持仓）** | 15% | **25%** | 完整研究完成 + 催化剂明确（≤30天）+ bear case可控 | 15-20% / 20-25% |
| **B级（重点持仓）** | 10% | **15%** | thesis清晰 + 有明确催化剂 + 研究充分 | 10-15% / 15-20% |
| **C级（观察仓/Scout）** | 5% | **8%** | 初步thesis + 验证中，快速止损是保留子弹的机制 | 7-10% / 10-12% |
| **T级（事件交易仓）** | 4% | **8%** | 短期事件驱动，有明确退出条件，催化剂触发后快速离场 | -5%（更紧） |

**A级升降级规则**:
- 升级(C→B→A): 催化剂落地且符合预期 → 可申请升1级，加仓至新等级目标仓位；需有明确理由写入daily-review
- 降级(A→B→C): 催化剂推迟>14天 → A降B；叙事变化(C类下跌) → B降C；bear case扩大超20% → 强制降至C或清仓
- 清仓: 任何等级遭遇D类下跌（thesis被证伪）→ 当日清仓，无商量余地

### 3.2 组合级硬约束

| 约束 | 上限/下限 | 违反时行动 |
|------|---------|---------|
| 单只仓位 | ≤15%（A级可至25%，按等级分级执行） | 超额减至对应等级目标值（A级20% / B级12% / C级6%） |
| 单板块仓位（A股） | ≤40% | 新增该板块需先平仓 |
| 单板块仓位（美股） | ≤35% | 新增该板块需先平仓 |
| 单一主题仓位 | ≤30% | 同一叙事标的超过3只时，对比选最优2只，其余清仓 |
| 跨市场同行业 | ≤45%（全球合并暴露） | 防止中美双重下注同一赛道 |
| 现金比例 | **≥20%（硬规则）** | 低于20%前停止一切新建仓；强催化剂期可降至15% |
| 单日新建仓 | ≤3只 | 第4只等次日 |
| 最大持仓数量 | ≤12只 | 超过12只必须淘汰一只 |
| 最大持仓数量（A股） | ≤8只 | A股分析成本更高，集中优于分散 |
| C级标的总仓位 | ≤25% | 新C级标的先减旧的 |

### 3.3 超配处理规则

| 情况 | 触发条件 | 处置 | 时限 |
|------|---------|------|------|
| A级超配 | 当前仓位 > 25% | 减至20%（保留弹性区间） | 次日开盘 |
| B级超配 | 当前仓位 > 15% | 减至12% | 当日 |
| C级超配 | 当前仓位 > 8% | 减至6% | 当日 |
| 板块超配 | A股单板块>40% / 美股>35% | 新建仓禁止进入该板块；现有持仓自然减持 | 实时 |
| 被动超配（因涨价） | A级涨至>25% | **不强制减仓**，等非A类下跌时顺势减至20% | 不强制 |

### 3.4 加仓规则

**可以加仓（全部满足）**:
1. 当前持仓 < 该等级上限（A<25%，B<15%，C<8%）
2. 现金充足（加仓后现金 ≥ 20%）
3. thesis确认（未被证伪）
4. 下跌类型 ≠ D（D类不加仓）
5. 催化剂仍在30天内（或有新催化剂）

**禁止加仓（任一满足即禁止）**:
- 价格低于成本且drawdown_class = D
- 已超过等级上限
- 当日新建仓已达3只限额（加仓不计入此限额）
- 现金加仓后低于20%
- 同板块仓位已超板块上限

### 3.5 现金部署速度规则（新增）

> Why: 旧版无时间约束，导致Day 1现金68%，严重拖累组合表现。"足够好就行动"，不等完美。

| 时间节点 | A股现金目标 | 美股现金目标 | 触发机制 |
|---------|-----------|-----------|---------|
| Day 1–3 | ≤50% | ≤45% | 有任何B级以上机会立即建仓 |
| Day 4–7 | ≤40% | ≤35% | 一周内完成主仓布局 |
| Day 8–14 | ≤30% | ≤30% | 进入稳定运营阶段 |
| Day 15–25 | 25–35% | 25–35% | 维持弹药库，等催化剂窗口 |
| Day 26–31（收官） | ≥30% | ≥30% | 逐步降敞口，不追新建仓 |

**现金空转预防**: 若现金连续3天超过时间阈值，强制从watchlist选1只B级以上标的建仓（5-10%），理由写入daily-review。

## 4. 选股框架

### 4.1 A股选股框架（v3.0全面重写）

> **核心变更**: 旧版第一步"供给侧优先"改为"市场在信什么"。原因：A股行情由叙事和情绪驱动，用美股价值框架选A股是系统性错误（用户直接纠正）。

#### 新五层信号体系（替代旧版6条排序）

| 层级 | 信号类型 | 权重 | 说明 |
|------|---------|------|------|
| **L1: 市场在信什么** | 主题共鸣度 | 40% | 资金正在追什么叙事？主题处于上升期？这是第一问 |
| **L2: 催化剂前置性** | 催化剂是否还在前面 | 30% | 催化剂已兑现=不买；催化剂在前面=涨停也可买 |
| **L3: 龙头地位** | 行业内绝对领先 | 20% | A股资金追最先进/最领先，不追估值洼地 |
| **L4: 动量信号** | 游资/机构资金行为 | 7% | 换手率/主力净流入/涨停板连板数 |
| **L5: 基本面下限** | 不是空壳/不是诈骗 | 3% | 仅作为排除条件，不作为正向打分 |

**A股评分制（100分制）**:
- L1×40 + L2×30 + L3×20 + L4×7 + L5×3
- ≥60分: 可建仓（C级scout，8%上限）
- ≥75分: 正常建仓（B级，15%上限）
- ≥90分: 快速建仓（A级，25%上限）
- <60分: 不建仓，写排除理由

**A股选股步骤（v3.0三步法）**:
1. **第一步: 市场在信什么** — 扫描板块ETF资金流入/游资席位/涨停板名单，识别当前主题
2. **第二步: 催化剂是否还在前面** — 该主题是否有具体日期的催化剂尚未兑现
3. **第三步: 供给侧约束验证** — 该标的是否为行业龙头，供给壁垒是否构成护城河（从"第一步"降为"第三步"）

#### A股Bear Case规则（v3.0分级，替代旧版二元硬规则）

> Why: A股bear case天然偏高，高弹性成长股bear case永远>20%，旧版规则会系统性过滤所有A股alpha来源。

| Bear Case | A级（高conviction） | B级（研究充分） | C级（试探性） |
|-----------|------------------|--------------|------------|
| ≤20% | 建仓，可到上限25% | 建仓，正常15% | 建仓，8%内 |
| 20-35% | 建仓，需要明确止损点 | 减半size，严格止损 | 不建仓 |
| 35-50% | 仅A级可建，初始仓位≤50%上限 | 不建仓 | 不建仓 |
| >50% | 不建仓（业绩造假/退市风险） | 不建仓 | 不建仓 |

**补充规则（防止机械套用）**:
- 有强催化剂（确定日期）的标的，bear case可放宽5%
- 市场处于强势主题期（板块ETF连续资金流入），bear case阈值+5%
- 已在持仓的标的，不需每次重新过bear case筛（只有thesis重大变化时重评）

### 4.2 A股错误模式完整清单（E1-E16）

| 编号 | 错误模式 | 具体表现 | 对策 |
|------|---------|---------|------|
| E1 | 节后追高 | 节前停牌，复牌直接高开8%+ | 等首日高开回落再看，节后第2个交易日才是真资金入场 |
| E2 | 政策当天追 | 政策发布当日直接冲涨停 | 政策后第5-10个交易日布局（240天实证） |
| E3 | 板块热点追尾 | 板块连续上涨5天以上才买 | 第3天开始观察，第1-2天才是入场窗口 |
| E4 | 概念无业绩 | 纯概念股无实际收入 | 验收：业绩贡献>10%总营收才可建仓 |
| E5 | 替代品陷阱 | 买不了A就买B（低相关） | 买不了A就等，不买低相关的B |
| E6 | 先买后研究 | 看到涨先建仓再想thesis | 无thesis不建仓 |
| E7 | 止损后回买 | 当日止损后当日重买 | 止损后至少隔1个交易日才可重评 |
| E8 | 追涨停 | 涨停板追入 | 涨停后等次日回调，不追板（流动性扭曲）|
| **E9** | **用美股价值框架选A股** | 买低PE/高分红/破净股 → 价值陷阱 | A股低PE="市场有理由不给估值"；只买催化剂驱动+最先进龙头 |
| **E10** | **"等回调"伪逻辑** | 涨了就等，等没等到，踏空 | 正确问题是"催化剂在前还是后"，不是"涨了多少" |
| **E11** | **龙头不动≠主题死** | 看到龙头没涨，认为主题已过 | 小票先飞是正常两段式第一段，龙头滞后是布局窗口 |
| **E12** | **过度淘汰（Phase 2教训）** | bear case >20%无条件排除，导致25+标的0个通过 | A股用仓位分级替代hard cutoff |
| **E13** | **共识≠price in（A股版）** | 看到卖方都推某股就放弃 | A股散户主导，卖方共识比美股更滞后；检查资金流向而非研报数量 |
| **E14** | **忽略市场温度强行操作** | 成交<8000亿的冷市还追热点 | 先看情绪温度计，冷市守底仓，不新建仓 |
| **E15** | **主题成熟度判断错误** | 买已充分炒作的主题（光模块/EML激光器PE 417x） | 主动检查主题成熟度：极早期/早期/部分定价/充分定价 |
| **E16** | **游资股用基本面止损** | 游资主导的妖股跌了3%就按bear case止损 | 游资主导时切技术框架，止损逻辑完全不同 |

### 4.3 美股选股规则（v3.0修订）

| 规则 | 旧版 | v3.0修订 | Why |
|------|------|---------|-----|
| 共识信号 | 15/15=priced in=排除 | 共识类型三分法（A/B/C型），不直接排除 | 共识是滞后指标，定价取决于预期差而非分析师数量 |
| Bear case | >20%=无条件不建仓 | 分级：≤15%可建B/A级；15-25%最高C级；25-35%仅T级；>35%排除做多 | NVDA历史bear case从未<20%，旧规则会排除所有成长股 |
| 入场方式 | 等完美回调 | 趋势初期小仓入场，Beat后加仓 | 牛股不给完美买点 |
| Momentum | 不看 | 必看（三层框架：相对强度/资金流/52周位置） | 机构主导，动量因子Alpha显著 |
| 催化剂 | 30天内硬规则 | 保留30天硬规则，但A级conviction可例外 | 长期底仓逻辑在短期模拟盘中效率低 |

#### 美股共识类型三分法

| 类型 | 定义 | 操作 |
|------|------|------|
| **Type A: 共识+预期已充分定价** | 股价已反映最乐观情景，forward P/E > 52w avg × 1.3 | 排除做多，列入做空watchlist |
| **Type B: 共识+预期仍有差距** | 分析师看多，但近3季beat率>70% | 可建仓，"预期差"驱动，不是共识本身 |
| **Type C: 共识+赛道扩张期** | 整个赛道TAM在扩张，EPS增速在加速 | 可持有底仓，重仓等催化剂 |

**共识判断流程**:
```
共识是否15/15看多？
  → 是 → 看EPS beat率（近3季）
       → beat率>70% → Type B，可做多（预期差驱动）
       → beat率<50% → 看PE vs 历史均值
              → PE高于历史1.3x → Type A，排除做多
              → PE在历史均值内 → Type C，底仓
  → 否 → 正常评估
```

#### 美股Momentum三层框架

| 层级 | 定义 | 阈值 |
|------|------|------|
| 相对强度（RS） | 标的52周涨幅 vs SPY涨幅之差 | RS>+10%=动量股优先；RS<-10%=需更强基本面理由 |
| 资金流（Price-Volume） | 连续3天收盘价上涨+成交量>20日均量 | 出现=主动资金流入；放量下跌=主动资金流出 |
| 52周位置 | 当前价在年内区间的分位数 | >80%分位=需更强催化剂；40-80%=标准仓位；<40%=C级侦查等趋势 |

**Momentum整合决策**:
- RS>+10% AND 资金流入 → 正常仓位（按A/B/C计划建仓）
- RS>+10% AND 资金流出 → 减半仓位，等资金回流
- RS在[-10%, +10%] → 正常仓位，不主动加仓
- RS<-10% → 只建C级观察仓（2-3%），等动量改善

### 4.4 美股做空规则

| 条件 | 具体标准 |

### 4.5 A股专用方法论（新增）

#### 市场呼吸5个量化指标（每日W1执行前扫描）

**B1: 情绪温度计（关键决策依据）**:
```
沪市成交额（过去5日均值）:
  · <8000亿 = 市场低温，不追高，守底仓，不新建仓
  · 8000-12000亿 = 正常，按策略操作
  · >12000亿 = 过热，减仓等回调
TMT板块成交占比:
  · >45% = 拥挤度过高，即便看多也不加仓
两融余额:
  · 破2.1万亿 = 牛市加速信号
北向资金:
  · 连续5日净流入 = 外资确认主题
```

**B2: 板块轮动热度**:
- 今日换手率TOP20个股所在板块 → 识别当日领涨板块
- 同一板块连续3日领涨 = 过热信号，新进资金收益有限

**B3: 主力资金净流入（板块级）**:
- 板块连续3日主力净流入>0 + 换手率放大 → 机构跟进确认
- 板块主力净流出但个股上涨 → 拉高出货信号，不追

**B4: 游资主导度**（来源：龙虎榜）:
- 游资营业部≥3家 → 游资主导（切技术框架，不用基本面）
- 机构席位≥2家 → 机构主导（可用基本面框架）
- 游资席位占买入总额>60% = 纯妖股，不参与（E8+E16规则）

**B5: 催化剂密度日历**:
- 未来30天有3+个明确催化剂节点 = 市场有说法，可以部署
- 催化剂间隔>3周 = 动能可能衰减，轻仓或等

#### 板块轮动两段式结构（用户验证的核心规律）

```
第一段：游资/小票先飞
  信号: 小盘股（市值<50亿）集中涨停 + 龙虎榜游资营业部扎堆
  含义: 主题刚被发现，游资试水，机构尚未进场
  操作: 这是布局窗口——不追妖股，找该主题的真龙头买入

第二段：机构/龙头后动
  信号: 该主题ETF/大盘股放量 + 主力净流入转正
  含义: 机构确认主题，进入快速拉升阶段
  操作: 龙头仍可买，但不是第一波了
```

**关键陷阱**: 龙头不动≠主题死（E11规则）。龙头滞后于小票是正常节奏，不要因龙头还没动就否认主题。

**游资先飞的4个具体信号（满足3条=布局龙头黄金窗口）**:
1. 同一主题下有2只以上小票连板（非单个妖股）
2. 龙虎榜同一天多只小盘股出现相同游资营业部（抱团主题）
3. 东财股吧/雪球主题词频比前7日均值>3倍
4. 该主题真龙头涨幅<小票（机构还没跟进）

**三阶段操作规则**:

| 阶段 | 特征 | 操作 |
|------|------|------|
| **孵化期** | 小票零散涨停，尚无ETF资金 | 侦查仓（3-5%），观察龙头 |
| **爆发期** | 龙头放量突破，ETF申购增加，主力净流入>50亿/日 | 主仓（8-15%，按conviction等级），追龙头 |
| **尾声期** | 游资撤退（龙虎榜变成机构卖出+游资小手），板块连续5日无新高 | 分批止盈，不追高 |

#### A股进场时机规则

| 场景 | 规则 | 原理 |
|------|------|------|
| 政策发布后 | 等第5-10个交易日布局 | 政策当天情绪顶，资金真正入场在第5-8天 |
| 产业大会前 | 会前1-2周布局，会期观望 | 预期炒作阶段 |
| 节后复盘 | 节后第2个交易日 | 节后第1天是情绪高开，真资金第2天验证 |
| 涨停次日 | 涨停是确认信号，不是阻力 | 涨停=市场确认催化剂为真，不要等回调 |
| 美股映射 | 美股龙头隔夜大涨>5%，次日开盘前30分钟评估 | 映射窗口极短，超过30分钟已被定价 |
| 业绩超预期 | 预告发布后等1个交易日 | 当日gap-up，等情绪稳定后的第1天才是真入场 |

#### A股出场时机规则

| 场景 | 规则 | 违反代价 |
|------|------|---------|
| 催化剂已兑现 | 立即评估，3天内决定是否退出 | "见光死"是A股最高频死法 |
| 2周无催化 | 14个交易日无新催化为退出评估依据 | 不主观设"该退了" |
| 板块连续5日无新高 | 尾声信号，开始分批止盈 | 不恋战，主题退潮快 |
| 游资龙虎榜变"卖出" | 原来买入的游资营业部出现在卖出席位 | 游资撤退=主题第一阶段结束 |
| 大涨日不减仓 | T+1约束，卖了当天买不回来 | 如果明天继续涨则永久踏空 |

#### "催化剂在前面"判断标准（纠正等回调伪逻辑）

**错误逻辑**: "涨停了所以等回调" — 这是用价格行为否定thesis

**正确框架**:
```
问题: 催化剂在前面还是后面？

如果在前面:
  → 涨停也买（更多买家还会来）
  → "等回调到¥XX"这句话不要说

如果已兑现/即将兑现:
  → 才该等或出场（见光死风险）
  → 不是因为价格涨了多少

案例: 德赛西威5/21涨停 → FSD Q3获批还在前面 → 不该等
反例: 某标的政策已出+股价大涨30% → 催化剂基本兑现 → 此时才叫"等"
```

### 4.6 美股专用方法论（新增）

**核心差异一句话**: A股做"主题+时机"，美股做"预期差+动量确认"。

**美股进场三情景决策树**:

情景A（上升趋势，RS强）:
```
不等回调 → 趋势初期小仓入场（C级3-4%）→ 第一个催化剂验证Beat → 加至目标仓位
理由: 错过趋势的成本 > 稍贵价格买入的成本
```

情景B（刚出现催化剂）:
```
事件当天不追 → 等3-5天资金验证（Volume消化）→ 确认高点不破再建仓
理由: 事件当天追涨是追买后来者
```

情景C（下跌趋势，RS弱）:
```
不接飞刀 → 等趋势反转信号（底部放量+3日连涨）→ 再建C级侦查仓
理由: 便宜不等于便宜够了
```

**美股实操检查表（替代"等回调"的模糊规则）**:
```
□ 今天价格在20日均线上方？（是=趋势向上，可买；否=等趋势确认）
□ 上周成交量是否高于平均（=资金在流入）？
□ 30天内是否有具体催化剂？（是=催化剂驱动；否=等）
□ 过去1周有-5%以上回调但未破前低？（是=这就是回调窗口）
```

### 5.2 ABCD下跌分类——下跌先分类再行动

> **CRITICAL**: ABCD分类之前，禁止任何卖出动作。先分类，后行动。60秒内完成分类。

| 类型 | 名称 | 判别标准 | 行动 | 止损规则 |
|------|------|---------|------|---------|
| **A** | 大盘系统性下跌 | 参考指数（KWEB/SOX/沪深300）同步跌≥1.5%；无个股新闻 | **Hold**（不操作，不恐慌，不补仓） | 只有同时触发组合级止损才处理 |
| **B** | 板块轮动噪音 | 指数稳或涨；无新闻；下跌<3% | **Hold**，观察1-2天 | 持续3天且无基本面变化仍持有 |
| **C** | 叙事切换（非破坏性） | 有新闻但不涉及核心thesis；市场情绪切换 | **评估**：减仓0-30%（按信心等级），写下重新评估结论再操作 | 24h内给出C→D升级结论或维持C |
| **C+** | 待确认叙事变化 | 单一信源/传闻/预期打压，无硬数据 | **观察**：24h内主动核实；核实前不操作 | 核实后升级C或D；超过48h视为D |
| **D** | Thesis被证伪 | 财报核心指标miss≥2个季度/重大客户流失公告/政策文件否决/管理层不明原因离职 | **立即清仓**，48h内执行，不等反弹 | 无例外。D类不存在"再等等看" |

**60秒分类流程**:
```
1. 看参考指数：跌≥1.5%？ → 是 = A类 → Hold，不操作
2. 有没有今天的新闻？ → 没有 = B类 → Hold 1-2天观察
3. 新闻改变了thesis核心逻辑吗？
   → 没改变 = C类 → 评估，写结论，可减0-30%
   → 改变了但只有传闻/单一信源 = C+类 → 24h核实，暂不操作
   → 有硬数据支撑（财报/公告/政策文件）= D类 → 48h内清仓
记住：只有D类是必须卖的。
```

**C与D的高发混淆场景**:

| 场景 | 分类 | 理由 |
|------|------|------|
| 财报beat但市场叙事切换（META案例） | C类 | 公司基本面未变，市场重定价叙事 |
| 竞争对手发布更好产品 | C+→D | 需核实对份额/GPM的具体影响 |
| 大客户单季度减少采购 | C+类 | 单季度可能是库存调整，≥2季度升D |
| 核心技术/专利被否 | D类 | 硬壁垒消失，thesis破坏 |
| 传闻监管打压（无正式文件）| C+类 | 等文件落地；A股尤其高发 |
| 正式监管文件落地（负面）| D类 | 政策确定，无讨论空间 |
| 管理层CEO替换（内部提拔）| B-C类 | 评估新任背景 |
| 管理层创始人离职（无交代）| D类 | 创始人vision是核心资产时直接D |
| 财报in-line但股价-9%（HSAI案例）| **C类，不是D类** | in-line非miss，止损前先分类 |

### 5.3 出场5条规则（v3.0更新）

1. **D类下跌**（thesis被证伪）: 立即止损，无商量
2. **止损触发**: 无条件执行，同日不回买
3. **目标达成**: 分批出（50%在目标价1 + 50%用trailing stop锁剩余）
4. **2周无催化**: 14个交易日无新催化，重新评估是否继续持有
5. **再平衡**: 单只超过信心等级上限 → 减仓至对应等级目标值（A级20%/B级12%/C级6%），当日执行

**止盈分批规则（新增）**:

| 阶段 | 触发条件 | 操作 |
|------|---------|------|
| 第一批 | 达到目标价的70% | 卖出25%（留75%等催化剂完整兑现）|
| 第二批 | 达到目标价 | 再卖25%（剩50%用trailing stop保护）|
| 第三批 | 超过目标价10% | 再卖25%（剩25%长期底仓）|
| 清仓 | 2周无新催化 OR 基本面变化 | 全出 |

**Trailing Stop参数（A股）**:
- 建仓后+15%前: 硬止损（成本-8%，B级）
- 达到+15%后: 从高点回落12%触发trailing stop
- 达到+30%后: 从高点回落10%触发trailing stop
- 达到+50%后: 从高点回落8%触发trailing stop

**A股T+1约束**（禁止操作）:
- 大涨日卖出"等回调再买" → T+1不可执行，永久踏空风险

## 8. 行为铁律（防止重蹈覆辙）

### L1-L9: 原有铁律（更新L1）

| 编号 | 铁律 | 违反后果 |
|------|------|---------|
| L1 | **不全仓** | 单只最大15%（A级可至25%），现金最低20%，违反立即强制减仓 |
| L2 | **不先买后研究** | thesis未写完不执行交易，无论价格怎么动 |
| L3 | **赢了不加码** | 同标的第二轮size ≤ 第一轮建仓量（首次加仓规则：升级后分步补仓）|
| L4 | **不系统性偏多** | 每日核对总部署vs计划，不因乐观情绪超配 |
| L5 | **下跌先分类** | 按ABCD分类后再行动，不情绪止损（C类≠止损！）|
| L6 | **不替代品陷阱** | 买不了A就等，不买低相关的B |
| L7 | **价格必须实时查询** | 所有价格用`yf price TICKER`实时查，禁止用记忆/旧文件中的价格 |
| L8 | **模拟盘与真实交易独立** | 模拟盘决策不参考用户真实仓位 |
| L9 | **计划偏差强制记录** | 任何偏离strategy的操作，先记录到daily-reviews再执行 |

### L10-L15: 新增行为铁律（v3.0）

**L10 — 关注点漂移：看别人涨了就想换**

- **定义**: 持有标的X未动，看到Y/Z涨了 → 萌生"换到Y/Z"的想法
- **诊断信号**: 卖出原因是"别的股票涨了它没涨"（而不是thesis变化）
- **铁律**: 卖出前必须回答①thesis变了吗？②我是因为看了别的股票才想卖吗？如果②是"是"→关闭行情软件，等1小时后再决定
- **验证案例**: 肇民05-21早盘，机器人板块其他股涨，肇民没动→卖出→下午肇民+2.86%
- **Why**: 相对表现焦虑，和追涨是同一情绪的两面

---

**L11 — 催化剂前离场：在最关键的时候撤退**

- **定义**: 有明确催化剂日期，在催化剂前1-3天因为"已经涨了不少"而减仓
- **诊断信号**: 建仓时写了催化剂日期；催化剂未到就开始考虑减仓
- **铁律**: 催化剂前默认持有。减仓需要给出"不是催化剂原因"的充分理由（如ABCD分类为D类，或已大幅超过目标价）
- **Why**: 从赔率角度：催化剂前持有期望值 > 未持有。短期获利欲望压过赔率思维

---

**L12 — 赢小输大反置：小赚就跑，大亏死扛**

- **定义**: 持仓小幅盈利时快速获利了结；持仓大幅亏损时扛着不止损
- **诊断信号**: 盈利方向仓位越来越小；亏损方向仓位维持或扩大
- **铁律**: 卖出时问自己"如果这是一笔新投资，我会买吗？"是→持有；不是→这才是真正的卖出信号
- **验证案例**: 蓝思赚了¥6870快速清；HSAI亏$757扛着（部分合理因为有thesis）
- **Why**: 前景理论（亏损厌恶）

---

**L13 — 追涨三联动：FOMO驱动的系统性错误（三行检查表，强制）**

- **定义**: 任何由"今天才知道它涨了"驱动的买入决策
- **铁律（三行检查表，强制，缺一不可）**:
  - ①买的理由是什么（不是"涨了"）
  - ②跌20%我割不割（说不出→不买）
  - ③是不是今天才知道它的（是→不买）
- **第③条最关键**: 今天才知道 = 信息链末端 = 别人早就知道 = 你在买入别人的出货
- **例外**: 底部放量涨停+真实催化剂（如德赛西威从¥96反弹，是底部启动不是高位追涨）
- **Why**: FOMO（错失恐惧）驱动的系统性错误

---

**L14 — 仓位-Conviction倒挂：最大仓位是最不确定的**

- **定义**: 仓位大小与研究深度/信心等级呈反比（A级标的仓位最小，C级标的仓位最大）
- **诊断信号**: 说不出thesis的标的持仓最重；研究最深的标的持仓最轻
- **铁律**: 每次建仓前先定信心等级（A/B/C）→对应仓位上限（25%/15%/8%）→按上限建仓。说不出thesis = C级，C级上限8%
- **验证案例**: 德赛电池1200股无thesis vs 思源100股最高conviction（完全倒置）
- **Why**: 买入时没有强制sizing决策框架，凭感觉

---

**L15 — Thesis错配不自知：涨了不代表你的理由对了**

- **定义**: 买入理由（thesis）与标的真实驱动力不一致，短期涨了而强化错误thesis
- **诊断信号**: 说不清楚"它现在为什么涨，和我买的理由是同一个吗？"
- **铁律（强制一步，买前执行）**: 搜索"[标的名] 最近为什么涨"→对比搜索结果和自己的买入理由→不同=不买
- **验证案例**: 圣泉605589——用户因"钠电硬碳"买入，实际涨幅驱动是PPO树脂/AI PCB（H1+51%全靠PPO），钠电收入<3%。Thesis完全错配，当日全清
- **Why**: 自我确认偏误（confirmation bias）

- [ ] 价格是刚查的实时价格，不是记忆值
- [ ] thesis一句话能说清楚

- [ ] 如果是计划外操作，thesis已写到daily-reviews
- [ ] 三行检查表过了吗？（①买的理由 ②跌20%割不割 ③今天才知道吗）

## 9. If-Then预承诺格式（v2.0升级版）

**核心规则: 没有预承诺 = 没有操作。禁止临场发挥。**

### 标准格式（v2.0含行为防护）

```markdown
## [标的] If-Then 预承诺表
建仓日期: XXXX-XX-XX
建仓价格: ¥/$ X.XX
催化剂日期: XXXX-XX-XX
Conviction级别: A/B/C
止损线: ¥/$ X.XX（成本价 -X%）
目标价1: ¥/$ X.XX
目标价2: ¥/$ X.XX（trailing stop触发后）

### 催化剂前（现在到XX/XX）
- If 股价下跌5-10%（A/B类下跌）: 持有，不止损
- If 股价下跌>12%（需判断C/D类）: 执行ABCD分类，D类立即止损
- If 出现更好的买入机会冲动: 先问"今天才知道它的吗？是→不动"（L13）

### 催化剂后（XX/XX当天及后续）
- If [超预期结果]: [具体操作，如"加仓至X%"]
- If [In-line结果]: [具体操作，如"减仓25%，保留底仓等下一催化"]
- If [Miss/负面结果]: [具体操作，如"执行ABCD分类，D类全清"]

### 时间止损（无催化剂期）
- If 14个交易日无新催化 AND 处于B/C类下跌: 重新评估是否继续持有
- If 14个交易日无新催化 AND 仓位亏损>-8%: 清仓

### 行为防护条款
- 卖出前必须确认: "我卖出的理由不是因为看了别的股票"（L10）
- 加仓前必须确认: "不是因为今天涨停才想加"（L13）
- 催化剂前不减仓默认持有（L11）
```

## 13. 组合级风控触发阈值

### 日内触发

| 触发条件 | 立即行动 |
|---------|---------|
| 单日组合亏损 >2.5% | 暂停新建仓（当日剩余时间）|
| 单日组合亏损 >4% | 暂停新建仓 + 检查D类标的是否需要止损 |
| 单只持仓单日亏损 >7% | 强制ABCD分类，C+及以上减仓50% |
| 单只持仓单日亏损 >10% | 强制分类，D类立即清仓，C类减仓至C级上限以下 |

### 周度触发

| 触发条件 | 行动 |
|---------|------|
| 周度组合亏损 >5% | 下周前3个交易日禁止新建仓，专注持仓review |
| 周度组合亏损 >8% | 全仓review，所有B/C级标的重新评级，降级的立即减仓 |
| 连续2周亏损（合计 >6%） | 强制策略复盘（找根因，不是找解释）|

### 月度触发

| 触发条件 | 行动 |
|---------|------|
| 月度亏损 >10% | 仓位上限临时降至70%（现金下限30%）持续1个月 |
| 月度亏损 >15% | 仓位上限降至50%，持续2个月；必须做完整复盘才能解除 |

### 赢家模式（继续复制）

| 模式 | 代表案例 | 改进 |
|------|---------|------|
| **L1: 无催化剂的赛道配置** | GOOGL/ADBE/安集 | 短期模拟盘中，纯赛道配置不适用 |
