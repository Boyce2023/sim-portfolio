# Claude模拟盘交易策略 v1.0

## 账户规模
- A股: ¥1,000,000 (T+1, 只做多)
- 美股: $150,000 (T+0, 可做空, 2倍杠杆上限=$300,000总敞口)
- 周期: 2026-05-19 → 2026-06-18 (1个月)

## 美股特殊规则
- T+0: 日内可反复买卖同一标的
- 做空: 允许裸空/对冲空, 空头仓位计入总敞口
- 杠杆上限: 总敞口(多头市值+空头市值) ≤ $300,000 (2x初始资本)
- 做空止损: 反向+15%强平(空头止损比多头更紧)
- 日内交易: 催化剂日允许盘中操作, 非催化日仅开/收盘窗口

## 策略哲学

供给侧优先 + 催化剂驱动 + 严格风控。不做纯概念炒作，不追涨停，不猜政策。

核心原则来自用户10步流程和240天A股复盘教训：
1. 每笔交易必须有明确的thesis和催化剂日期
2. Bear case downside > 20% 不建仓
3. 事件落地当天不动手，等1周资金验证
4. 退出看"2周无催化"，不主观设时间
5. 先淘汰再推荐——研究宽度是为了缩窄

## 仓位管理

### 单只上限
| 信心等级 | 仓位占比 | 触发条件 |
|---------|---------|---------|
| 核心持仓 (A级) | 10-15% | 自主研究完成+催化剂明确+bear case <15% |
| 重点持仓 (B级) | 5-10% | thesis清晰+有催化剂 |
| 观察仓 (C级) | 2-5% | 初步thesis+验证中 |
| 交易性仓位 | 2-5% | 短期事件驱动 |

### 组合约束
- 单只上限: 15% (A股) / 15% (美股)
- 单板块上限: 30%
- 现金下限: 20% (保持灵活性)
- 同日新建仓不超过3只
- 计划外加仓需先记录thesis再执行（杜绝"先买后研究"）

### 止损规则
- 每只必须有硬止损（入场即设）
- 组合级止损: 单日亏损>3% → 暂停新建仓1天
- 单只止损后不在同一交易日回买

## 出场规则

1. **D类下跌(thesis被证伪)**: 立即止损
2. **止损触发**: 无条件执行
3. **目标达成**: 分批出(50%目标/50% trailing)
4. **2周无催化**: 评估是否退出
5. **组合再平衡**: 单只超15%→减仓至12%

## 每日流程 (5个监测窗口)

### 09:30 BJT — A股开盘
1. 获取A股实时开盘价, 检查隔夜跳空
2. US昨日收盘对A股的映射信号
3. 执行A股开盘交易(如果预案触发)
4. 检查止损是否在开盘即触发

### 13:30 BJT — A股午后
1. 更新A股半日价格变动
2. 尾盘交易决策(加仓/减仓/新建仓)
3. 检查日内ABCD分类
4. 如有催化剂落地(公告/新闻), 执行预案

### 21:30 BJT — US开盘
1. 获取US实时开盘价, 检查盘前变动
2. 执行US开盘交易(催化剂日内可日内交易)
3. 做空机会扫描(如有thesis)
4. A-share映射的US对标变动

### 01:00 BJT — US盘中
1. 检查US盘中波动, 持仓状态
2. 日内交易机会(T+0, 催化剂日)
3. 止损/目标检查
4. 盘中新闻扫描

### 04:30 BJT — US收盘
1. 记录US收盘价, 日内P&L
2. 盘后earnings反应(如NVDA after-hours)
3. 更新portfolio_state.json
4. 写每日复盘
5. 为次日A股开盘准备映射信号
